# raw/

원본 파일 보관 디렉토리.

LLM은 이 폴더를 **읽기만** 하고 수정하지 않는다.

## 용도

- 법령 원문 PDF
- 산림청 고시·지침 원본 파일
- Obsidian Web Clipper로 수집한 원문 마크다운

## 현황

현재 비어있음. 원본 파일이 필요할 경우 이 폴더에 추가한다.

- 행정규칙 원문은 [legalize-kr/admrule-kr](https://github.com/legalize-kr/admrule-kr) 참조
- 산림청 공식 자료는 [산림청 통합자료실](https://www.forest.go.kr) 참조
