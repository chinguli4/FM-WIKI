# 산림경영 Wiki

산림청 지침 및 관련 법령 기반의 산림경영 실무 가이드입니다.

## 구조

```
FM wiki/
├── raw/               ← 원본 파일 보관 (PDF 등, 불변)
└── wiki/              ← LLM이 작성·관리하는 위키
    ├── index.md       ← 전체 페이지 목록
    ├── log.md         ← 운영 이력
    ├── 행정규칙/      ← 법령별 원문 wiki화
    │   ├── 고시/
    │   ├── 예규/
    │   ├── 지침/
    │   └── 훈령/
    ├── 산림작업/      ← 사업별 합성 페이지
    │   ├── 조림/
    │   ├── 숲가꾸기/
    │   └── 벌채/
    └── 산림계획/      ← 사업별 합성 페이지
        ├── 경영계획/
        └── 수확조절/
```

## 사용법

```bash
# 특정 지침 검색
grep -r "간벌" wiki/산림작업/

# 지침 변경 이력
git log -- wiki/산림작업/벌채/지침.md

# 두 지침 비교
diff wiki/산림작업/벌채/지침.md wiki/산림작업/숲가꾸기/지침.md
```

## 근거 법령

- 산림자원의 조성 및 관리에 관한 법률 (산림자원법)
- 산림경영계획 작성 및 운영규정 (산림청 훈령)
- 산림작업 설계·감리 요령
- 조림·숲가꾸기 사업 시행지침

## 업데이트

지침 개정 시 해당 파일을 수정하고 커밋 메시지에 개정 연월 및 고시번호를 기재합니다.
