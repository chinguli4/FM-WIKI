# 산림경영 Wiki

산림청 지침 및 관련 법령 기반의 산림경영 실무 가이드입니다.

## 구조

```
kr/
├── 산림작업/
│   ├── 벌채/          ← 주벌·간벌·수확작업 지침
│   ├── 조림/          ← 수종별 조림기준, 식재밀도
│   └── 숲가꾸기/      ← 풀베기·어린나무가꾸기·가지치기·솎아베기
└── 산림계획/
    ├── 경영계획/      ← 산림경영계획 작성·인가·운영
    └── 수확조절/      ← 연년벌채량, 수확조절 방법
```

## 사용법

```bash
# 특정 지침 검색
grep -r "간벌" kr/산림작업/

# 지침 변경 이력
git log -- kr/산림작업/벌채/지침.md

# 두 지침 비교
diff kr/산림작업/벌채/지침.md kr/산림작업/숲가꾸기/지침.md
```

## 근거 법령

- 산림자원의 조성 및 관리에 관한 법률 (산림자원법)
- 산림경영계획 작성 및 운영규정 (산림청 훈령)
- 산림작업 설계·감리 요령
- 조림·숲가꾸기 사업 시행지침

## 업데이트

지침 개정 시 해당 파일을 수정하고 커밋 메시지에 개정 연월 및 고시번호를 기재합니다.
