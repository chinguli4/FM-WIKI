# 벌채 작업설계 기준

> 근거: 산림작업 설계·감리 요령 (산림청 훈령)

## 1. 작업설계서 작성 대상

- 국유림: 모든 벌채작업
- 공·사유림: 벌채 면적 1ha 이상 또는 벌채재적 50㎥ 이상

## 2. 작업설계서 구성

```
1. 사업 개요
   - 사업명, 위치, 면적, 사업량
2. 현황 조사
   - 임분 현황 (수종, 영급, ha당 재적)
   - 지황 (경사, 사면방향, 토양)
   - 임도·작업로 현황
3. 작업 계획
   - 벌채 방법 및 범위
   - 집재 방법
   - 잔재물 처리 계획
4. 환경보전 계획
   - 존치 구역 지정
   - 토사 유출 방지 계획
5. 사업비 산출
```

## 3. 임분 조사 기준

### 3.1 표준지 설정
- 표준지 크기: 0.04ha (20m × 20m) 원칙
- 표준지 수: 사업지 면적에 따라 결정
  - 1ha 미만: 3개소 이상
  - 1~5ha: ha당 2개소 이상
  - 5ha 초과: ha당 1개소 이상

### 3.2 조사 항목
| 항목 | 조사 내용 |
|------|-----------|
| 수종 | 교목층·아교목층·관목층 |
| 흉고직경 | 6cm 이상 전수 또는 샘플 조사 |
| 수고 | 우세목 10본 이상 |
| 수령 | 생장추 또는 수간석해 |
| ha당 본수 | 전수조사 |
| ha당 재적 | 재적표 적용 |

### 3.3 재적 산출

```
단목재적 = f(흉고직경, 수고, 형수)
ha당 재적 = Σ단목재적 × (10,000 / 표준지 면적)
```

수종별 재적표: 국립산림과학원 수확표 사용

## 4. 집재 계획 기준

### 4.1 집재 방법 선정 기준

| 경사도 | 권장 집재 방법 |
|--------|---------------|
| 15° 이하 | 트랙터 집재 |
| 15~30° | 포워더, 소형 가선 |
| 30° 이상 | 가선집재, 가반식 집재기 |
| 급경사·암반 | 인력 집재 |

### 4.2 집재로 설치 기준
- 집재로 폭: 3m 이내
- 집재로 간격: 40~60m
- 집재로는 작업설계도에 명시

## 5. 사업비 산출 기준

- 노임 단가: 산림청 고시 노임 단가 적용
- 기계 사용료: 건설기계 표준 시장 가격
- 재료비: 실거래가 또는 물가정보지

### 주요 공종별 단가 적용

| 공종 | 적용 단가 기준 |
|------|---------------|
| 벌목·조재 | 재적㎥당 또는 본당 |
| 가선집재 | 재적㎥당 |
| 트랙터집재 | 재적㎥당 |
| 잔재물처리 | ha당 |

## 6. 작업설계도 작성 기준

- 축척: 1/5,000 이상 (지형도 기반)
- 표시 항목:
  - 벌채 구역 경계
  - 존치 구역 (계류변, 능선 완충지)
  - 집재로, 임도
  - 표준지 위치
  - 토사유출방지시설 위치

## 7. 설계 감리

- 총 사업비 1억 원 이상: 감리 의무
- 감리원 자격: 산림기술사 또는 산림경영기술자 1급
- 감리 시기: 착공 전·중·완료 후
