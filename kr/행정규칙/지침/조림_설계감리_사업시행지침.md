## 제1장 총칙

**목적** 조림사업의 설계기준, 시행 요령, 사업비 산출기준, 감리기준을 규정한다.

**기본방향**
- 조림사업의 계획적 추진
- 품질 향상
- 합리적인 사업비 산출

**주요 정의**

| 용어 | 정의 |
|------|------|
| 발주자 | 사업을 발주하고 계약을 체결하는 자 |
| 감독자 | 발주자를 대리하여 사업을 감독하는 자 |
| 감리자 | 발주자의 감독권한을 대행하는 자 |
| 사업시행자 | 설계도서에 따라 조림사업을 시행하는 자 |

**사업 구분** 면적과 시행 주체에 따라 5가지로 구분된다.

| 구분 | 조건 |
|------|------|
| 1유형 | 3ha 이상, 용역 설계·감리 |
| 2유형 | 3ha 미만, 간이설계 |
| 3유형 | 지자체 직접 시행 |
| 4유형 | 산주 직접 시행 |
| 5유형 | 국가 직접 시행 |

---

## 제2장 설계

**사업계획 연계** 다음 계획과 연계하여 작성하여야 한다.
- 지역산림계획
- 경제림 육성단지 관리계획

**실시설계 완료 시기**
- 원칙: 사업시행 전년부터 당해연도 착수 전까지
- 예외: 산림재해·예산 변동 시 필요 시점에 작성 가능

**실시설계 서류 목록**

| 서류 | 내용 |
|------|------|
| 설계설명서 | 사업 개요, 설계 기준 |
| 일반시방서·특별시방서 | 작업 방법·기준 |
| 현황도·작업지시도 | 위치, 소반 경계, 작업 내용 |
| 예정공정표 | 작업 일정 |
| 사업비 원가계산서·설계내역서 | 비용 산출 |

## 제3장 사업시행

**현장대리인**
- 산림기술자를 현장대리인으로 선임·배치하여야 한다.
- 일정 조건 하에 3개 사업장까지 통합 배치 가능

**작업원 자격**
- 산림경영기술자 기능2급 이상: 전체 인원의 30% 이상
- 산주 직접 시행의 경우 적용 제외

**제출 서류**

| 단계 | 서류 |
|------|------|
| 착공 시 | 착수계, 작업계획서, 현장대리인 선임계, 안전관리계획서 |
| 완료 시 | 완료계, 완료검사신청서 |

**품질관리**
- 설계와 현장이 다를 경우: 감리자에게 즉시 보고
- 부실 시: 사업시행자가 재작업 비용 부담

**안전관리**
- 사업시행자가 작업원 안전에 관한 모든 법규 이행에 책임을 진다.
- 안전교육일지를 현장에 비치하여야 한다.

---

## 제4장 감리

**감리 대상** 실시설계를 용역으로 시행한 경우 감리를 의무적으로 실시한다.

**감리 범위**
- 조림예정지정리
- 식재작업
- 조림감리표준지조사

**감리원 권한** 다음의 경우 사업시행자에게 시정 또는 작업중지를 명할 수 있다.
- 작업 품질이 불량한 경우
- 실시설계서대로 작업이 되지 않는 경우
- 안전상 필요한 경우

**감리 서류**

| 단계 | 서류 |
|------|------|
| 착공 전 | 사전검토 보고서 |
| 시행 중 | 중간감리 보고서, 감리일지 |
| 완료 | 완료검사 결과보고서, 완료도면 |

---

## 제5장 사업완료

| 완료 구분 | 절차 |
|-----------|------|
| 실시설계 완료 | 도서 검토 후 이상 없으면 완료 처리 |
| 사업시행 완료 | 예비완료검사 후 필요 조치 완료 시 최종 처리 |
| 감리 완료 | 관련 서류·전자기록매체 제출 후 검토·완료 처리 |
