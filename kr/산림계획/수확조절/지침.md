# 수확조절 지침

> 근거: 산림자원의 조성 및 관리에 관한 법률, 산림경영계획 작성 및 운영규정

## 1. 수확조절의 개념

수확조절(收穫調節)이란 산림의 지속가능한 경영을 위해 연간 벌채 가능량을 산정하고 계획적으로 수확하는 방법이다.

**핵심 원칙**: 연간 벌채량 ≤ 연간 생장량 (지속가능성 보장)

## 2. 주요 수확조절 방법

### 2.1 면적법 (Area Method)

윤벌기 동안 매년 동일한 면적을 벌채하는 방법.

```
연년벌채 면적 = 총 작업급 면적 ÷ 윤벌기
```

**예시**
- 소나무 작업급 면적: 720ha
- 윤벌기: 60년
- 연년벌채 면적 = 720 ÷ 60 = **12ha/년**

**장점**: 계산 단순, 공간적 작업 분산
**단점**: 임분 생장 차이 미반영 → 수확량 불균등

### 2.2 재적법 (Volume Method)

연간 생장량을 기준으로 벌채량을 결정하는 방법.

```
연년벌채 재적 = 총 생장량 (정기평균생장량 기준)
```

**예시**
- 작업급 총 재적: 10,000㎥
- 윤벌기: 50년
- 연년생장량: 3.5㎥/ha/년 × 200ha = 700㎥/년
- 연년벌채 재적 = **700㎥/년**

**장점**: 생장량 기반, 현실적
**단점**: 정기적 임분 조사 필요

### 2.3 수확표법 (Yield Table Method)

수종별 수확표(생장표)를 이용하여 지위지수·임령별 재적을 산출하고 벌채량을 결정.

```
연년벌채 재적 = Σ(소반별 ha당 재적 × 소반 면적) ÷ 윤벌기
```

**국립산림과학원 수확표 활용**
- 수종·지위지수·임령 → ha당 재적 조회
- 우리나라 주요 수종 모두 구비

**장점**: 표준화된 방법, 정밀도 높음
**단점**: 현실 임분과 오차 발생 가능

### 2.4 생장률법 (Growth Percentage Method)

현존 재적과 생장률을 이용하여 수확량을 결정.

```
연년벌채 재적 = 현존 재적 × 생장률(%)
```

**예시**
- 현존 재적: 15,000㎥
- 생장률: 4%/년
- 연년벌채 재적 = 15,000 × 0.04 = **600㎥/년**

## 3. 수확조절 방법 선택 기준

| 산림 유형 | 권장 방법 |
|-----------|-----------|
| 동령 단순 인공림 | 면적법 또는 수확표법 |
| 다령 혼효림 | 재적법 또는 생장률법 |
| 소규모 사유림 | 재적법 (단순 적용) |
| 정밀 경영 필요 | 수확표법 + GIS 통합 |

## 4. 연년벌채량 산출 절차

```
Step 1. 임분 현황 조사
        → 소반별 수종, 임령, 재적, 지위지수

Step 2. 윤벌기 설정
        → 경영 목표에 따른 수종별 윤벌기

Step 3. 작업급 편성
        → 동일 조건 소반 묶음

Step 4. 수확 가능량 계산
        → 선택한 수확조절 방법 적용

Step 5. 생장량과 비교
        → 벌채량 ≤ 생장량 확인

Step 6. 연년 작업 계획 반영
        → 경영계획서 제9장 작성
```

## 5. 간벌 수확량 산정

간벌(솎아베기)은 주벌과 별도로 수확량을 산정한다.

```
간벌 수확량 = Σ(간벌 소반 ha당 재적 × 간벌율 × 소반 면적)
```

**간벌율 기준**
- 경도 간벌: 현존 재적의 20~25%
- 중도 간벌: 현존 재적의 25~35%
- 강도 간벌: 현존 재적의 35~40%

## 6. 지속가능성 검토

### 6.1 가이드라인
- 연간 총 벌채량(주벌+간벌) ≤ 연간 총 생장량
- 작업급 내 영급 배치가 균형 잡힐 것 (편중 지양)
- 10년 계획 종료 시 재적이 계획 시작 시점 이상 유지

### 6.2 영급 구조 점검

이상적인 영급 배치 (정상림):
```
영급   : Ⅰ  Ⅱ  Ⅲ  Ⅳ  Ⅴ  Ⅵ
면적비 : 균등 배분 (각 영급 약 1/6씩)
```

현실 임분 진단:
- 특정 영급 편중 → 수확량 불균등 위험
- 고령급 과다 → 단기 수확량 과대 후 공백
- 저령급 과다 → 향후 수확량 부족

### 6.3 탄소흡수 고려
- 벌채 후 재조림 시 탄소 재흡수 기간 반영
- 탄소흡수원 인증 목적 시: 연간 순흡수량 유지 의무

## 7. 주요 수종별 생장량 참고 (평균)

| 수종 | 지위 | ha당 연년생장량(㎥) |
|------|------|-------------------|
| 소나무 | 우량 | 6~8 |
| 소나무 | 중간 | 4~6 |
| 낙엽송 | 우량 | 10~14 |
| 낙엽송 | 중간 | 7~10 |
| 잣나무 | 우량 | 7~10 |
| 참나무류 | 우량 | 6~9 |
| 백합나무 | 우량 | 12~16 |

> 실제 산출 시 반드시 국립산림과학원 수종별 수확표 사용

## 8. 수확조절 결과 기재 양식 (경영계획서)

```
작업급명: 소나무 1작업급
총 면적: 120ha
윤벌기: 60년
현존 재적: 8,400㎥ (평균 70㎥/ha)
연년생장량: 480㎥ (4.0㎥/ha × 120ha)

[주벌 수확조절 - 면적법]
연년벌채 면적: 2.0ha/년
연년벌채 재적: 140㎥/년 (추정)

[간벌 수확조절]
간벌 주기: 10년 1회
1회 간벌 면적: 20ha
1회 간벌 재적: 600㎥

[총 수확량 검토]
연년 총 벌채량: 200㎥/년 ≤ 연년생장량 480㎥/년 ✓
```
