## 1. 핵심 원칙 — 수사의뢰는 형사 트랙만 가동

국무조정실 일괄 수사의뢰는 **③ 형사벌칙 트랙**(산림기술법 §31·산림자원법 §77)을 가동시킬 뿐, 행정처분 트랙(① 자격처분·② 등록처분·④ 과태료·⑤ 벌점)은 처분권자가 별도로 진행해야 한다.

| 트랙 | 권한자 | 국무조정실 수사의뢰로 자동 진행? |
|---|---|---|
| ① 기술자 자격취소·정지 (산림기술법 §12) | **지방산림청장** | ❌ 별도 진행 |
| ② 용역업 등록취소·영업정지 (산림기술법 §18) | **지방산림청장** | ❌ 별도 진행 |
| ② 시행업(산림사업법인) 영업정지·등록취소 (산림자원법 §25) | **시·도지사** | ❌ 별도 진행 |
| ③ 형사벌칙 (산림기술법 §31·산림자원법 §77) | 경찰·검찰 → 법원 | ✅ 수사의뢰로 진행 |
| ④ 과태료 (산림기술법 §33·산림자원법 §79) | 지방산림청장·시·도지사 | ❌ 별도 진행 |
| ⑤ 벌점 (산림기술법 §24 별표 4) | 발주청 | ❌ 별도 진행 |

> **핵심 명제**: 행정처분은 형사절차 결과를 기다릴 필요 없다. 행정조사 결과(또는 국무조정실 합동조사 결과 인계)만으로 즉시 §12·§18·§25 처분 진행 가능. ([[wiki/산림작업/자원조성/산림기술자_자격관리_체계]] §6⑤ 근거)

## 3. 단계별 실무 체크리스트

### [1~4] 합동조사 결과 인계 단계

- [ ] **합동조사 결과 공식 수령** — 국무조정실 공문 또는 회의록 형식으로 위반사실·증거자료·피의자 목록 정리본 확보
- [ ] **양 기관 간 자료 공유** — 지방산림청·시·도지사가 동일 자료에 접근하도록 합동조사 결과 동시 인계 요청
- [ ] **자체 행정조사 자료로 활용 근거 확보** — 산림기술법 §11⑤ (지방청)·산림자원법 §67① (양 기관) 행정조사 권한 행사로 갈음
- [ ] **위반 유형별 분류** — 자격증 대여(§11①)·차용(§11②)·알선(§11③)·이중취업(§11④) / 등록증 대여(§15⑤·⑥, §24④) 별도 정리
- [ ] **업체별 분류** — 산림기술용역업(지방청 관할) vs. 산림사업법인(시·도지사 관할) 분리

### [2] 수사의뢰 중복 방지

- [ ] **국무조정실 수사의뢰 범위 확인** — 자격증 대여·이중취업·등록증 대여 등 모든 §31·§77 위반행위 포함 여부 명시 확인
- [ ] **별도 고발 불요 확인** — 국무조정실 일괄 수사의뢰가 산림기술법 §31·산림자원법 §77 전 범위를 포괄하면 지방산림청·시·도지사의 형사소송법 §234② 직무상 고발 의무는 사실상 이행됨 (중복 고발 회피)
- [ ] **감사처분심의위원회 심의 불요** — 본 사건은 산림청 자체감사 결과 고발 트랙(감사규정 §36조9호·§39조의2)이 아니므로 본청 위원회 심의 불필요. [[wiki/법령/산림기술법/조문발췐_자격대여_이중취업]] §39조의2 트랙 구별 표 참조
- [ ] **보충 고발 필요성 검토** — 행정조사 중 합동조사 범위 밖 추가 혐의 발견 시 보충 고발 가능

### [6] 지방산림청 행정처분 절차

#### ① 사전통지 (행정절차법 §21)
- [ ] 처분 제목·당사자 성명·주소 / 처분 원인 사실·내용·법적 근거 / 의견제출 안내 명시
- [ ] **자격취소·등록취소** — 청문일 **10일 전까지** 통지 (§21②), 청문 주재자 소속·직위·일시·장소 포함
- [ ] **자격정지·영업정지·과징금** — **10일 이상**의 의견제출기한 부여 (§21③)
- [ ] 송달 — §14~§16 우편·교부·정보통신망

#### ② 청문 또는 의견제출

  - 청문 주재자: 지방산림청 소속 직원 (행정절차법 §28)
  - 시작 7일 전까지 자료 통지 (§28③)
- [ ] **의견제출 의무 대상**: 자격정지·영업정지·과징금·벌점
- [ ] 청문조서 작성·증거조사·청문 종결 → §35의2 처분 반영 의무

#### ③ 처분 결정 (행정절차법 §23·§24)

- [ ] **수사 진행** — 경찰 → 검찰 송치 → 기소 여부 결정 → 법원 재판

- [ ] **산림자원법 §77 4호** — 1천만원 이하 벌금 (산림사업법인 등록증 대여)
- [ ] **양벌규정** — 산림기술법 §32 / 산림자원법 §78 — 법인에도 벌금형
- [ ] **산림청 특별사법경찰관 수사권 없음** — 사법경찰관리직무법 §6 직무범위(임야 내 산림·임산물·수렵 범죄)에 자격관리 행정 위반은 미해당. 일반 경찰·검찰이 수사

## 4. 청문 일정 분리 — 세 건의 별도 청문

| 청문 대상 | 주재자 | 근거 | 본 사건 처리 |
|---|---|---|---|
| 기술자 자격취소 | 지방산림청장 (소속 직원) | 산림기술법 §29 | 위반 기술자별 각 1회 |
| 용역업 등록취소 | 지방산림청장 (소속 직원) | 산림기술법 §29 | 위반 용역업체별 각 1회 |
| 산림사업법인 등록취소 | **시·도지사** | 산림자원법 §68 3호 | 위반 시행업체별 각 1회 |

> 동일 사건의 세 청문이 **두 기관에서 별도 개최**된다. 청문일 10일 전 사전통지(행정절차법 §21②), 청문 시작 7일 전까지 자료 통지(§28③).

> 💡 **실무 조정 권고**: 가능하면 양 기관이 청문 일정을 조율해 당사자 부담을 줄이고, 동일 사실관계에 대한 진술 일관성을 확보. 다만 법적 의무는 아니며 각 기관 처분권 독립 행사.

## 5. 발주청 체크포인트

본 사건 관련 산림사업 발주청(국유림관리소·지방산림청·시·도·시·군·구)이 점검할 사항:

1. **계약 계속수행 여부 결정** — 처분 통지받은 후
   - 용역업: 산림기술법 §18⑦·⑧ **30일 이내** 해지 또는 동의 결정 — 필수취소(§18①1·3·4·5호)는 무조건 중단
   - 시행업: 산림자원법 §26③ 통지·인지일부터 **30일 이내** 도급계약 해지 가능
2. **현장배치 기술자 일치 확인** — 자격증 효력 정지·취소된 기술자가 현장에 잔류하면 §25② 현장이탈·§33①10호 100만원 이하 과태료 추가
3. **벌점 조회·입찰 감점** — 산림기술정보체계 공개 정보로 입찰 사전심사 감점 적용 (별표 4 제5호)
   - 별표 4 1.7-1) 무자격자 참여(설계) 2점 / 3.7-1) 무자격자 참여(감리) 3점
   - 1점 이상~20점 이상: 0.2~5점 입찰 감점
4. **하도급 점검** — 자격대여 업체가 하도급에 포함된 경우 발주청도 자격 미달 확인 의무
5. **신규 발주 시 결격 확인** — 자격 3년·시행업 3년·용역업 2년 결격기간 중인 자·업체 입찰 참가 차단
