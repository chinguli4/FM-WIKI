# 산불피해지 조림복원 매뉴얼 — 제1장 개요

> 출처: 산불피해지 조림복원 매뉴얼 (산림청, 2025.12.17)  
> 원본: `raw/manuals/2025-12-17 산불피해지 조림복원 매뉴얼.md` §1장 (lines 1-175)  
> 관련 페이지: [[2장_복원복구계획_피해조사]] · [[3장_조림복원_실행]] · [[4장_사후관리]]

---

## 1. 매뉴얼 목적

산불피해지의 피해 정도와 입지 여건별로 차별화된 복원·복구계획을 수립하고, 체계적인 조림복원을 실행하기 위한 절차와 방법을 제시한다.

| 핵심 용도 | 내용 |
|----------|------|
| 피해조사 표준화 | 산불 발생 직후 조사 항목·절차·방법을 정리하여 조사자의 판단 편차를 줄임 |
| 복원·복구계획 수립 | 피해면적·피해강도·피해액을 산정하여 복구 우선순위, 예산 편성, 정책 수립 자료로 활용 |
| 기능별 조림복원 | 산림의 6대 기능과 입지 조건을 연계해 조림복원 방향과 실행 방안을 결정 |
| 현장 실무 지원 | 경험이 적은 조사자도 피해강도·대상지·비용 산정 기준을 적용할 수 있도록 수치 기준 제시 |

---

## 2. 관계법령

| 구분 | 원문상 근거 | 주요 내용 | 기준일 해석 |
|------|------------|----------|------------|
| 산불피해지 조사 | 「산림재난방지법」 제48조 | 산불 원인, 피해 현황, 대응 분석·평가 등 산불 조사 실시 | 현행 법률 제21622호, 시행 2026-05-12 |
| 산불피해지 복구 | 「산림재난방지법」 제49조 | 산불로 피해가 발생한 산림재난피해지 복구, 원인제공자 비용 청구 가능 | 2026-05-12 개정으로 진화비용 청구 근거 추가 |
| 산불피해지 복원계획 | 「산림보호법」 제43조 | 산불피해지 복구 또는 산림복원계획 수립·시행 | 원문에 2026-02-01 폐지로 표기 |

> 주의: 이 매뉴얼 본문 일부에는 「산림보호법」 제43조 체계가 남아 있다. 2026-02-01 이후 실무 판단에서는 원문이 제시한 「산림재난방지법」 제48조·제49조 체계를 우선 확인해야 한다. 현행 조문 요약은 [[법령/산림재난방지법_산불조사복구]]를 함께 본다.

## 3. 피해지 유형

| 유형 | 정의 | 조사 방식 |
|------|------|----------|
| 재난성 피해지 | 산불피해가 지역주민 및 국가에 경제·사회적으로 큰 영향을 미쳐 국가재난으로 선포된 지역 | 시민·학계 등과 합동조사 |
| 일반 피해지 | 지역사회 이슈가 된 산불 피해지 및 소면적 산불 피해지 | 시·도, 시·군 또는 지방산림청·국유림관리소 자체조사 |

## 4. 복원·복구 개념

| 개념 | 목적 | 적용 방향 |
|------|------|----------|
| 산림복원 | 훼손된 산림생태계의 구조와 기능 회복 | 조림복원·생태복원·자연복원 포함 |
| 산림복구 | 훼손되었거나 훼손 우려가 있는 산림을 재해가 발생하지 않도록 안정화 | 재해방지 중심, 응급복구·항구복구 포함 |
| 응급복구 | 산사태·토사유출 등 2차 피해를 막기 위해 당해년도 장마 전 사방복구 실시 | 산불 진화 후 즉시 조사·실행 |
| 항구복구 | 중장기 계획에 따라 피해지를 지속적으로 복구 | 응급복구 완료 대상지도 항구복구 계획에 포함 |

## 5. 복원 방법별 기준

| 구분 | 적용 대상 | 핵심 판단 |
|------|----------|----------|
| 조림복원 | 피해정도가 중 또는 극심 이상이고 생태·환경·경제적 가치를 높일 수 있는 곳 | 산주의견과 임지 여건을 반영해 산림의 6대 기능을 빠르게 회복 |
| 자연복원 | 피해가 경미하거나 소생 가능, 임지생산력이 낮거나 암석지·석력지 등 조림이 곤란한 곳 | 자연적 회복 유도 |
| 생태복원 | 법정보호구역 등 생태계 원래 기능 회복이 중요한 곳 | 자생식물·자연재료 활용, 산림생태복원과 소관 |
| 위험목제거(긴급벌채) | 피해목 전도·절단으로 인명·시설물 2차 피해가 우려되는 곳 | 생활권·도로·관광지 등 보호대상 영향권 중심 |

> NDMS에서는 매뉴얼의 “조림복원”이 “복구조림” 공종명으로 표시된다.

## 6. 생태복원 우선 법정보호구역

다음 보호구역은 생태계·생물종 보호·보전을 목적으로 지정된 구역이므로 조림복원보다 생태복원 검토가 우선된다.

| 보호구역 | 근거 법령 | 지정 목적 |
|----------|----------|----------|
| 산림유전자원보호구역 | 산림보호법 | 식물 유전자·종 또는 산림생태계 보전 |
| 백두대간보호지역 | 백두대간 보호에 관한 법률 | 백두대간 생태계·자연경관·산림 보호 |
| 자연공원 | 자연공원법 | 자연생태계 또는 자연·문화경관 대표 지역 보전 |
| 야생생물 보호구역 | 야생동물 보호 및 관리에 관한 법률 | 멸종위기 야생생물 보호·번식 |
| 생태·경관보전지역 | 자연환경보전법 | 생물다양성·생태적 중요성·경관 가치 보전 |
| 특정도서 | 독도 등 도서지역의 생태계 보전에 관한 특별법 | 도서지역 우수 자연생태계·지형·자연환경 보전 |

## 1. 복원·복구계획 목적과 체계

**목적**: 산불피해지를 경제·생태·경관·환경적으로 가치가 높은 숲으로 조속히 복원·복구하여 지역주민 생활 안정과 국민 삶의 질을 높이는 것.

**기본 체계**

1. 산불 진화 후 즉시 토사유출·산사태 우려지 조사
2. 응급복구 실행
3. 피해면적·피해정도에 따라 재난성 피해지와 일반 피해지 구분
4. 피해상황 조사 결과를 토대로 연차별 복원·복구계획 수립
5. 복원·복구사업 완료 후 모니터링과 사후관리 실시

---

## 2. 대형산불 피해지 추진절차

| 단계 | 주요 행위 | 주체 |
|------|----------|------|

| 2 | 위험목제거사업 보조금 교부 | 산림청 → 광역지자체 → 기초지자체 |

| 4 | 피해지 복원·복구계획 용역 착수 | 기초지자체 |
| 5 | 정밀조사 실시 | 용역수행자 |
| 6 | 전문가·지역주민 의견수렴 | 기초지자체, 복원추진 협의회, 용역보고회 |
| 7 | 복원·복구계획 수립 완료 | 기초지자체 |
| 8 | 복원·복구 실시설계 | 기초지자체 |

## 3. 피해조사 기본방침

| 항목 | 기준 |
|------|------|
| 비용 규정 | 「사회재난 구호 및 복구비용 부담 등에 관한 규정」 적용·지원, 사유·공공시설 복구비는 「자연재난 구호 및 복구비용 부담 등에 관한 규정」 준용 |
| 조사 기준 | 지자체 자체 피해조사 결과를 토대로 민·관 합동조사, 허위·과장·누락 방지 |
| 사유시설 | 시·도 책임하에 지자체 조사, 조사·복구반은 정확성 검토 |
| 공공시설 | 시·도·지방청 입력 사항을 현장 확인 |
| 국고보조 | 공공시설은 피해액 3천만원 이상, 복구액 5천만원 이상인 경우 국고보조금 지원 |
| 지방이양시설 | 숲길, 등산로, 가로수, 공·사립 자연휴양림 등은 국비 지원 대상 아님 |

## 4. 조사 대상 구분

### 응급복구지

| 항목 | 기준 |
|------|------|
| 대상지 | 산불피해 후 집중호우 시 산사태·토사유출 등 2차 피해가 우려되는 지역 |
| 조사자 | 지자체·지방산림청 원칙, 필요시 외부 사방 전문가 공동조사 |
| 조사시기 | 산불 진화 후 즉시 |
| 조사항목 | 경사도, 모암, 수문학적 조건, 식생 피복도 등 사방사업 대상지 선정 항목 |
| 조사방법 | 산사태 예지 프로그램, 지형도 분석 등 실내작업과 현지조사 병행 |

### 복원·항구복구지

| 항목 | 기준 |
|------|------|
| 대상지 | 응급복구지를 포함한 산불피해지 중 계획적 복원·복구사업 필요지역 |
| 재난성 피해지 | 피해 규모·특성에 따라 민·관·학·연 공동조사단 구성 |
| 일반 피해지 | 시·도(시·군) 또는 지방산림청(국유림관리소) 주관 조사 |
| 조사분야 | 피해평가, 식생, 육상·수서생태계, 토양, 사회·경제적 특성 등 |
| 조사방법 | 항공사진·위성사진 분석 등 실내작업과 현장조사 병행 |

## 5. 산림자원 분야 중점확인

### 조림복원 조사

- NDMS 입력 대상: 피해면적, 단비, 피해액
- 피해액 산정을 위해 피해면적과 재적 확인 필요
- 위성영상 분석, 시·군 조사자료 비교, 현장 확인을 거쳐 피해면적과 재적을 최종 결정

### 조림복원 제외지

| 제외 유형 | 기준 |
|----------|------|
| 법정보호구역 | 국립공원 등은 생태복원으로 추진. 산림유전자원보호구역, 백두대간보호지역, 자연공원, 야생생물 보호구역, 생태·경관보전지역, 특정도서 포함 |
| 경미 피해지 | 산불피해 경지역 및 자연환경보전림은 자연복원 유도 |
| 조림불가지 | 암석지, 석력지, 피해 전 미입목지, 절험지 등 |

## 6. 위험목 제거사업 판단

### 산불피해지역 구분

| 구분 | 정의 |
|------|------|
| 산불피해지역 | 실제 산불이 발생해 피해가 발생한 전체 지역. 일반벌채지역과 위험목 제거사업 대상지역 포함 |
| 일반벌채지역 | 산불피해지역 중 위험목 제거사업 대상지역을 제외한 피해목 벌채지역 |
| 위험목 제거사업 대상지역 | 피해목이 쓰러지거나 부러져 인명·가옥 등 시설물 피해를 줄 수 있는 지역 |
| 2차피해영향권 | 산림경계에서 보호대상 방향 50m 이내인 지역. 주택·도로·주요 관광지 등 인명·재산피해 우려지역 |

### 위험목 대상지 선정 요령

| 기준 | 판단 |
|------|------|
| 공통기준 | 생활권으로부터 산림방향 수평투영거리 60m 또는 주계곡부 수평투영거리 30m 이내 |
| 제외 | 생활권·건물·도로·관광시설이 없거나, 피해등급이 경이거나, 임연부 입목이 피해를 입지 않은 경우 |
| 주계곡부 | 피해시점부터 계곡방향 최대 60m, 계곡 안쪽 피해 확인 시 포함 가능 |
| 유역 구분 | 유역면적에 따라 개소 단위로 피해등급 판정 가능. 능선 기준으로 구분하며 능선 너머 2차 피해가 없으면 제외 |
| 임연부 | 산림과 비산림 경계에서 산림 방향 30m 내외 |

### 위험목 제거사업지 외 일반벌채지역

| 기준 | 적용 |
|------|------|
| 피해등급 | 대분류 피해심(High), 피해중(Moderate) 지역 |
| 공통범위 | 위성영상 산불피해지역 중 위험목 제거사업 대상지 제외 지역 |
| 보호구역 | 법정보호구역 제외 |
| 지목 | 지적도상 지목이 “임”인 지역 |
| 산사태 우려 | 산사태위험등급 1등급 지역은 대상 유무 판정 가능. 2차 피해 우려가 없으면 일반벌채 제외 |

## 7. 산불피해강도와 면적 산정

### 피해강도 등급

| 대분류 | 세분류 | 기준 |
|--------|--------|------|
| 피해심(High) | 수관전소(Extreme) | 수관화, 100% 고사 |
| 피해심(High) | 수관열해(High) | 수관열해, 60% 초과 고사 |
| 피해중(Moderate) | 피해중 | 수관화와 지표화 혼재, 30~60% 생존 |
| 피해경(Low) | 피해경 | 지표화지역, 61% 초과 생존 |
| 미피해지(Unburn) | 미피해지 | 미피해 |

> 주의: 위성영상 기반 피해강도·피해면적은 현장조사 결과와 다를 수 있으며, 입목피해율과 같은 개념이 아니다.

### 피해면적 산정

- 실제 산불이 지나가 지상입목·관목·시초 등을 연소시킨 면적만 산불피해면적으로 산정
- 목측, 실측, 항공사진, 1:25,000 지형도 활용
- 소수점 둘째자리까지만 산정, 최소 보고면적 0.01ha
- 피해면적 10ha 이상은 GPS, 항공기, 드론으로 산출 가능

---

## 8. 피해재적 산정

### 산불관리통합규정 방식

| 항목 | 기준 |
|------|------|
| 자료 활용 | 산림조사부, 경영계획서상 재적, 항공사진, 유사 임분 시업지 조사재적 |
| 표준지 | 피해지 주변 미연소 임지 0.02ha(10m×20m) 표준지 3개소 이상 |
| 소면적 | 3개소 이상 표준지가 불필요하면 1개소 가능 |
| 조사대상 | 가슴높이지름 6cm 이상 입목 |
| 재적 단위 | ㎥, 소수점 첫째자리 |
| 6cm 미만 | 시초와 같이 속으로 표시. 1속은 1m 둘레 묶음 |

### 국가산림정보시스템 기반 방식

| 상황 | 산정 방법 |
|------|----------|
| 수목 존치(피해 중·경) | 표준지 조사로 피해 재적 산출. 대상 면적 2% 이상, 인공조림지·동령림은 1% 이상 원칙 |
| 수목 미존치 + 자료 있음(피해 심) | 산림조사부·경영계획서 기록, 항공사진, 유사임분 조사재적 적용 |
| 수목 미존치 + 자료 없음(피해 심) | 임업정보 다드림, 임상도 속성정보, NFI 평균 흉고직경·본수, 수고추정식, 대체 재적표 적용 |

### 기타 수종 대체 기준

| 임상도 수종 | 대체 적용 |
|------------|----------|
| 기타 침엽수종 | 중부지방소나무 |
| 기타 활엽수종 | 신갈나무 |
| 혼효림 | 중부지방소나무 50% + 신갈나무 50% |

## 9. 피해율과 피해액 산정

### 입목피해율

| 피해정도 | 피해정도(%) | 피해보고 적용비율 |
|----------|------------|------------------|
| 극심지역 | 71 이상 | 100% |
| 중 지역 | 41~70 | 50% |
| 경 지역 | 21~40 | 30% |
| 소생지역 | 20 이하 | 미적용 |

**재적 예시 공식**

```
ha당 재적 × 피해율 × 피해면적 = 피해조사 결과 재적
예: 100㎥/ha × 50% × 100ha = 5,000㎥
```

### 입목 피해액

| 대상 | 산식 |
|------|------|
| 지름 6cm 이상 | 기준가격(원/㎥) × 재적(㎥/ha) × 산불면적(ha) × 피해율(%) |
| 지름 6cm 미만 | 조림·육림비용(원/ha) × 산불면적(ha) |

| 수종 | 기준가격 |
|------|----------|
| 소나무 | 48,300원/㎥ |
| 잣나무 | 24,900원/㎥ |
| 낙엽송 | 28,700원/㎥ |
| 기타침엽수 | 10,900원/㎥ |
| 기타활엽수 | 6,340원/㎥ |

| 6cm 미만 비용 | 단가 |
|--------------|------|
| 조림비 | 11,205천원/ha |
| 육림비 | 27,415천원/ha = 풀베기 15,710천원 + 덩굴제거 9,356천원 + 어린나무가꾸기 2,385천원 |

## 10. 복구비 산정

| 구분 | 기준단가 | 적용구역 |
|------|----------|----------|
| 위험목제거(긴급벌채) | 31.7백만원/ha | 산불피해지 중 생활권 주변, 위험목제거사업 집행기준에 따른 지역 |
| 조림복원 | 15백만원/ha | 인공적 조림이 필요한 구역, 자연복원 가능 피해지는 제외 |
| 생태복원 | 48.39백만원/ha | 산림유전자원보호구역 등 생태계 보전 필요 구역 |

**산불피해액 및 복구비 산정 틀**

```
산불피해액 = 직접피해액 + 복원비용 + 진화비용 + 산림의 공익적 기능 손실액
```

- NDMS 피해액·복구액 입력과 직접 연관되는 항목은 직접피해액(입목피해액)과 복원비용
- 조림복원 대상면적 = 산불피해지 - 조림복원 제외지
- 민유림은 내년도 조림사업(산림재해피해지복구조림) 본예산 반영 지원
- 국유림은 내년도 조림사업(큰나무조림-산림재해방지조림) 본예산 반영 추진

> 2026-05-12 시행 「산림재난방지법」 제49조제2항 개정으로 원인제공자 비용청구 대상에 산불 진화비용이 추가되었다. 복구액·비용청구 판단에서는 [[법령/산림재난방지법_산불조사복구]]를 함께 확인한다.

## 11. 재해대장 및 NDMS 입력

### 공공시설 공통

- 소관에 따라 재해대장을 별도 생성
  - 국유림·국가관리 시설: 관리소 입력
  - 타부처 국유림: 지자체 입력
  - 공·사유림·지방관리 시설: 지자체 입력
- 피해 규모에 따라 시·군·구, 읍·면·동, 리 단위로 작성 가능

### 산림자원 분야 입력

| 공종 | 피해내역 | 복구내역 | 유의사항 |
|------|----------|----------|----------|
| 조림복원(NDMS: 복구조림) | 기타항목 선택 후 산림피해 수기 입력 | 복구조림 전부 입력 | 피해액과 복구비 모두 입력 |
| 위험목 제거 | 공란 | 긴급벌채 입력 | 피해액 미작성 |
| 생태복원 | 공란 | 생태복원 입력 | 피해액 미작성 |

### 기타 재해대장 작성 유의

- 사방 분야는 피해내역과 복구내역 모두 입력
- 공공시설 항목에 포함되지 않는 시설은 견적가로 입력
- 현장사진은 피해상황이 전반적으로 보이도록 원경 사진 반영, 광범위하면 드론 촬영 가능
- 피해·복구내역은 피해 및 복구단면도와 산출근거를 각각 해당란에 작성
- 조사반별 일일 조사 결과와 최종 확정물량은 팀장 주관으로 보고

## 1. 조림복원 개요

| 항목 | 기준 |
|------|------|
| 목적 | 산불로 훼손된 산림을 생태·경관·경제·환경적 가치가 높은 건강한 숲으로 회복 |
| 대상지 | 산불피해지 중 조림복원이 필요한 산림 |
| 추진절차 | 조사 → 응급복구 → 조림복원 → 모니터링 및 사후관리 |
| 기본 방향 | 산림의 6대 기능과 연계하여 피해정도·입지여건별로 복원 |

### 복원·복구 유형과 소관

| 구분 | 개념 | 소관부서 |
|------|------|----------|
| 응급복구 | 산불 이후 2차 피해 예방을 위한 사방 복구 | 산사태방지과 |
| 조림복원 | 산주의견과 임지 여건을 고려해 경제적·환경적으로 산림 기능을 회복 | 산림자원과 |
| 생태복원 | 자연회복력 기반으로 자생식물·자연재료 등을 활용해 생태계 기능 회복 | 산림생태복원과 |
| 자연복원 | 피해가 경미하여 자연적 회복을 유도 | 별도 소관 명시 없음 |

## 2. 표준 의사결정 기준

산불피해지 복원 표준 의사결정 흐름도는 초기 정책 결정을 지원하기 위한 기준이며, 복원·복구계획 수립 시 현장여건, 전문가 의견, 산주·주민 의견을 반영하여 조정할 수 있다.

| 고려요소 | 적용 기준 |
|----------|----------|
| 의견수렴 | 산주 또는 지역주민 의견을 설명회·공청회 등으로 계획 수립 전·중·필요시 수시 반영 |
| 보호구역 | 산림유전자원보호구역, 백두대간보호지역, 자연공원, 야생생물보호구역, 생태경관보전지역, 특정도서 등은 생태복원 우선 검토 |
| 피해등급 | 국립산림과학원 산불피해등급도상 심(수관화), 중(열해), 경(지표화) 적용 |
| 식생피복도 | 드론으로 필지별 조사. 봄철 산불은 7~8월, 그 이후 산불은 발생 후 3개월 이상 경과 시 조사 |
| 평균경사도 | 필지 단위 평균 경사도 적용 |
| 임지생산력 | 피해 이전 임지생산능력급수 적용. 필요시 판정표 활용 |
| 목재생산림 | 기능구분도상 목재생산림 |
| 2차피해우려지 | 산사태취약지역, 민가·도로·숲길·임도·보호시설 주변 수고 높이 이상 범위 |
| 움싹갱신 | [[행정규칙/훈령/지속가능한_산림자원_관리지침_일반지침]] 기준 적용 |
| 기대회복력 | 임지생산력, 모암, 지형, 경사, 토양수분·양분 등 토양질 요소를 종합 고려 |

### 복원 유형 판단

| 유형 | 적용 방향 |
|------|----------|
| 조림복원 | 산림용 묘목 또는 종자파종으로 복원 |
| 생태복원 I | 도복목·토사유출 등 2차 피해가 우려되는 지역에서 위험목 제거, 토사유출 방지 등 재해예방 중심 |
| 생태복원 II | 자연회복력과 기대회복력이 낮아 자연 식생 유입이 어려운 지역에서 피해목 제거, 토양안정, 식재 등 적극 복원 |
| 생태복원 III | 자연회복력은 낮지만 기대회복력이 높아 최소 관리로 식생 유입이 가능한 지역에서 피해목 존치, 파종, 천연하종 등 소극 복원 |
| 자연복원 | 자연력 복원을 원칙으로 하되 필요시 움싹본수 조절, 생태보완조림 가능 |

## 4. 조림 방법별 복원

### 식재조림

| 항목 | 기준 |
|------|------|
| 대상지 | 경제수종 대량생산 산림, 천연갱신 곤란 벌채지·미립목지, 토사유출·풍해 예방 특수수종 식재지 |
| 정방형 식재 | 5,000본/ha는 1.4m 간격, 3,000본/ha는 1.8m 간격 |
| 3본 군상식재 | 조림목간 0.6m, 식재군간 3.3m×3.0m |
| 5본 군상식재 | 조림목간 1.2m, 식재군간 4.1m |
| 2열 부분밀식 | 조림목간 1m, 식재군간 6.6m |

**식재시기**

| 지역 | 춘기 | 추기 |
|------|------|------|
| 온대북부(경기·강원) | 3월 하순~4월 하순 | 9월 하순~10월 중순 |
| 온대중부(전북·경북·충청) | 3월 중순~4월 중순 | 10월 상순~10월 하순 |
| 온대남부(전남·경남) | 3월 상순~4월 상순 | 10월 중순~11월 상순 |
| 난대(제주·남부해안) | 2월 하순~3월 하순 | 10월 하순~11월 중순 |

> 가을철 조림은 최소화하고, 적설량이 적거나 바람이 심한 지역에서는 가을철 식재를 지양한다.

### 파종조림

| 항목 | 기준 |
|------|------|
| 대상지 | 발아가 잘되는 수종, 식재조림 활착률이 낮은 급경사지 등 특수지역 |
| 수종 | 소나무·해송 등 침엽수, 가래나무·밤나무·상수리나무·굴참나무·졸참나무·갈참나무·신갈나무 등 |
| 시기 | 봄: 중부 4월 상순, 남부 3월 하순 / 가을: 10~11월 |
| 방법 | 지름 50~60cm 지피물 제거, 중앙 30~40cm 경운, 10cm 높이 상 조성, 2~10립 파종, 종자 지름 2~3배 복토 |

### 용기묘 조림

| 항목 | 기준 |
|------|------|
| 대상지 | 세근 발달이 좋지 않은 직근성 수종, 경사지 등 일반 조림 곤란 지역 |
| 시기 | 연중 가능하나 봄철 3~4월, 가을철 10~11월 권장 |
| 운반 | 운반 2~3일 전 물주기 중지, 바람 차단 후 현지까지 운반, 그늘 보관 |
| 식재본수 | 상수리나무·소나무 등은 3,000본/ha 기준, 현지여건에 따라 조정 |

### 맹아갱신

| 항목 | 기준 |
|------|------|
| 대상지 | 피해정도 경~중인 목재생산림 |
| 최소 요건 | 피해목 벌채 후 최소 직경 7cm 이상의 참나무류 그루터기 900개/ha 이상 균일 분포 |
| 그루터기 정리 | 지상 10cm 이내로 낮게 절단, 물 고임 방지 |
| 풀베기 | 토사유실로 인한 임지척박화를 고려해 원칙적으로 실시하지 않음. 보완조림 시 조림목 주변 풀베기 가능 |
| 움싹 조절 | 자연경합 후 우열이 분명해지는 2~3년 후, 11월~다음해 5월 실시 |
| 남길 움싹 | 상수리·굴참나무 1~2개, 신갈·갈참나무 2~3개 |
| 보완조림 | 3년 이내 움싹이 3,000그루/ha 미만이면 실시 |
| 수확 | 15~20년 후 직경생장량을 고려해 수확 |

### 재해피해지 조림복원 특이 기준

- 입지여건, 묘목수급, 산주 요구를 종합해 조림수종 선정
- 산불피해정도(심·중·경)를 고려해 자체 복원계획에 따라 조림복원과 자연복원을 병행
- 식재수종은 대묘 1,500본/ha 이상 큰나무를 원칙으로 하되, 현지여건과 묘목수급에 따라 소묘·관목류 병행 가능
- 소나무재선충병 반출금지구역은 「소나무재선충병 방제특별법」 제12조에 따라 소나무류 조림 금지

## 2. 2022년 강릉 산불 조림복원 계획 사례

### 큰나무 조림

| 항목 | 계획 기준 |
|------|----------|
| 조림면적 | 가옥, 시설지, 도로 등 현장조사 결과를 반영하여 결정 |
| 식재유형 | 식재유형별로 조림면적 구분 산출 |
| 수종 선정 | 대상지구분과 식재유형별 추천수종 중 강릉시 실정에 맞게 선택 |
| 경관·내화 요구지 | 가옥·도로 주변은 화목류 식재 |
| 토질 양호지 | 유실수를 입지·토양조건에 따라 선정 |
| 묘목 규격 | 산불피해지 큰나무 조림단비표에 따른 수고 2.5m 내외 분뜨기묘 |
| 조림시기 | 2023년 봄철 완료 계획 |
| 사업비 | 2022년 산불피해지 큰나무조림 추진계획의 단비표 일괄 적용 |

### 항구복원 조림

| 항목 | 계획 기준 |
|------|----------|
| 면적 산출 | 현장조사 결과를 바탕으로 자연복원지역과 제외지를 빼고 조림 가능 지역만 선발 |
| 수종 선정 기준 | 시설물 유무, 맞춤형 조림지도 추천수종, 접근성, 현장 확인 |
| 내화수종 우선순위 | 내화수림대 → 대규모 시설 인접지 → 소규모 시설 인접지 |
| 소나무류 우선순위 | 환경보전림 → 능선부 → 척박지(저수고) → 소나무 단순림 |
| 참나무류 우선순위 | 토양조건 양호지역 → 계곡부 → 임연부 |
| 유실수 우선순위 | 산주 요청지역 → 토양조건 양호한 평탄지 → 계곡부 |
| 조림시기 | 2023~2025년 3년간 실시 |
| 사업비 | 산불피해지 조림복원 단가표 ha당 15백만원 적용 |

### 묘목수급 및 사후관리

| 항목 | 기준 |
|------|------|
| 큰나무 묘목 | 강릉시 보유 현황과 나무시장 공급 활용 |
| 묘목구입 우선순위 | 동해안 지역 → 강원 도내 → 경기도 → 경상북도 → 충청남북도 → 전라북도 |
| 항구복구 묘목 | 강원도 내 지정양묘 우선, 없으면 타 지역 구입 |
| 조림예정지 덩굴걷기 | 2023~2025년, 남은 조림대상 면적의 10% 일괄 적용 |
| 조림지 사후관리 | 식재 후 3년간 풀베기 6회, 덩굴제거 2회 |
| 사후관리 기간 | 2023~2027년 |

## 3. 2022년 울진 산불 조림복원 계획 사례

### 복원구역 결정

| 구분 | 면적 | 비율 |
|------|------|------|
| 조림복원 | 6,900ha | 73.1% |
| 자연복원 | 2,540ha | 26.9% |

울진 사례는 의사결정 흐름도상 목재생산림만 인공복원구역으로 결정한 결과를 그대로 따르지 않고, 대면적 피해 특성과 주민·산주·전문가 의견을 반영하여 수원함양림, 산사태방지림, 자연환경보전림 일부를 조림복원구역에 추가 편입했다.

### 복원대상 구분

| 유형 | 기준 |
|------|------|
| 인공복원 | 피해 정도가 중 이상이며 조림·갱신으로 생태계 서비스와 경제적 가치를 높일 수 있는 곳 |
| 조림복원 | 원칙적으로 목재생산림 대상. 필요시 자연환경보전림을 제외한 5대 기능림도 흐름도에 따라 조림복원 |
| 송이산 복원 | 기존 송이 발생지를 대상으로 소나무 조림 또는 자연력 기반 소나무림 복원 |
| 단기소득 | 주민 의견을 수렴하되 토양조건, 사후관리, 산지전용 여부를 고려 |
| 내화수림 | 수관화·비산화를 줄일 수 있는 위치에 내화성수종 조림 또는 자연복원력 활용 |
| 자연복원 | 수관층 유지, 낮은 임지생산력, 급경사지 등 조림복원 외 지역 |

### 기능별 복원 방향

| 기능 | 계획 방향 |
|------|----------|
| 목재생산림 | 경제수, 밀원수, 복층림, 활엽수림, 특용수림으로 계획 |
| 생활환경보전림 | 경관수·보호림, 특용수 생산단지, 밀원수, 해안경관림, 내화수림 등으로 계획 |
| 산림재해방지림·수원함양림 | 활엽수 복층림, 해안경관림, 수원함양림 등으로 계획 |

> 울진 사례의 기능구분과 조림수종은 현장 또는 주민 의견에 따라 변동 가능성이 있음을 전제로 한다.

## 4. 2023년 홍성 산불 조림복원 계획 사례

### 계획 반영요소

| 요소 | 내용 |
|------|------|
| 입목벌채허가구역 | 산주의 조림신청수종 반영. 소나무·편백 신청이 많고 유실수·경관림 의견도 반영 |
| 산불 전 임상도 | 소나무·리기다소나무·곰솔 등 소나무과가 65.4%로 가장 많고, 상수리나무·참나무류·아까시나무 등 활엽수도 분포 |
| 맞춤형 조림지도 | 소나무, 곰솔, 굴참나무, 편백 등 적합 수종 확인 |
| 내화수림대 | 수관화·비산화를 줄일 수 있는 위치에 내화성 수종 조림 또는 자연복원력으로 활엽수림 유도 |
| 중요시설 주변 | 민가·중요시설 주변 식재 가능 지역은 내화수종 식재로 내화수림대 조성 |
| 능선부 | 산정에서 시작되는 능선부를 주된 내화수림대로 구획 |
| 방풍림 | 해안선 가까이 연속 설치, 끊어진 부분 없도록 계획. 주요 수종은 곰솔 |
| 경관림 | 해안가 주변 단일수종 활엽수림 계획 |

## 5. 사례 활용 시 유의점

- 강릉·울진·홍성 사례는 표준단가와 의사결정 기준을 현장 여건, 산주 의견, 묘목수급, 지역산림 특성에 맞춰 조정한 예시다.
- 사례 수종·면적을 그대로 적용하기보다 [[3장_조림복원_실행]]의 기능별 기준과 [[2장_복원복구계획_피해조사]]의 제외지 기준을 먼저 적용한다.
- 묘목수급, 사후관리 횟수, 단가 적용은 사업연도 고시와 예산편성 기준을 반드시 확인한다.

## 7. 서식 1: 산림피해지 조사(입목피해)

### 주요 입력 항목

| 구분 | 내용 |
|------|------|
| 기본정보 | 시군, 읍면, 리, 지번, 지적, 지목, 소유자 주소·성명 |
| 피해면적 | 지적면적, 전체 피해면적, 6cm 이상·미만 입목 피해면적 |
| 6cm 이상 입목 | 수종, ha당 본수·재적, 피해정도별 적용비율, 피해조사 결과 재적, 피해액 |
| 6cm 미만 입목 | 수종, 피해유형, 피해정도, 조림비, 육림비, 피해액 |

### 작성요령

- 지적은 토지대장상 면적으로 기재
- 지목은 피해지 내 임야인 필지만 입력
- 전체 피해면적은 도상 수평투영면적 기준이나 지적면적을 초과하지 않도록 입력
- 수종은 소나무, 낙엽송, 잣나무, 기타침엽수, 기타활엽수로 입력
- 필지별 면적이 4ha 이상이고 2개 이상 수종이 분포하면 수종별 면적 0.5ha 이상 집단화 시 구분
- 필지 면적 4ha 미만은 혼효율 낮은 수종이 25% 이상이면 구분, 25% 미만이면 우점 수종으로 일괄 작성
- 피해정도는 극심, 중, 경, 소생 중 하나만 선택
- 소나무·잣나무는 지표화가 있어도 지면 0.5m 이상 수간이 탄 경우 극심으로 작성하고 소생 입력 금지
- 낙엽송은 수관 전소 시 극심, 지표에서 수간 1m 이하 탄 경우 소생, 1m 이상 탄 경우 중 또는 경 입력
- 활엽수는 수관 전소라도 극심 입력 금지. 수관 전소는 중, 일부 피해는 경, 수간만 피해는 소생 입력
