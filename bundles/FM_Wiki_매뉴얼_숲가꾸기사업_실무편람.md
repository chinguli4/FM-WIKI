# 숲가꾸기사업(설계·감리·사업) 실무편람

> 출처: 2025 숲가꾸기사업(설계·감리·사업) 실무편람  
> 원본: `raw/manuals/2025_숲가꾸기사업(설계·감리·사업)_실무편람.md`  
> 성격: 산림청 숲가꾸기사업 담당자용 실무편람. 법령·훈령 원문이 아니라, 기존 지침·매뉴얼과 숲가꾸기사업 모니터링에서 확인된 문제점·개선사항을 행정 실무 관점으로 재정리한 자료.

## 왜 중요한가

이 편람은 국유림관리소·지자체 담당자가 숲가꾸기사업을 발주·감독할 때 실제로 빠뜨리기 쉬운 지점을 잘 짚는다.

| 구분 | 활용 가치 |
|------|-----------|
| 절차 | 계획, 설계, 사업시행, 감리의 흐름을 필요자료·주요행위·결과물·관련서식으로 정리 |
| 품질 | 표준지, 소반별 시방서, 작업지시도, 선목, 할인·할증, 잔존본수, 산물처리 등 부실 가능 지점 제시 |
| 안전 | 안전관리계획, 위험성 평가자료, 위험구역 표시, 보호구, 안전교육, 비상연락망, 현장점검 기록을 사업시행단계 자료로 제시 |
| 감리 | 설계 사전검토, 현장 감리, 시정조치, 보완시공 확인, 감리보고서 근거 확인을 담당자 책무와 연결 |
| 민원·사후관리 | 주민설명, 묘지·등산로·임도변 산물처리, 사후 모니터링과 행정기록 보관을 강조 |

## 발주관리 핵심 반영사항

### 1. 실시설계는 현장 위험을 도면과 성과품으로 고정하는 단계

설계자는 사업계획서·기초자료·법령지침을 검토한 뒤 현장 상세조사를 해야 한다. 현장조사에서는 경계, 수종·임령, 생육상태, 경사도, 장애물, 접근로, 재해·작업장 위험요소를 확인하고 사진·야장·도면으로 남겨야 한다.

관리소 담당자는 다음 질문으로 설계 성과품을 검토한다.

| 질문 | 확인 대상 |
|------|-----------|
| 급경사지, 암반, 낙석, 침수, 민가, 등산로, 계곡부, 위험목이 도면에 표시됐는가 | 현장조사야장, 위험요소 조사표, 작업지시도 |
| 작업동선·운반로·장비 진입로가 안전하고 실제 가능한가 | 작업지시도, 집재·운재 계획도, 사진대지 |
| 작업공정표와 안전관리계획이 실시설계 도서에 들어갔는가 | 설계설명서, 작업계획서, 작업원 운영계획서 |
| 소반별 현장 차이가 시방서와 단가산출서에 반영됐는가 | 소반별 시방서, ha당 단가산출서 |

### 2. 작업지시도와 표준지는 설계 품질의 1차 판별 지점

편람은 부실 설계의 반복 사례로 작업내용이 없는 작업지시도, 표준지 편중, 소반별 시방서·단가산출서의 일괄작성, 할인·할증 미적용을 제시한다.

관리소 담당자는 설계 검수 때 다음을 직접 보아야 한다.

| 항목 | 검토 포인트 |
|------|-------------|
| 표준지 | 도면·좌표·사진이 일치하는지, 도로·임도 주변 등 접근성 좋은 곳에 편중되지 않았는지 |
| 작업지시도 | 작업구역, 제외구역, 작업방법, 집재방향, 토장, 위험구역이 도면만으로 이해되는지 |
| 소반별 시방서 | 각 소반의 경사, 임상, 접근성, 잔존목, 산물처리 차이가 다르게 서술됐는지 |
| 할인·할증 | 경사도, 장애물 정도, 규격재 생산, 산물수집 조건 등이 실제 내역에 반영됐는지 |

### 3. 감리는 사업시행 후 서류검사가 아니라 설계 사전검토부터 투입해야 한다

편람은 감리자를 담당자의 조력자로 활용하고, 실시설계 완료 전에 감리 용역을 발주하여 설계도서를 사전 검토하게 하는 절차를 강조한다.

감리자에게 요구할 핵심 산출물은 다음과 같다.

| 단계 | 받아야 할 자료 |
|------|----------------|
| 감리 착수 | 감리 착수보고서, 감리원 자격·선임자료, 감리수행계획서 |
| 설계 사전검토 | 설계검토 의견서, 표준지·작업지시도·시방서 검토 결과, 안전·품질 위험 검토의견 |
| 현장 감리 | 감리일지, GPS 트랙 또는 현장 이동기록, 현장사진, 중간점검표, 시정지시서 |
| 보완 확인 | 보완시공 전후 사진, 시정완료 확인서, 감리자 확인의견 |
| 준공 | 감리완료보고서, 준공검사 의견, 자료 인계 목록 |

### 4. 사업시행 착수 전 안전자료는 형식 제출이 아니라 작업허용 조건이다

사업시행자는 착수계, 현장대리인 선임계, 작업계획서, 작업원 운영계획서와 함께 현장 안전관리계획을 제출해야 한다. 편람은 현장 안전관리 및 예방조치 단계에서 위험성 평가자료, 작업장 위치도, 안전교육 기록, 보험가입증명서, 입간판 설치 확인서 등을 필요한 자료·서식으로 제시한다.

관리소 담당자는 다음 자료가 제출·승인되기 전에는 현장 착수를 허용하지 않는 방식으로 관리한다.

| 자료 | 확인할 내용 |
|------|-------------|
| 안전관리계획서 | 위험구역, 안전구역, 비상통로, 집합장소가 지도에 표시됐는지 |
| 위험성평가표 | 급경사지, 낙석, 장비운용, 체인톱, 집재·운재, 등산로, 계곡부 위험이 작업단위별로 반영됐는지 |
| 안전교육 이수 기록 | 착수 전 교육, 작업별 안전수칙, 보호구 착용, 응급처치, 위험구간 출입제한 교육이 기록됐는지 |
| 보호구·장비 점검 | 헬멧, 장갑, 안전화, 고글, 안전로프, 체인톱·집재장비 점검이 실제 확인됐는지 |
| 보험·비상대응 | 산재보험 등 보험가입, 비상연락망, 구급장비, 보고체계가 갖춰졌는지 |

### 5. 설계와 다른 장비·집재 방식 변경은 공식 승인 없이는 허용하지 않는다

편람 Q&A는 설계서에 인력 집재로 되어 있는데 현장 여건상 소형 장비를 쓰려는 경우를 다룬다. 원칙은 설계도서와 다른 방식의 시공은 불가이며, 불가피한 변경은 감리원·감독자 합동 현장확인, 변경 타당성 검토, 안전성·환경영향 최소화 방안, 발주자 공식 승인을 거쳐야 한다.

솎아베기와 산물수집에서는 다음을 특히 본다.

| 변경 유형 | 관리소 확인 |
|-----------|-------------|
| 인력 집재 → 기계화 집재 | 장비 진입로, 토양답압, 잔존목 훼손, 계곡부 훼손, 작업반경 통제 검토 |
| 집재방향 변경 | 원목 굴러떨어짐, 하부 작업자 배치 금지, 와이어로프 내각 출입통제 확인 |
| 토장 위치 변경 | 민원, 등산로·임도 통행, 수해·유실 위험, 반출 동선 검토 |
| 작업로 신설·변경 | 설계변경 필요성, 복구계획, 산림훼손 최소화 확인 |

## 편람 기반 담당자 질문

현장에서 바로 쓰기 좋은 질문은 다음과 같다.

| 대상 | 질문 |
|------|------|
| 설계자 | 각 소반의 차이점이 시방서와 작업지시도에 어떻게 반영됐습니까? |
| 설계자 | 표준지가 임분을 대표한다고 판단한 근거와 GPS 좌표·사진은 무엇입니까? |
| 감리자 | 설계 사전검토에서 보완 요구한 사항과 최종 반영 여부는 무엇입니까? |
| 감리자 | “적정” 판단의 근거가 되는 표준지 조사결과, 현장사진, GPS 트랙은 어디에 있습니까? |
| 사업시행자 | 설계와 다른 작업방법·장비를 쓰려는 구간이 있습니까? 있다면 변경승인 절차를 거쳤습니까? |
| 사업시행자 | 오늘 작업구역의 위험성평가와 작업 전 안전회의 기록은 실제 작업자 참여로 작성됐습니까? |
