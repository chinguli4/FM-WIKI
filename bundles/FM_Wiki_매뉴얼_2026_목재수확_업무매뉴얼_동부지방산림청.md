# Ⅰ. 목재수확 업무절차와 적용범위

> 근거: 「국유림의 경영 및 관리에 관한 법률」 §9·§10·§27, 「산림자원의 조성 및 관리에 관한 법률」 §36·§36의4·§36의5

## 적용 경계

- 대상: 산림청 소관 국유림에서 국유임산물을 입목 상태로 매각하고 매수인이 벌채·반출하는 업무
- 주체: 지방산림청, 국유림관리소, 조사·설계 수탁자, 매수인
- 비대상: 공·사유림 일반 허가업무, 조림·숲가꾸기 도급사업 자체, 산불 등 긴급재난 벌채의 특례 판단
- 효력: 업무 참고자료. 법적 의무는 법률·시행령·시행규칙·행정규칙·계약서에서 확인

## 전체 흐름

1. 국유림경영계획과 사업이력으로 후보지 추출
2. 법적 제한지역·기능별 벌채금지구역 제척
3. 경계·기반시설·반출여건 사전 현장조사
4. 군상·수림대·산림영향권과 운재로 가설계
5. 주민설명회 및 이해관계 확인
6. 10ha 이상이면 전문기관 사전타당성조사
7. 20ha 이상이면 지방산림청 심의위원회 심의
8. 심의결과를 반영하여 경계·운재로·재적·품등·대금사정 인자 조사
9. 국가산림경영정보시스템에서 예정가격 산정·평정
10. 온비드 공고, 현장설명, 낙찰자 결정, 계약
11. 대금납부 확인 후 15일 이내 인도
12. 운재로 준공검사 후 벌채·반출, 수시점검
13. 반출완료 현장검사, 훼손·미반출·재해예방 확인, 계약종결

## 단계별 산출물

| 단계 | 핵심 산출물 | 다음 단계 통제점 |
|---|---|---|
| 후보지 | 경영계획·사업이력 조회결과, 후보지도 | 현차기·관리구역·사업종 조건 확인 |
| 제척·가설계 | 제한지역 중첩도, 벌채구역·잔존구역·운재로안 | 최신 공간정보와 현장 일치 |
| 타당성·심의 | 조사결과서, 심의안건, 의결서 | 조건부 의결사항 설계 반영 |
| 현장조사 | 경계·실측도, 매목·수고·품등 조사서 | 원자료와 시스템 입력값 대조 |
| 예정가격 | 인자조사서, 시가조사서, 사정표·평정서 | 품등·운반거리·운재로 비용 검산 |
| 계약·인도 | 공고문, 계약서, 인도증, 첨부도서 | 대부·사용허가 및 산지일시사용 면적 부기 |
| 반출·종결 | 변경계약, 운재로 준공서, 완료검사서 | 계약보증금 반환 전 현장검사 |

## 중단 조건

다음 중 하나라도 해소되지 않으면 다음 단계로 넘기지 않는다.

- 경영계획 미편성 또는 변경승인 미완료
- 제한지역·벌채금지구역 경계 불확정
- 10ha 이상 사전타당성조사 미완료
- 20ha 이상 심의 미완료 또는 조건부 의결 미반영
- 벌채구역·운재로 실측과 계약서 면적 불일치
- 원자료와 시스템 품등·재적·비용 입력값 불일치
- 운재로 준공검사 전 반출

# Ⅱ. 관련법령 및 친환경벌채

> 현행 확인: 2026-07-13. 법령과 행정규칙이 매뉴얼보다 우선한다.

## 근거 체계

| 영역 | 주된 근거 | 실무 적용 |
|---|---|---|
| 경영계획·목재생산 | 국유림법 §9·§10 | 경영계획 편성·변경 후 실행 |
| 국유임산물 매각 | 국유림법 §27·§28, 시행규칙 §27~§30·별표 2·별지 19호 | 매각방법·계약·해제·반환 |
| 벌채 허가·점검 | 산림자원법 §36 | 제한지역, 운반로 의제, 실행·복구 점검 |
| 대면적 벌채 | 산림자원법 §36의4·§36의5, 시행령 §43의2~§43의7 | 10ha 조사, 20ha 심의 |
| 벌채 세부기준 | 산림자원법 시행규칙 별표 3 | 기준벌기령·벌채면적·연접거리 |
| 친환경벌채 | 친환경벌채 운영요령 §3~§9 | 군상·수림대·산림영향권·사후관리 |
| 가격·반출 | 국유임산물 매각예정가격 사정기준 등 시행요령 | 조사·가격·유찰·운재로·반출기간 |
| 계약 | 국가계약법령 | 경쟁·수의·재입찰·재공고 |
| 운반로 | 산지관리법 §15의2·§38, 국유림법 시행규칙 §27⑤ | 일시사용·복구·계약서 부기 |

## 친환경벌채 적용 판단

친환경벌채 운영요령은 모두베기, 왜림작업, 불량림 수종갱신에 적용한다. 5ha 기준과 존치비율은 시행규칙 별표 3과 함께 읽는다.

| 항목 | 기준 |
|---|---|
| 하나의 벌채구역 | 최대 30ha |
| 군상·수림대 존치 | 벌채면적의 20% 이상 |
| 군상 | 1개 이상 권장, 최소 폭 50m 이상, 가급적 원형 |
| 수림대 | 가능한 최소 폭 20m 이상, 기존 임분과 연결 |
| 산림영향권 | 군상·수림대 경계에서 평균 수고만큼 |
| 합계 | 군상+수림대+산림영향권이 벌채구역의 50% 이상 |
| 배치 우선 | 8부 능선, 급경사지, 계곡부, 도로변, 임연부 |
| 연접벌채 | 5ha 이상 30m, 5ha 미만 20m; 조림 준공 후 4년 경과 예외 확인 |

## 사전점검

- 희귀·멸종위기 동식물과 서식처 조사, 발견 시 보호조치 반영
- 등산로·도로·주요 조망점의 경관 영향 확인
- 마을 상수원·저수지·농경지로의 토사유출 방지계획 수립
- 기능별 벌채금지구역, 계곡부, 암석지·석력지 제척
- 군상·수림대가 생태적으로 연결되는지 현장 확인
- 운재로가 수림대·계곡부를 불필요하게 단절하지 않는지 검토

## 실행·사후관리

- 잔존목 손상을 예방하고 손상목 발생 시 즉시 보고·수량조사·처리
- 조재부산물은 수집·활용을 우선하고, 존치 시 유실되지 않도록 정리
- 군상은 병해충 등 불가피한 경우 외에는 훼손 금지
- 벌채 전·중·후 경계, 산림영향권, 운재로 임의변경 여부를 확인
- 오벌·도벌은 피해수량을 조사하고 계약·형사 절차를 구분하여 처리

## 주의

`군상·수림대·산림영향권 50%`는 실제로 나무를 전혀 베지 않는 면적이 50%라는 뜻이 아니다. 시행규칙 별표 3의 존치면적 20%와 운영요령의 영향권 합계 50%를 구분해 도면과 면적표에 각각 표시한다.

# Ⅲ. 대상지 선정·사전타당성조사·심의

> 근거: 산림자원법 §36·§36의4·§36의5, 시행령 §41·§43의2~§43의7, 시행규칙 별표 3·별표 12

## 후보지 선정

1. 국가산림경영정보시스템에서 현차기 경영계획부를 조회한다.
2. 주벌·수종갱신·왜림 등 사업종과 관리구역을 조건으로 추가한다.
3. 평균임령이 국유림 기준벌기령 이상인 임분을 추출한다.
4. 기준벌기령 미달지는 가슴높이지름 24cm 이상 입목이 50% 이상인지 현장 확인한다.
5. 수종갱신은 별도 수종갱신 판정표에 따른 불량림 여부를 확인한다.
6. 기존 목재생산·숲가꾸기 이력과 중첩하여 중복 후보지를 제거한다.

> 시스템 조회값은 후보지 선별자료일 뿐 현장조사를 대신하지 않는다. 특히 평균경급 조건은 개별 입목 분포비율을 보증하지 않는다.

## 경영계획 통제

- 승인된 국유림경영계획에 편성된 사업인지 확인
- 미편성지라면 경영계획 변경승인을 받은 뒤 실행
- 벌채 장소·양·방법과 운반로 설치장소가 승인 내용과 맞는지 확인
- 변경 전 조사·설계가 필요하더라도 사업 실행은 승인 후에만 진행

## 제한지역 제척

공간정보와 현장을 함께 사용하여 다음을 확인한다.

- 채종림·시험림·입목벌채 제한지역
- 산림보호구역, 백두대간보호지역, 사방지
- 국립공원·보호지역·국가유산·군사시설 등 타법 제한
- 산사태·토사유출 우려지, 계곡부, 암석지·석력지
- 기능별 산림관리상 벌채 제한: 수원함양림·산지재해방지림·자연환경보전림·생활환경보전림 등
- 멸종위기종·희귀종 서식지와 생태·경관 민감지역

## 사전 현장조사

- 자연경계와 인공경계를 이용해 예정구역을 임시 구획
- 기설 임도·작업로·중토장·반출도로와 민가·농경지 확인
- 계곡·능선·급경사·불안정 사면을 도면에 표시
- 군상·수림대·산림영향권을 가설계하고 평균 수고를 확인
- 운재로 종단·횡단 기울기, 절·성토, 배수·복구 가능성 확인
- 주민 이용도로·상수원·농작물 피해와 민원 가능성 확인

## 10ha 이상: 사전타당성조사

시행령 §43의2의 `10만㎡ 이상`은 10ha 이상이다. 지방산림청장이 전문기관에 의뢰하며, 현행 전문기관은 한국산림재난안전기술공단·한국산지보전협회·한국산림기술인회다.

조사자료에는 사업 타당성, 법적 제한, 지속가능성, 경관·재해 영향, 운반로·복구, 주민·생태 영향과 도면을 포함한다. 결과서와 사업계획서·현황도면은 보관한다.

## 20ha 이상: 심의위원회

시행령 §43의6의 `20만㎡ 이상`은 20ha 이상이다. 산림청 소관 국유림은 지방산림청장이 위원장이며 위원장·부위원장을 포함해 10명 이내로 구성한다.

심의안건은 다음을 명확히 보여야 한다.

- 경영계획 부합 여부
- 법적 제한지역 제척 결과
- 생태·경관·재해를 고려한 벌채 디자인
- 군상·수림대·산림영향권 면적표
- 운재로 시설·복구계획
- 주민설명·의견수렴 결과
- 사전타당성조사 지적사항과 반영표

의결은 적격·조건부 적격·보류·부적격으로 관리한다. 조건부 의결사항은 `심의의견 → 설계도서 위치 → 반영 결과 → 확인자` 형식으로 추적한다.

> **정오 각주**: 원본 업무 흐름도에는 `20ah 이상`으로 되어 있으나 시행령 §43의6①1과 본문 §Ⅲ.4는 모두 20ha이므로 `20ha`로 정정했다.

## 주민설명회

법정 조사·심의와 별개로 반출로, 교통, 소음, 농작물, 상수원, 작업기간을 설명하고 의견·조치계획을 기록한다. 단순 개최 여부보다 `의견-조치-설계반영`의 추적성이 중요하다.

# Ⅳ. 현장조사·재적조사·입목처분 위탁

> 근거: 국유임산물 매각예정가격 사정기준 등 시행요령 §3·§12 및 별지, 산림기술법 §25·시행령 §15 별표 5

## 구역과 운재로 확정

- 사전타당성조사·심의 결과를 반영하여 벌채구역과 잔존구역 확정
- 경계목·표지·GPS 실측선이 도면과 일치하는지 확인
- 군상·수림대·벌채금지구역을 별도 레이어와 면적표로 관리
- 운재로는 기존 시설 활용을 우선하고 신설 길이·절성토를 최소화
- 벌채구역 밖 운재로 면적을 별도로 산출하여 계약서 부기자료로 사용
- 변경 가능성이 높은 구간은 대안노선과 변경 승인 절차를 설계서에 표시

## 재적조사

### 조사방법 결정

| 방법 | 적용 | 통제점 |
|---|---|---|
| 전수조사 | 정확성이 특히 필요한 구역·소규모·불균일 임분 | 누락·중복번호 방지 |
| 표준지조사 | 매뉴얼상 벌채 대상면적 5ha 이상에서 가능 | 임상별 대표성·배분 근거 |

표준지는 수종·영급·혼효율·경급·지형을 대표해야 한다. 위성영상·드론·현장 견취로 임상 비율을 정하고, 소규모 군상 수종은 별도 전수조사를 검토한다.

### 조사 묶음

1. 매목조사: 수종·경급·본수·표지
2. 수고조사: 경급별 대표목, 수고곡선과 평균수고
3. 조재율·품등조사: 공시목 안배, 벌채·조재 후 재장·말구직경·품등
4. 대금사정 인자: 집재·운반·운재로·시장거리·작업난이도

원자료, 집계표, 시스템 입력자료를 분리 보관하고 세 값이 일치하는지 검산한다.

## 조재율·품등조사 통제

- 공시목 안배조서와 실제 벌채 공시목의 수종·경급 일치
- 원목 재장·말구직경·수고의 현장 기록 보존
- 수고조사야장과 품등조사에서 측정한 수고 상호 대조
- 품등별 생산재적 합계와 입목재적·조재율 관계 검산
- 특수재는 일반 원목가격으로 자동 처리하지 말고 별도 시가조사

## 위탁 실시설계

- 매뉴얼 기준 유효기간: 1년
- 과업: 구역·재적·조재율·품등·운재로·대금사정 인자 조사와 설계도서 작성
- 산림기술자 배치는 산림기술법 시행령 별표 5의 **산림 조사사업** 기준으로 계약 시점의 현행표를 확인

매뉴얼 기재 등급은 다음과 같다.

| 면적 | 매뉴얼 기재 배치등급 |
|---|---|
| 5ha 이하 | 초급 |
| 5ha 초과~10ha 이하 | 중급 |
| 10ha 초과~20ha 이하 | 고급 |
| 20ha 초과 | 특급 |

> 위 표는 원문 재현이다. 계약 전 반드시 산림기술법 시행령 별표 5 최신본을 다시 확인한다.

## 위탁성과 검수

- 착수 전 기술자 자격·소속·실제 참여 확인
- 면적구간별 책임기술자 등급 확인
- 현장야장 원본과 최종 집계표 표본 대조
- 구역면적 = 벌채면적 + 잔존면적 관계 확인
- 운재로 길이·폭·면적·절성토가 도면·내역서와 일치
- 공시목 수량과 원가산출 수량 일치
- 조사일·도면 버전·성과품 제출일 기록
- 성과품 검수 없이 시스템 매각자료로 전환하지 않음

## 감사 핵심

면적에 맞지 않는 기술자 배치는 단순 서류 흠결이 아니라 설계의 적법성과 계약대가에 영향을 준다. 계약서의 배치기준, 실제 참여, 성과품 서명을 함께 확인한다.

# Ⅴ. 국유임산물 매각예정가격·시스템·계약

> 근거: 국유림법 §27·시행령 §27·시행규칙 별표 2, 국유임산물 매각예정가격 사정기준 등 시행요령

## 사전 준비

- 매목·수고·조재율·품등 원자료 확정
- 적용 재적표와 수종 대응 확인
- 분기별 원목시장가격 최신화, 특수재 별도 조사
- 집재·운반 방식, 중토장, 시장별 거리 조사
- 운재로 신설·보수 길이와 비용 산출
- 노임·기계경비·보험료·이윤·결손율·자금회수기간 기준일 확인

## 시스템 처리 흐름

1. 입목처분 신규등록: 대상지·경영계획·공간정보
2. 자재조사 보고: 조사방법, 수종·경급·본수·수고
3. 조재율·품등: 공시목 원자료 입력과 집계 확인
4. 운재로 설계: 기설·신설·보수, 벌채구역 밖 면적 구분
5. 시장가격·운반거리: 시장별 가격과 대·소운반비
6. 기본자료: 노임·기계경비·보험료·이윤·결손율
7. 평정: 평정액·평정일·평정관
8. 산물매각예정조서·평정서 출력 후 역검산

## 검산 관계

- 입목재적: 조재 전 수간재적
- 생산재적: 조재 후 판매 가능한 원목재적
- 조재율: 생산재적 ÷ 입목재적
- 평정서의 입목재적과 생산재적·조재율 관계 확인
- 시장가역산 시 모든 공제비용의 단위가 `원/㎥ 생산재적`인지 확인
- 운재로 비용은 길이 감소·변경 시 예정가격과 정산 영향 재검토

## 시스템 오류 예방

생산재 검수조서를 일괄 입력할 때 시스템이 공시목 품등비율 대신 원목규격 품등 최대값을 적용할 수 있다는 감사사례가 있다. 일괄 입력 후 수종별·경급별 품등비율을 원자료와 다시 대조한다.

## 매각 절차

가격평정 → 매각의뢰 → 공고 → 현장설명 → 낙찰자 결정 → 계약 → 대금고지·납부확인 → 산물인도 순으로 처리한다.

### 입찰 방식

- 원칙: 일반경쟁입찰
- 수의계약: 국유림법령 또는 국가계약법령의 해당 사유가 있을 때만
- 재입찰·재공고: 최초 가격·조건 변경 금지 원칙 확인
- 2회 이상 유찰: 시행요령 §15에 따른 감액 가능 여부와 신규 매각건 처리 확인

## 공고문 필수 고지

- 표준지조사 여부와 수량 산정 특성
- 입찰 전 현장 확인 책임
- 벌채구역·잔존구역·운재로와 제한사항
- 운재로 변경 사전협의·변경계약·준공검사
- 부산물 처리와 재해예방 조치
- 반출기간과 연장 요건

## 계약서 부기

임산물 운반로가 있으면 별지 제19호 국유임산물 매각계약서 §13에 다음을 명확히 적는다.

- 대부·사용허가 면적: 벌채구역 전체 + 벌채구역 밖 운재로 시설면적
- 산지일시사용 면적: 운재로 시설면적
- 위치·노선·폭·기간·복구조건
- 추가·변경 시 변경계약 의무

부기를 빠뜨리면 대부·사용허가 및 산지일시사용 의제의 근거가 불완전해진다.

# Ⅵ. 인도·반출기간·사후관리

> 근거: 국유림법 시행규칙 별표 2, 국유임산물 매각계약서 §2·§7·§12·§13, 국유임산물 매각예정가격 사정기준 등 시행요령 §12·§22

## 물건 인도

매각대금 납부를 확인한 뒤 매수인에게 인도를 통보하고, 통보일부터 15일 이내 산물 소재지에서 인도증을 교부한다. 수의계약의 일부 예외는 국유림법 시행령 §27을 확인한다.

인도증에는 다음을 붙인다.

- 벌채구역도 1/25,000 또는 1/50,000
- 벌채경계·면적이 표시된 실측도 1/6,000
- 운반로 설계도서
- 재적조서

## 인도 시 현장 설명

- 벌채경계, 수림대·군상, 벌채금지구역
- 운재로 노선·폭·배수·복구조건
- 소나무류 생산확인 등 반출서류
- 마을도로·교통·적정 차량·주민 유의사항
- 운재로 준공검사 전 반출 금지
- 오벌·도벌·잔존목 피해 발생 시 즉시 보고

## 반출기간

반출기간은 인도일 또는 채취허가일부터 시행요령 별표 7의 매각량별 기준에 따라 산정한다. 계약서·인도증·시스템의 시작일과 종료일을 일치시킨다.

### 연장

- 원칙적으로 요건을 충족하는 경우 1회
- 대부·사용허가 또는 임도 지장목: 해당 사업기간 범위
- 천재지변·재난·벌채지 복구 등 불가피한 경우: 당초 반출기간 범위
- 총 반출기간 중 재난이 2회 이상 발생한 경우: 규정상 추가 연장 가능 여부와 횟수 확인
- 연장 시 계약보증보험 기간도 함께 연장

단순 우천, 작업지연, 장비수급 사정만으로 자동 연장하지 않는다. 신청서, 현장확인, 기상·재난 자료, 미반출량과 귀책사유를 남긴다.

## 미벌채목 생장량 대금

매뉴얼은 당초 반출기간의 4월~10월 생장기간에 대해 다음 식을 제시한다.

`생장량 대금 = 1㎥당 단가 × 미벌채 재적 × 0.05 × 생장기간 해당 개월 수 ÷ 7`

1㎥당 단가는 매각 예정가격 단가와 계약단가 중 높은 가격을 사용한다. 다만 매뉴얼 자체가 반출연장기간 해석에 이견과 규제개선 검토가 있음을 적고 있으므로, 실제 부과 전 시행요령 최신본과 산림청 유권해석을 확인한다.

## 운재로 변경·준공

1. 변경 필요성·대안·환경영향 검토
2. 변경 전 현장확인과 실측
3. 계약서의 노선·산지일시사용 면적 변경
4. 예정가격에서 공제한 운재로 비용의 증감 검토
5. 절·성토·노폭·배수·복구 상태 준공검사
6. 준공검사 후 반출

설계보다 신설 길이가 감소하면 시장가역산 시 공제된 개설비용을 추가 징수해야 하는지 검토한다.

## 반출완료 검사

- 미반출 임산물·미벌채목 존재 여부
- 경계침범·오벌·도벌·잔존목 피해
- 불법 형질변경과 운재로 임의변경
- 계곡부·임도 배수로 내 부산물 적치
- 횡단배수로·측구·사면안정·파종 등 재해예방
- 계약서·설계도와 실제 운재로 비교
- 드론·GPS 기록과 현장사진 보존

완료검사와 손해·미이행 정산을 끝내기 전 계약보증금을 반환하지 않는다.

# Ⅶ. 주요 감사 지적과 재발방지

> 원본 HWPX에만 남아 있던 감사사례 이미지를 복원했다. 사례 속 관리소·개인 식별정보는 원본에서 이미 비식별화되어 있다.

## 1. 위탁사업 기술자 배치기준 미준수

![목재수확 위탁사업 기술자 배치기준 감사사례](assets/감사_01_위탁사업_기술자배치.png)

- 지적: 사업규모별 책임기술자 등급을 잘못 적용하고 설계서에 반영
- 위험: 과다 지급, 설계 적법성·품질 저하
- 통제: 계약 전 면적구간 판정표 작성 → 자격증·소속 확인 → 착수계 승인 → 성과품 서명 대조

## 2. 매각계약서의 대부·사용허가 및 산지일시사용 부기 누락

![매각계약서 부기 누락 감사사례](assets/감사_02_매각계약서_부기.png)

- 지적: 운반로에 필요한 국유림 대부·사용허가와 산지일시사용 면적·기간을 계약서에 부기하지 않음
- 위험: 운반로 시설의 법적 근거·복구책임 불명확
- 통제: 벌채구역 전체, 벌채구역 밖 운재로, 산지일시사용 면적을 구분한 부기 점검표 사용

## 3. 임목벌채 확인·점검 의무 소홀

![임목벌채 확인점검 감사사례](assets/감사_03_임목벌채_확인점검.png)

- 지적: 운반로 완료통보·준공검사·임목벌채 현장 확인 누락
- 위험: 무단 변경, 산림훼손, 복구 미흡, 계약보증금 부당 반환
- 통제: 완료통보 접수번호 → 준공검사서 → 벌채 중간점검 → 반출완료검사 순서 강제

## 4. 매각대금·계약보증금 관리 부적정

![매각대금과 계약보증금 감사사례](assets/감사_04_매각대금_계약보증금.png)

- 지적: 납부기한 또는 계약기간을 부적정하게 연장하고 보증보험 기간을 함께 연장하지 않음
- 통제: 연장결재 시 대금·계약기간·보증보험 만료일을 한 화면에서 확인하는 3일자 점검

## 5. 반출기간 연장 부적정

![반출기간 연장 감사사례](assets/감사_05_반출기간_연장.png)

- 지적: 기상증명 등 객관자료 없이 반복 연장하거나 기간 종료 후 다시 연장
- 통제: 만료 전 신청, 규정상 사유, 증빙, 미반출량, 귀책사유, 연장한도, 보증보험을 결재문에 의무 기재

## 6. 생산재 품등 입력 오류

![생산재 품등 입력 오류 감사사례](assets/감사_06_품등입력.png)

- 통제: 입력 후 원자료·집계표·평정서의 수종별 품등비율 3자 대조, 일괄입력 건 별도 표본검사

# Ⅷ. 친환경벌채 사례

## 우수 설계의 공통원칙

- 능선부·계곡부·산림습지 등 민감구역을 수림대로 연결
- 군상 위치를 획일적으로 두지 않고 산림영향권과 생태축에 따라 조정
- 기설 작업로를 활용하고 신설 운재로를 최소화
- 운재로가 계곡·수림대를 가로지르는 횟수 최소화
- 경관 노출이 큰 곳은 잔존대 폭과 배열을 조정
- 주민설명회에서 반출로·농작물·소음·교통 문제를 사전 해소
- 부산물은 수집·활용하고 농경지·계류로 유실되지 않도록 관리

원본의 우수사례 사진·도면에는 구체적 지역·공간정보가 포함되어 있어 기관명 외 식별정보를 남기지 않는 원칙에 따라 위키에는 원칙만 재구성했다.

## 현장관리 미흡사례

### 1. 계곡부 부산물 적치

![계곡부 부산물 적치 사례 1](assets/미흡_01_계곡부_부산물_1.png)

![계곡부 부산물 적치 사례 2](assets/미흡_01_계곡부_부산물_2.png)

집중호우 시 유목·배수막힘·토사유출을 일으킨다. 계류와 배수로에서 이격하고 유실방지 방향으로 정리한다.

### 2. 계곡부 무분별 벌채

![계곡부 벌채 미흡사례 1](assets/미흡_02_계곡부_벌채_1.png)

![계곡부 벌채 미흡사례 2](assets/미흡_02_계곡부_벌채_2.png)

계곡 수림대 단절은 수질·서식처·사면안정 기능을 동시에 약화한다. 벌채금지구역과 수림대 경계를 현장 표지하고 작업자에게 인계한다.

### 3. 천연림보육지 생산재 저가 매각

![천연림보육지 생산재 저가매각 사례](assets/미흡_03_저가매각.png)

일괄 펄프재 판정 전에 수종·경급·결함·용도별 품등을 다시 조사한다. 사진은 대경재·특수재 가능성을 현장에서 확인해야 함을 보여준다.

### 4. 무분별한 운재로 개설과 산림훼손

![운재로 산림훼손 사례 1](assets/미흡_04_운재로_산림훼손_1.png)

![운재로 산림훼손 사례 2](assets/미흡_04_운재로_산림훼손_2.png)

노선 임의변경, 과도한 절성토, 배수 미설치는 재해와 계약 위반을 동시에 초래한다. 변경 전 승인과 변경계약, 준공검사를 분리하지 않는다.

### 5. 사방시설 주변 벌채

사방시설의 집수·배수·토사저지 기능을 훼손하지 않도록 영향권을 별도 표시하고 관리기관과 협의한다. 구체적 위치가 드러나는 원본 도면은 수록하지 않았다.

### 6. 과도한 운재로 변경

계획노선보다 총연장이 크게 늘어나면 단순 현장조정이 아니다. 환경영향·산지일시사용 면적·비용·복구량을 재산정하고 변경계약을 체결한다. 구체적 노선이 드러나는 원본 도면은 수록하지 않았다.

## 현장 판정 질문

1. 이 노선·벌채가 계곡·능선·급경사의 보호기능을 약화하는가?
2. 기설 시설로 대체하거나 신설 길이를 줄일 수 있는가?
3. 부산물이 물길로 이동할 가능성이 있는가?
4. 수림대·군상·산림영향권이 도면뿐 아니라 현장에서도 연결되는가?
5. 변경 내용이 계약·면적·예정가격·복구량에 반영되었는가?

# 국유림 목재수확 통합 체크리스트

> 사용법: 각 단계 종료 전 `확인자·확인일·증빙 경로`를 기록한다. `해당 없음`은 사유를 적는다.

## A. 대상지 선정

- [ ] 현차기 국유림경영계획 편성 확인
- [ ] 미편성·변경 대상은 변경승인 완료
- [ ] 기준벌기령 또는 가슴높이지름 24cm 이상 50% 요건 현장 확인
- [ ] 수종갱신 판정표 적용 근거 확보
- [ ] 과거 목재생산·숲가꾸기·재해복구 이력 중첩 확인
- [ ] 최신 관리구역·사업종 조건으로 시스템 재조회

## B. 제한지역·사전조사

- [ ] 산림자원법상 제한지역 제척
- [ ] 산림보호구역·백두대간·사방지·국가유산·공원 등 타법 검토
- [ ] 기능별 산림관리상 벌채 제한 검토
- [ ] 희귀종·멸종위기종·서식처 조사
- [ ] 계곡·능선·급경사·암석지·석력지 현장 표시
- [ ] 주민도로·상수원·농경지·민원요인 확인

## C. 면적 분기

- [ ] 벌채구역·벌채면적·잔존면적을 구분 산출
- [ ] 10ha 이상이면 전문기관 사전타당성조사 완료
- [ ] 20ha 이상이면 지방산림청 심의 완료
- [ ] 조건부 의결사항을 설계 반영표로 추적
- [ ] 하나의 벌채구역 30ha 이하 확인

## D. 친환경벌채 설계

- [ ] 군상·수림대가 벌채면적의 20% 이상
- [ ] 군상+수림대+산림영향권이 벌채구역의 50% 이상
- [ ] 군상 최소 폭 50m, 수림대 가능한 최소 폭 20m 검토
- [ ] 8부 능선·계곡·급경사·도로변·임연부 우선 배치
- [ ] 연접벌채 거리와 조림 준공 후 4년 예외 확인
- [ ] 운재로가 수림대·계곡을 불필요하게 단절하지 않음

## E. 현장조사·위탁검수

- [ ] 경계·GPS 실측·도면 일치
- [ ] 조사방법과 표준지 대표성 근거 기록
- [ ] 매목·수고·조재율·품등 원자료 확보
- [ ] 공시목 안배와 실제 조사 일치
- [ ] 책임기술자 등급·소속·참여 확인
- [ ] 운재로 길이·폭·면적·절성토·배수 검토
- [ ] 원자료·집계표·성과품 표본 대조

## F. 예정가격·시스템

- [ ] 분기별 원목시장가격과 특수재 가격 최신화
- [ ] 입목재적·생산재적·조재율 관계 검산
- [ ] 수종별·경급별 품등비율 원자료 대조
- [ ] 시장거리·중토장·대소운반비 확인
- [ ] 운재로 비용의 단위·길이·생산재적 환산 확인
- [ ] 일괄입력 후 최대품등 자동적용 오류 확인
- [ ] 평정서 출력값을 원자료로 역검산

## G. 공고·계약·인도

- [ ] 공고문에 표준지조사·현장확인·운재로 조건 명시
- [ ] 재입찰·재공고 시 가격·조건 변경 금지 확인
- [ ] 계약서 §13에 대부·사용허가 면적 부기

- [ ] 인도증에 구역도·실측도·운재로도·재적조서 첨부

## H. 실행·반출·종결

- [ ] 운재로 준공검사 후 반출
- [ ] 변경노선은 사전승인·변경계약·비용조정

# 국가산림경영정보시스템 입력 주의사항

> 시스템 화면은 개편될 수 있다. 메뉴명보다 입력자료·검산 관계·결재 증빙을 기준으로 사용한다.

## 후보지 조회

- 관리구역과 사업종을 반드시 `조건추가`한 뒤 검색
- 현차기 선택 확인
- 평균경급 검색은 24~100cm, 혼효율 50~100% 등 원문 조건을 후보지 추출에만 사용
- 평균경급 24cm 이상 검색결과를 `24cm 이상 입목 50% 분포`로 간주하지 않고 현장 확인
- 경영계획부, 사업이력, 제한구역 레이어의 기준일 기록

## 공간정보

- 경영계획부와 소반도의 조인 누락·중복 확인
- 목재생산·숲가꾸기 등 기존 사업이력 중첩 제외
- 제한지역 레이어 최신성 확인
- 벌채구역·벌채면적·잔존구역을 서로 다른 필드로 관리
- 벌채구역 밖 운재로 면적을 별도 산출
- 군상·수림대·산림영향권 면적 합계와 도형 면적을 교차검산

## 자재조사 입력

- 조사방법, 조사면적, 표준지 수·면적, 확대계수 일치
- 매목 본수·수고·재적의 수종·경급 합계 일치
- 공시목 안배조서와 조재율·품등조사서 일치
- 원자료 수정 시 집계표와 시스템을 함께 갱신하고 버전 기록

## 품등 입력

> 감사 고위험 항목

- 일괄입력 전 공시목 품등비율 원자료 확정
- 일괄입력 후 최대품등 자동적용 여부 확인
- 수종별·경급별 품등비율을 원자료와 대조
- 생산재적 합계와 품등별 재적 합계 일치
- 평정서 출력 뒤 품등비율을 다시 역검산

## 운재로·비용

- 기설·신설·보수 운재로 구분
- 벌채구역 안·밖 면적 구분
- 중토장 운영 여부와 대·소운반 거리 입력
- m당 구축비를 ㎥당 비용으로 환산한 분모가 생산재적인지 확인
- 노선 감소·변경 시 공제비용과 추가징수 검토

## 가격·평정

- 원목시장가격의 조사분기와 적용일 기록
- 특수재 별도 시가조사
- 노임·유류·기계경비·보험료 기준일 통일
- 입목재적·생산재적·조재율 산식 검산
- 시장별 산출금액 중 적용시장 선택 근거 확인
- 평정일·평정관·첨부파일·기안문 누락 확인

## 전자계약·반출

- 계약서 부기 면적과 시스템 면적 일치
- 대금납부일·인도일·반출 시작일·종료일 일치
- 연장 시 계약·시스템·보증보험 종료일 동시 변경
- 운재로 변경계약과 준공검사 문서 연결
- 완료검사 후에만 계약종결·보증금 반환 처리
