# 산림경영계획서 작성기준

> 근거: 산림경영계획 작성 및 운영규정 별표 (산림청 훈령)

## 1. 현황도 작성 기준

### 1.1 산림현황도 (임상도)
- 기준 축척: 1/5,000 (임업용 산림지도 기반)
- 구분 항목:
  - 임상 (침엽수·활엽수·혼효림·무립목지)
  - 영급 (색상으로 구분)
  - 지위지수 (기호로 표기)
  - 특수림 구역 (보호구역 등)
- 작성 도구: GIS (ArcGIS, QGIS) 또는 산림청 산지정보시스템

### 1.2 경영계획도
- 기준 축척: 1/5,000
- 표시 항목:
  - 작업급 구획 및 번호
  - 임소반 경계
  - 임도·작업로 위치
  - 연도별 작업 예정지

### 1.3 임소반 구획
- **임반(林班)**: 약 100ha 단위의 산림 관리 구획
- **소반(小班)**: 임반 내 동질한 임분 단위 (최소 0.1ha)
- 소반 번호: 임반번호-소반번호 (예: 1-1, 1-2, 2-1)

## 2. 표준지 조사 원부 작성

### 2.1 표준지 조사 항목

```
[임분 조사]
- 위치: 소반번호, GPS 좌표
- 면적: 0.04ha (20m × 20m) 원칙
- 조사 일자, 조사자

[수목 조사 (흉고직경 6cm 이상 전수)]
- 수종
- 흉고직경 (DBH, cm)
- 수고 (H, m) — 우세목 10본 이상
- 수령 — 생장추 또는 수간석해

[지황 조사]
- 경사도 (°)
- 사면방향 (N/NE/E/SE/S/SW/W/NW)
- 해발고도 (m)
- 토양 깊이·질·배수
```

### 2.2 재적 산출

**단목재적 계산**
```
V = π/4 × DBH² × H × f
(V: 재적(㎥), DBH: 흉고직경(m), H: 수고(m), f: 형수)
```

수종별 형수:
| 수종 | 형수(f) |
|------|---------|
| 소나무 | 0.42~0.46 |
| 잣나무 | 0.44~0.48 |
| 낙엽송 | 0.46~0.50 |
| 참나무류 | 0.48~0.52 |

> 실무에서는 **국립산림과학원 수확표(재적표)** 사용 권장

**ha당 재적 환산**
```
ha당 재적(㎥/ha) = Σ단목재적 × (10,000 / 표준지 면적)
```

### 2.3 생장량 산출
- **정기평균생장량**: (현재 재적 - 과거 재적) ÷ 기간
- **연년생장량**: 현재 재적 - 전년도 재적
- 실무: 수확표의 지위지수별 생장량 곡선 활용

## 3. 수종별 주요 지위지수 기준

| 수종 | 우량 (S.I.) | 중간 (S.I.) | 불량 (S.I.) |
|------|------------|------------|------------|
| 소나무 | 18 이상 | 14~18 | 14 미만 |
| 잣나무 | 18 이상 | 14~18 | 14 미만 |
| 낙엽송 | 22 이상 | 16~22 | 16 미만 |
| 참나무류 | 20 이상 | 15~20 | 15 미만 |

> S.I. = 50년생 우세목 평균 수고(m)

## 4. 연년 작업 계획 작성 양식

```
연도 | 작업종 | 소반 | 면적(ha) | 수량 | 비고
-----|--------|------|----------|------|----
1년차 | 솎아베기 | 2-3 | 3.5 | 200㎥ |
1년차 | 조림(낙엽송) | 1-2 | 2.0 | 6,000본 |
2년차 | 풀베기 | 1-2 | 2.0 | 2.0ha |
...
```

## 5. 경영 목표 기술 기준

### 5.1 목표 유형
| 목표 유형 | 세부 내용 |
|-----------|-----------|
| 목재 생산 | 연간 목재 생산량(㎥), 주요 수종 |
| 탄소흡수 | 탄소흡수원 인증 획득, 흡수량(tCO₂/년) |
| 생태계서비스 | 수원함양, 토사유출 방지, 생물다양성 |
| 산림복지 | 산림치유, 휴양, 경관 |
| 복합 경영 | 임산물 채취, 임간축산, 수액 채취 |

### 5.2 목표 기술 방법
- 구체적·측정 가능하게 기술
- 예: "10년간 낙엽송 우량재 500㎥ 생산"
- 예: "탄소흡수원 인증을 통해 연간 50tCO₂ 흡수 확보"

## 6. 서식 및 제출 형식

- 경영계획서: A4, 표지 포함 (인가기관 양식 우선)
- 도면: A3 이상 출력 또는 GIS 파일 (SHP) 함께 제출
- 전산 등록: 산림청 산림경영계획 정보시스템 (FMIS) 입력 의무
  - 국유림: 의무
  - 공·사유림 인가: 의무
  - 사유림 비인가: 선택

## 7. 주요 오류 유형 (반려 사유)

| 오류 유형 | 설명 |
|-----------|------|
| 면적 불일치 | 임야대장 면적 ≠ 계획서 면적 |
| 재적 과대 산정 | 표준지 수 부족, 표준지 대표성 없음 |
| 수확량 초과 | 연년벌채량 > 생장량 |
| 도면 불일치 | 소반 경계가 현장과 다름 |
| 서명 누락 | 산림경영기술자 서명 없음 |

# 산림경영계획 지침

> 근거: 산림자원의 조성 및 관리에 관한 법률 제14조, 산림경영계획 작성 및 운영규정 (산림청 훈령)

## 1. 개요

산림경영계획이란 산림 소유자 또는 경영자가 산림을 지속적으로 경영하기 위해 수립하는 10년 단위의 종합 계획이다.

### 법적 근거
- **의무 대상**: 국유림, 공유림
- **임의 대상**: 사유림 (단, 50ha 이상은 산림청 인가 권장)
- **계획 기간**: 10년 (5년마다 중간 점검)

### 인가의 이점 (사유림)
- 조림·숲가꾸기 보조금 우대 지급
- 벌채 허가 절차 간소화
- 산림경영자금 융자 우선 지원
- 세금 감면 혜택 (산지세 등)

## 2. 계획 수립 절차

```
1. 산림 현황 조사
        ↓
2. 경영 목표 설정
        ↓
3. 작업급 편성
        ↓
4. 수확조절 계획 수립
        ↓
5. 연년 작업 계획 작성
        ↓
6. 인가 신청 및 인가
        ↓
7. 계획 실행 및 모니터링
        ↓
8. 중간 검토 (5년 후)
        ↓
9. 차기 계획 수립
```

## 3. 경영계획서 구성

| 장 | 내용 |
|----|------|
| 1장 | 사업지 개황 (위치, 면적, 행정구역) |
| 2장 | 산림 현황 (임상도, 영급, 재적) |
| 3장 | 경영 목표 및 방침 |
| 4장 | 작업급 편성 |
| 5장 | 수확 계획 (연년벌채량) |
| 6장 | 갱신 계획 (조림·천연갱신) |
| 7장 | 보육 계획 (숲가꾸기) |
| 8장 | 보호 계획 (산불·병해충) |
| 9장 | 연년 작업 계획 (10년 스케줄) |
| 부록 | 산림현황도, 경영계획도, 표준지 조사 원부 |

## 4. 인가 신청

### 4.1 신청 기관
- 국유림: 국유림관리소
- 공·사유림: 지역 산림청 또는 시·군·구청

### 4.2 제출 서류
- 산림경영계획서 (정본 1부, 사본 2부)
- 임야대장 등본
- 토지이용계획확인원
- 산림현황도 (임상도)
- 표준지 조사 원부

### 4.3 인가 처리 기간
- 접수 후 30일 이내 (보완 요청 시 기간 정지)

## 5. 산림 현황 조사 방법

### 5.1 임상 구분
| 임상 | 기준 |
|------|------|
| 침엽수림 | 침엽수 재적 비율 75% 이상 |
| 활엽수림 | 활엽수 재적 비율 75% 이상 |
| 혼효림 | 침·활엽수 비율 각각 25~75% |
| 무립목지 | ha당 30본 미만 (또는 재적 5㎥ 미만) |

### 5.2 영급 구분
| 영급 | 임령 |
|------|------|
| Ⅰ영급 | 1~10년 |
| Ⅱ영급 | 11~20년 |
| Ⅲ영급 | 21~30년 |
| Ⅳ영급 | 31~40년 |
| Ⅴ영급 | 41~50년 |
| Ⅵ영급 이상 | 51년 이상 |

### 5.3 지위지수 (Site Index)
- 수종별 우세목의 50년생 수고(m)로 표현
- 우량지 (S.I. 20 이상) / 중간지 (S.I. 14~20) / 불량지 (S.I. 14 미만)

## 6. 작업급 편성

### 6.1 작업급의 개념
동일한 수종·작업종·윤벌기로 관리되는 임분의 집합

### 6.2 편성 기준
| 구분 | 기준 |
|------|------|
| 수종 | 주수종별 구분 |
| 작업종 | 교림작업·왜림작업 구분 |
| 윤벌기 | 목표 벌채 임령 설정 |
| 작업급 면적 | 가급적 10ha 이상 |

### 6.3 주요 수종별 윤벌기 기준

| 수종 | 표준 윤벌기 |
|------|------------|
| 소나무 | 60~80년 |
| 잣나무 | 70~100년 |
| 낙엽송 | 40~50년 |
| 편백 | 60~80년 |
| 참나무류 | 70~100년 |
| 백합나무 | 30~50년 |

## 7. 계획 변경

- **경미한 변경**: 작업 시기 조정 (±1년), 사업량 10% 이내 변경
- **중요 변경**: 수확량 10% 이상 변경, 수종 변경, 작업급 재편성 → 재인가 필요
- 변경 신청: 사유 발생 후 30일 이내

## 8. 점검 및 보고

- 연도별 실행 실적 보고: 매년 1월 말 (전년도 실적)
- 5년 중간 점검: 계획 달성도 평가 및 필요 시 수정
- 인가기관의 현장 확인: 필요 시 수시

# 수확조절 지침

> 근거: 산림자원의 조성 및 관리에 관한 법률, 산림경영계획 작성 및 운영규정

## 1. 수확조절의 개념

수확조절(收穫調節)이란 산림의 지속가능한 경영을 위해 연간 벌채 가능량을 산정하고 계획적으로 수확하는 방법이다.

**핵심 원칙**: 연간 벌채량 ≤ 연간 생장량 (지속가능성 보장)

## 2. 주요 수확조절 방법

### 2.1 면적법 (Area Method)

윤벌기 동안 매년 동일한 면적을 벌채하는 방법.

```
연년벌채 면적 = 총 작업급 면적 ÷ 윤벌기
```

**예시**
- 소나무 작업급 면적: 720ha
- 윤벌기: 60년
- 연년벌채 면적 = 720 ÷ 60 = **12ha/년**

**장점**: 계산 단순, 공간적 작업 분산
**단점**: 임분 생장 차이 미반영 → 수확량 불균등

### 2.2 재적법 (Volume Method)

연간 생장량을 기준으로 벌채량을 결정하는 방법.

```
연년벌채 재적 = 총 생장량 (정기평균생장량 기준)
```

**예시**
- 작업급 총 재적: 10,000㎥
- 윤벌기: 50년
- 연년생장량: 3.5㎥/ha/년 × 200ha = 700㎥/년
- 연년벌채 재적 = **700㎥/년**

**장점**: 생장량 기반, 현실적
**단점**: 정기적 임분 조사 필요

### 2.3 수확표법 (Yield Table Method)

수종별 수확표(생장표)를 이용하여 지위지수·임령별 재적을 산출하고 벌채량을 결정.

```
연년벌채 재적 = Σ(소반별 ha당 재적 × 소반 면적) ÷ 윤벌기
```

**국립산림과학원 수확표 활용**
- 수종·지위지수·임령 → ha당 재적 조회
- 우리나라 주요 수종 모두 구비

**장점**: 표준화된 방법, 정밀도 높음
**단점**: 현실 임분과 오차 발생 가능

### 2.4 생장률법 (Growth Percentage Method)

현존 재적과 생장률을 이용하여 수확량을 결정.

```
연년벌채 재적 = 현존 재적 × 생장률(%)
```

**예시**
- 현존 재적: 15,000㎥
- 생장률: 4%/년
- 연년벌채 재적 = 15,000 × 0.04 = **600㎥/년**

## 3. 수확조절 방법 선택 기준

| 산림 유형 | 권장 방법 |
|-----------|-----------|
| 동령 단순 인공림 | 면적법 또는 수확표법 |
| 다령 혼효림 | 재적법 또는 생장률법 |
| 소규모 사유림 | 재적법 (단순 적용) |
| 정밀 경영 필요 | 수확표법 + GIS 통합 |

## 4. 연년벌채량 산출 절차

```
Step 1. 임분 현황 조사
        → 소반별 수종, 임령, 재적, 지위지수

Step 2. 윤벌기 설정
        → 경영 목표에 따른 수종별 윤벌기

Step 3. 작업급 편성
        → 동일 조건 소반 묶음

Step 4. 수확 가능량 계산
        → 선택한 수확조절 방법 적용

Step 5. 생장량과 비교
        → 벌채량 ≤ 생장량 확인

Step 6. 연년 작업 계획 반영
        → 경영계획서 제9장 작성
```

## 5. 간벌 수확량 산정

간벌(솎아베기)은 주벌과 별도로 수확량을 산정한다.

```
간벌 수확량 = Σ(간벌 소반 ha당 재적 × 간벌율 × 소반 면적)
```

**간벌율 기준**
- 경도 간벌: 현존 재적의 20~25%
- 중도 간벌: 현존 재적의 25~35%
- 강도 간벌: 현존 재적의 35~40%

## 6. 지속가능성 검토

### 6.1 가이드라인
- 연간 총 벌채량(주벌+간벌) ≤ 연간 총 생장량
- 작업급 내 영급 배치가 균형 잡힐 것 (편중 지양)
- 10년 계획 종료 시 재적이 계획 시작 시점 이상 유지

### 6.2 영급 구조 점검

이상적인 영급 배치 (정상림):
```
영급   : Ⅰ  Ⅱ  Ⅲ  Ⅳ  Ⅴ  Ⅵ
면적비 : 균등 배분 (각 영급 약 1/6씩)
```

현실 임분 진단:
- 특정 영급 편중 → 수확량 불균등 위험
- 고령급 과다 → 단기 수확량 과대 후 공백
- 저령급 과다 → 향후 수확량 부족

### 6.3 탄소흡수 고려
- 벌채 후 재조림 시 탄소 재흡수 기간 반영
- 탄소흡수원 인증 목적 시: 연간 순흡수량 유지 의무
