# 국유재산법 — 산림청 소관 재산 적용 핵심

> 소관: 재정경제부  
> 산림 적용 맥락: 국유림법이 특별히 규정하지 않은 국유재산의 취득·관리·사용허가·대부·처분·대장·변상금에 적용  
> 현행 기준: 법률 제21065호, 2026-01-02 시행 기준

## 적용 구조

국유림법 §16③은 보전국유림을 행정재산, 준보전국유림을 일반재산으로 본다. 국유림에 대해서는 국유림법의 특별규정을 먼저 적용하고, 정하지 않은 사항은 국유재산법의 일반규정을 확인한다.

## 핵심 조문군

| 조문군 | 산림 실무 |
|---|---|
| §6 | 행정재산·일반재산 구분 |
| §8~16 | 소관·취득·기부채납·등기·관리전환 |
| §18 | 영구시설물 축조 제한 |
| §27~29 | 행정재산 관리·교환·양여·관리위탁 |
| §30~39 | 사용허가·방법·사용료·감면·기간·취소·청문·원상회복 |
| §40 | 용도폐지 |
| §41~55 | 일반재산 처분·대부·매각·교환·양여 |
| §66~71 | 대장·가격평가·결산·실태조사 |
| §72~73 | 변상금·연체료 |

## 사용허가와 대부

- 행정재산: §30 이하의 사용허가
- 일반재산: §46 이하의 대부
- 국유림은 국유림법 §21~26의 목적·기간·요율·취소 등 특별규정을 먼저 확인

## 변상금

§72는 무단점유자에게 원칙적으로 사용료·대부료의 120% 상당 변상금을 부과한다. 변상금은 사용권원이 아니며 반환·철거·복구와 별개다.

## 시행예정

법률 제21344호는 2026-08-20부터 일정 규모 이상 국유재산 처분의 국회 소관 상임위원회 보고를 신설한다. 시행 후 매각·교환·양여 사건은 후속 시행령을 확인한다.
