# 조달청 시설공사 적격심사세부기준

> 소관: 기획재정부 > 조달청
> 산림 적용 맥락: 산림청·지방산림청·국유림관리소가 **조달청을 통해** 시설공사(임도공사·산림토목·사방사업·산림보호시설 등) 발주 시 직접 적용. 산림청 자체 발주 시 예규 728호와 비교 기준.
> 종류: 지침 (조달청 제3621호, 2026-04-23 일부개정, 시행 2026-04-30)

## 1. 목적과 위치 (제1조)

계약예규 「적격심사기준」에 의거 **조달청이 집행하는 시설공사 계약**에 적용할 적격심사 세부기준.

---

## 2. 공사규모별 평가기준 (제2조)

| 추정가격 | 평가기준 |
|---------|---------|
| **100억원 미만 50억원 이상** 공사 | [별표 1] |
| 50억원 미만 **10억원 이상** 공사 (전기·정보통신·소방·국가유산은 50억 미만 3억 이상) | [별표 2] |
| **10억원 미만 3억원 이상** (건설산업기본법상 건설공사) | [별표 3] |
| 3억원 미만 **2억원 이상** (전기 등은 3억 미만 8천만원 이상) | [별표 4] |
| 2억원 미만 (전기 등은 8천만원 미만) | [별표 5] |

→ 산림사업 적용:
- 임도공사·사방사업 대부분은 **별표 2·3** (10억 미만 3억 이상) 적용
- 대형 산림복구 공사는 별표 1·2 적용

### 심사기준일

입찰공고일 (다르게 정한 경우 제외).

---

## 3. 시공경험·경영상태 평가자료 (제2조 ③항)

### 자료 출처

- **관련협회** (대한건설협회·대한전문건설협회·한국전기공사협회 등) 통보 또는 업체 직접 제출 → 국가종합전자조달(G2B)시스템에 등록
- **최근 5년간 시공경험·기술능력·시공평가결과·경영상태** 평가
- 시공능력공시액은 관련협회 증명 자료

### 경영상태 평가 (5호)

| 평가 방식 | 기준 |
|----------|------|
| 신용평가등급 | 신용정보업자가 입찰공고일 이전 평가한 **유효기간 내 등급** 중 가장 최근 (회사채·기업어음·기업신용평가등급) |
| 재무제표 | 적격심사기준 [별표] '2' 결산서 (관련협회 통보 또는 조달청 직접 제출) |
| 추정가격 100억 미만 50억 이상 공사 | **감사인의 회계감사** 후 감사보고서 또는 한국공인회계사회 검토보고서 첨부 |

**감사의견에 따른 감점** (추정가격 100억 미만 50억 이상):
- 한정의견 → 경영상태 평점에서 **5/100 감점**
- 부적정의견·의견거절 → **10/100 감점**

### 종합건설·전문건설 상호시장 진출 (제6항)

종합건설사업자가 전문공사에 참가하거나 전문건설사업자가 종합공사에 참가하는 경우:
- 시공경험: 국토교통부고시 「종합·전문업종간 상호시장 진출을 위한 건설공사실적 인정기준」 적용
- 경영상태: 종합/전문건설사업자의 가중평균비율 별도 적용

---

## 4. 신인도 평가 (제2조 ④항)

- 평가자료: G2B시스템 등록 자료 + 입찰자 직접 제출 자료
- 신인도 평가는 해당 공사규모별 평가기준표 [별표 1] ~ [별표 5]

---

## 5. 공동수급체 평가 (제3조)

### 시공경험·경영상태·신인도 평가

- **시공경험**: 구성원별 시공경험 × 시공비율 합산
- **경영상태·신인도**: 구성원별 산출 점수 × 시공비율 합산
- 등급별 유자격자명부 운영 시 별도 처리

### 시공비율 산정

| 평가대상 업종 | 시공비율 산정 방식 |
|-------------|-----------------|
| 건설산업기본법 적용 공사, 추정가격 50억↑(전문공사 5억↑) | **시공능력평가액 / 평가대상 업종 추정금액** (지분율 초과 불가) |
| 토목건축공사업 등록자 | 평가대상 업종 토목·건축분야 시공능력평가액 기준 |
| 전기공사·정보통신·소방 등 5억↑ | 시공능력평가액 / 평가대상 업종 입찰가격 |
| 그 외 | 평가대상 업종 공사참여 **지분율** |

### 지역업체 가산평가 (제3항)

- 공사현장 시·도 본사 둔 지역업체 → 가산평가 [별표 17]
- 단독 입찰 시 제외, 상호출자제한기업집단 계열회사 간 공동수급체 제외

### 평가 제외 (제4항)

다음 경우 평가 제외 + 시공비율 다른 구성원에 배분 불가:
1. 평가대상 업종 외 업종만 공동도급 참여
2. 평가대상 업종 시공비율이 **10% 미만**

## 6. 적격심사 세부절차 (제4조)

### 1단계: 예상종합평점 평가 (제1항)

- 최저가 입찰자부터 수행능력 + 입찰가격 + 자재·인력조달가격 적정성 + 하도급관리계획 적정성 평가
- 예상종합평점이 적격통과점수(**95점 이상**) 미달 시 평가점수 통보 → **3일 이내 이의신청** 가능

### 2단계: 적격심사대상자 선정 (제2항)

예상종합평점 95점 이상 최저가 입찰자를 대상자로 선정.

**추정가격 50억 이상 공사** 제출 서류:
- 하도급관리계획서 [별표 13]
- 자재·인력조달가격 평가서류 (100억 미만 50억 이상 한정)
- 경영상태 평가서류 (100억 미만 50억 이상 한정)
- 표준하도급계약서 등 사용계획서 (신청 시) [별표 18]
- 신인도 평가자료 (신청 시)

**제출 기한**: 통보일로부터 **7일** (경영상태 평가서류는 10일).

### 3단계: 결격사유 확인 (제3항)

기술자·자본금·사무실·시설·장비 보유 미달 여부 확인. **7일 이내** 서류 제출.

### 4단계: 종합평점 산출 (제5항)

- 95점 이상 → **낙찰예정자**
- 서류 미비·오류·불명확으로 미달 시 **1회 보완** (3일 이내)

### 5단계: 미제출·미달 시 (제7항)

소정 기한 내 미제출·보완 후에도 미달 → **차순위 입찰자** 동일 심사.

### 입찰내역서 보완 (제8항)

발주기관 배부 내역서상 공종별 목적물 물량 누락·변경 시 예정가격조서상 금액이 예정가격의 **5/100 이상**이면 계약체결 시까지 보완 가능.

---

## 7. 순공사원가 기준 낙찰 배제 (제5조) ★

### 적용 대상

예정가격 **100억원 미만 공사**.

### 배제 기준

입찰가격이 예정가격 중 **순공사원가 × 98% 미만**인 자는 심사대상에서 제외.

### 순공사원가 산정

```
순공사원가 = (예비가격기초금액 중 재료비·노무비·경비 + 부가가치세) × (예정가격 / 예비가격기초금액)
98% 기준 = 순공사원가 × 98 / 100
```

소수점 첫째자리 이하 올림.

→ 산림사업 적용: 임도공사 등 100억 미만 공사에서 덤핑 입찰 방지 장치. 산림사업 표준품셈 기반 예정가격에서 재료비·노무비·경비 비중 확인 필수.

---

## 8. 결격·재심사 (제6조·제6조의2)

### 적격심사 결격 (제6조)

낙찰자 선정 이전 다음 사유 발생 시 결격:
- 부도 (회생절차개시결정 제외)
- 파산·해산
- 부정당업자 제재·영업정지
- 입찰무효

공동수급체 대표자 결격 시 전체 결격. 대표자 외 구성원 결격 시 해당 구성원 제외하고 잔존 구성원 출자비율·분담내용 변경하여 재심사.

### 수행능력 결격 (제6조의2)

입찰공고일 현재 업종 등록기준의 **기술자·자본금·사무실·시설·장비 보유기준** 미달 시 수행능력 결격 → [별표 1~5]에 따라 **감점**.

**일시적 등록기준 미달 예외** 5가지 (예: 기술인력 사망·실종·퇴직으로 50일 이내 미달, 영업소소재지 변경 등) — 입찰서 제출 마감일 전일까지 충족 시 결격 처리.

확인 방법: 적격심사대상자 제출 서류 + **현장조사** (기술자 실제 근무, 자본금·사무실·시설·장비 충족).

---

## 9. 산림 실무 적용

### 임도공사·산림토목 발주 시 적용 흐름

1. **공사규모별 평가기준 선택** ([별표 1~5] — 추정가격 기준)
2. **순공사원가 98% 기준** 입찰공고 명시 (100억 미만)
3. **시공경험·경영상태 평가자료**: G2B시스템 + 관련협회 자료
4. **예상종합평점 평가** → 95점 미달자 이의신청 (3일)
5. **적격심사대상자** 서류 제출 (7일, 경영상태 10일)
6. **기술자·자본금·시설·장비** 현장조사
7. 결격사유 발생 시 차순위 진행

### 산림청 예규 728호와의 비교

| 항목 | 조달청 시설공사 (본 지침) | 산림청 예규 728호 |
|------|-----------------------|----------------|
| 적용 공사 | 조달청 의뢰 시설공사 | 산림사업 (산림자원법상) |
| 적격통과점수 | 95점 이상 | 95점 이상 |
| 순공사원가 98% 배제 | 100억 미만 공사 | (산림청 자체 기준) |
| 시공경험 자료 | 관련협회 + G2B시스템 | 산림사업 실적 |
| 결격사유 확인 | 7일, 현장조사 | (예규별 기준) |

발주청 입장:
- **조달청 의뢰**: 본 지침
- **산림청 자체**: 산림사업 적격심사 예규 728호 + 산림사업법인 등록요건

## 10. 산림사업 발주 담당자 체크리스트

### 임도공사 발주 시

- [ ] 공사규모 (3억/10억/50억/100억 기준) 확인 → 별표 선택
- [ ] 순공사원가 98% 기준 입찰공고 명시 (100억 미만 시)
- [ ] 시공능력공시액·경영상태 자료 협회 통보 확인
- [ ] 종합건설/전문건설 상호시장 진출 시 시공경험 인정기준 확인
- [ ] 지역업체 가산 적용 여부 확인
- [ ] 적격심사 7일 서류 제출 안내
- [ ] 기술자·자본금·시설·장비 현장조사 계획
- [ ] 결격사유 발생 시 차순위 진행 절차 준비
