## 조직 개요 (제3조)

지방산림청 본청에 **운영재산관리팀 · 산림재난안전과 · 산림경영과**를 둔다.

---

## 산림경영과 (제3조 ④)

### 1. 경영계획담당

| 항목 | 내용 |
|------|------|
| 가 | 지역산림계획 및 국유림종합계획 |
| 나 | 산림조사계획 수립 및 실행 |
| 다 | 국유림 경영계획 수립·실행·변경 |
| 라 | 국유림 경영계획 운영 및 사업실행 지도 |
| 마 | 국유림경영정보시스템 운영 |
| 바 | 목재품질단속, 산림바이오에너지, 목재유통 등 목재산업 |
| 사 | 목재수확·공급 |
| 아 | 임산물(단기소득·토석 포함) 생산계획 수립 및 지도·감독 |
| 자 | 산지이용구분 |
| 차 | 보전산지 지정·변경지정·해제 |
| 카 | 산지관리지역계획 수립 |
| 타 | 산지정보시스템 운영 |
| 파 | 기후변화 관련 국유림 경영 |
| 하 | 산림경영인증 계획 수립 |
| 거 | 국유림위원회 운영 |
| 너 | 국유림 공동산림사업 추진 |
| 러 | 산림경영대행 업무 |
| 버 | 선도 산림경영단지 |
| 서 | 산림기본통계 및 임산물생산통계 조사 |
| 어 | 산림공간정보 구축·운영 및 관리 |
| 처 | 민북지역 산지관리종합계획 수립 |

### 2. 자원조성담당

| 항목 | 내용 |
|------|------|
| 가 | 조림·숲가꾸기 사업계획 수립 |
| 다 | 채종림·수형목의 지정·해제 및 관리계획 수립 |
| 라 | 조림·숲가꾸기 자재수급 계획 |
| 마 | 영림단 양성 및 임업인력 관리계획 수립 |
| 사 | 임업기술지도 및 보급 |
| 자 | 산림사업 법인관련 사무 |
| 차 | 임업기계·장비운영의 개발 및 보급계획 수립 |
| 카 | 시범림 운영계획의 수립 |
| 타 | 산림사업(산림자원 분야) 안전사고 예방 관리 |
| 파 | 숲가꾸기 패트롤 지도·감독 |
| 하 | 국유림영림단 신규·변경등록 승인 |
| 거 | 그 밖에 자원조성에 관한 사항 |
| 너 | 시험림 지정·해제 및 관리 |

### 4. 산림복지문화담당 (항목 요약)

국민의 숲, 숲길, 산림문화행사, 산림레포츠, 유아숲체험원, 산림문화자산, 치유의 숲, 산림교육관, 국립산악박물관, 국립등산학교, 무궁화동산·도시숲·정원, 산림복지일자리

### 5. 양묘사업담당 (항목 요약)

양묘사업소 운영계획, 종자 채취·보관, 양묘시설물 관리, 묘목생산·나눠주기, 시설양묘, 종묘사업

### 6. 임업직불담당 (항목 요약)

공익직접지불금 지급·등록·단속·환수, 의무준수 이행점검, 임업경영체 등록·관리, 산림컨설팅

## 국유림관리소 자원조성담당 (제4조 ① 6항)

> 관할: 평창·강릉·양양·영월·정선·삼척·태백 국유림관리소

| 항목 | 내용 |
|------|------|
| 가 | 조림·숲가꾸기 사업의 세부실행계획 수립 및 사업실행 |
| 나 | 채종림 및 수형목의 관리 |
| 다 | 영림단의 조직·훈련 및 관리 운용 |
| 라 | 임업기술에 관한 지도 |
| 바 | 조합 및 산림사업 법인관련 사무 |
| 사 | 임업기계 장비의 보수·유지관리 |
| 아 | 시범림의 운영 |
| 자 | 숲가꾸기 임산물 인도(매각 산물의 현장안내 및 운재로 변경 등 포함) |
| 차 | 산림사업 안전사고 예방 관리 |
| 카 | 숲가꾸기 패트롤 운영 |
| 타 | 그 밖에 자원조성에 관한 사항 |
