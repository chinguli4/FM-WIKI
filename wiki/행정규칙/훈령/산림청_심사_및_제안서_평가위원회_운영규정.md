# 산림청 심사 및 제안서 평가위원회 운영에 관한 규정

> 종류: 훈령 (산림청 훈령 제1481호)

## 1. 적용범위 (제1조~제3조)

산림청이 추진하는 **물자구매·용역계약**의 심사 및 제안서 평가위원회 구성·운영.

| 적용 | 대상 |
|------|------|
| **추정가격 5천만원 이상** 의무 적용 | ① 규격·기술입찰서 심사 (영 §18) / ② 제안서 평가 (영 §43·43의2·43의3) / ③ 품질 등 심사 (영 §44) |
| **5천만원 미만** | 사업부서 자체 준용 (투명·공정성 확보 의무) |

→ 산림조사, 산림경영계획 용역, R&D, ICT·드론 통합용역 등 협상계약 시 본 규정 적용.

---

## 2. 평가위원 구성·선정 절차 ★ (제6조~제9조)

### 핵심: "추천 → POOL(2배수) → 교섭" 3단계

```
[추천 출처]              [POOL 작성]              [교섭·확정]
국가인재 DB        →    별표 1 명단(POOL)  →    별표 3 표준문안
소속공무원              필요 인원의            유선 교섭
정책평가·자문위원       2배수 이상             1~3일 전
대학교·학회·협회                               기피사유 5개 고지
임업단체                                       소요인원 충족 시 종료
```

### 제6조 — 추천 출처

평가위원은 다음으로부터 **추천 받은 위원**으로 구성:
- 인사혁신처 **국가인재 DB**
- 소속공무원
- 정책평가위원·정책자문위원
- 대학교·학회·협회·**임업단체**

※ 단서: 보안 등 사업 특성상 필요 시 **내부 직원 참여 가능**.

### 제7조 — 위원 수 (위원장 포함)

| 추정가격 | 위원 수 | POOL 최소(2배수) |
|----------|---------|------------------|
| 5천만~1억원 미만 | **5인 이상** | **10인 이상** |
| 1억~10억원 미만 | **7인 이상** | **14인 이상** |
| 10억원 이상 | **9인 이상** | **18인 이상** |

- **외부위원 반드시 과반수** (5천만원 미만 사업에도 준용)
- 위원장: 외부위원 중 호선 과반 지지
- 간사 1인: 사업부서장이 선정

### 제8조 — POOL 작성

1. 사업부서장이 **추정가격별 인원의 2배수 이상**을 평가위원 예비자로 선정
2. 별표 1 「심사 및 제안서 평가위원회 평가위원 명단(POOL)」 작성
3. 제9조에 따라 위원 교섭 후 별표 2 「선정 내역」 작성

### 제9조 — 교섭 ★

| 항목 | 기준 |
|------|------|
| 시기 | 심사·평가 시작 **1~3일 전** |
| 방법 | **유선**, 별표 3 「위원 교섭 표준문안」 |
| 고지사항 | 기피사유 5개 (이해당사자·재직경력·후원금 등) 설명 |
| 종료 | 부서장이 **전문성 고려 참여가능 여부 확인**, **소요인원 충족 시 교섭 종료** |
| 내부위원 선정 | 심사·평가 시작 **전일**에 실시 |

> 📝 **"추천→후보군→임의 선정" 구조의 실제 의미**: 훈령상 "무작위 추첨" 문언은 없다. 부서장이 2배수 POOL에서 전문성·기피사유 확인하며 유선 교섭해 소요인원이 차면 종료하는 방식. 별도 "추첨"은 **제11조 ② 제안서 발표순서 당일 추첨**에만 등장.

---

## 3. 기피·제척 사유 5개 (제9조 ② / 제12조)

| # | 사유 |
|---|------|
| 1 | 평가대상 사업과 관련 **용역·자문·연구 수행** 경험 |
| 2 | 평가대상 사업의 **이해당사자** (대리관계 포함) |
| 3 | **최근 3년 이내** 평가대상 업체 재직 경력 |
| 4 | 소속단체·학회가 특정업체로부터 받은 **후원금이 사업과 직접 관련** |
| 5 | 그 밖에 **공정한 심사 곤란** |

- 평가위원이 발견 시 **위원회 개최 전 위원장에게 기피 신청** 가능
- 위원장은 평가위원에게 사전 고지, 해당위원은 해당 안건 평가에서 배제

---

## 4. 평가위원회 운영 (제10조~제11조)

### 개의 정족수

- 위원장 포함 **2/3 이상 출석**
- 사업부서장은 사업 특성·중점평가사항 사전 설명 가능
- 제안서는 평가 당일 배부 원칙 (심도 검토 필요 시 사전 배부)

### 제안 설명회 (제11조)

- **발표 순서: 당일 추첨** ★
- 경쟁업체 제안 설명 청취 불가 (보안)
- 발표: 사업총괄책임자 + 5인 이내 보조자
- 발표자료가 제안서와 다를 경우 **감점 처리 가능**
- 화상평가 등 전자적 방식 가능
- **5천만원 미만 / 유찰 단일업체**: 제안 설명회 생략·서면심사 대체 가능

---

## 5. 기술평가점수 산출 (제5조)

- 각 평가위원 점수 중 **최저·최고점 제외 산술평균**
- 최저·최고가 2개 이상이면 **1개만 제외**
- **소수점 셋째자리 반올림**

---

## 6. 보안·행동요령 (제13조~제14조)

- 별표 4 「평가위원 위촉 동의 및 보안각서」 제출 의무
- 위원장·전원 동의 없이 **녹취·촬영·외부공개 금지**
- 심사 중 **전화(휴대전화 포함) 사용 금지** — 불가피 시 위원장 동의 후 공개 장소
- 평가결과서 **전원 서명 시까지 외부 출입 제한**

---

## 7. 수당 (제15조)

매년도 예산·기금운용계획 집행지침 따름. **심사 내용에 따라 탄력 지급 가능**.

---

## 8. 실무 적용 — 산림사업 발주 체크리스트

### 발주 1주일 전

- [ ] 추정가격별 위원 수 확인 (5/7/9인 이상)
- [ ] 추천 출처 다변화 (국가인재 DB + 학회 + 임업단체 등)
- [ ] **2배수 이상** POOL 명단 작성 (별표 1)

### 평가 1~3일 전

- [ ] 별표 3 표준문안으로 **유선 교섭**
- [ ] 기피사유 5개 모두 고지·확인
- [ ] **외부위원 과반수** 확보 확인
- [ ] 별표 2 「선정 내역」 작성

### 평가 당일

- [ ] **별표 4 보안각서** 징구
- [ ] 위원장 호선 (외부위원 중 과반 지지)
- [ ] 2/3 이상 출석 확인 후 개의
- [ ] **발표순서 당일 추첨**
- [ ] 전화 사용 통제, 평가결과서 전원 서명 후 외부 출입 허용

### 점수 산출

- [ ] 최저·최고 제외 산술평균, 소수점 셋째자리 반올림

---

## 9. 기재부 훈령과의 관계

본 산림청 훈령은 **재정경제부 훈령 「협상에 의한 계약 제안서 평가 업무처리 규정」**(계약예규 「협상에 의한 계약체결기준」 §7 근거) 구조를 산림청 자체 적용하기 위한 규정. 큰 틀(외부위원 과반·POOL 운용·기피사유·산술평균)은 동일하나 산림청 특유 사항은:

| 차이 | 기재부 훈령 | 산림청 훈령 |
|------|------------|------------|
| 위원 수 구간 | 1억원 기준 5인/7인 | **5천만·1억·10억** 기준 5/7/9인 |
| 추천 출처 | 명문 없음 | **국가인재 DB + 임업단체** 등 명시 |
| POOL 배수 | 명문 없음 | **2배수 이상** 명시 |
| 교섭 시기 | 명문 없음 | **1~3일 전 유선** 명시 |
