## 핵심 기준

### 등록 대상

산림기술 진흥 및 관리에 관한 법률 §15①에 따른 **산림사업 설계·감리 용역업**.  
업종: 종합업 / 전문업(설계 또는 감리) 구분.

### 등록 요건 적격여부 확인

#### ① 기술수준 (§3)

| 서류 | 비고 |
|------|------|
| 산림기술자 자격증 사본 (산림기술 진흥법 §9) | 보유 기술자별 확인 |
| 고용보험 피보험자격 취득신고 내역 | 재직 확인 |

필요 시 추가 서류 요구 가능: 개인별 경력신고서·경력확인서·고용계약서 사본.  
*(고용보험 적용 제외 근로자: 국민연금·건강보험·산재보험 가입증명 중 하나로 대체)*

#### ② 사무실 (§4)

- 관할 **시·도 내** 위치 (행정안전부 행정구역 현황 기준)
- 타 업종 사무실과 공유 시 **공간적으로 명확히 구분**
- 사무설비·통신설비·사무인력 상시 근무 가능 공간

**불인정 건물 유형:**  
단독주택·공동주택, 축사·온실·저장고, 무허가·가설건축물  
*(단, 용도변경 객관적 인정 시, 또는 시장·군수 허가 가설건축물 예외 적용 가능)*

**사무실 보유 증명 서류 (§4②):**

| 점유 형태 | 제출 서류 |
|-----------|-----------|
| 자기 소유 | 건물등기사항증명서 또는 건축물대장 |
| 임대차 | 임대차계약서 + 건물등기사항증명서 또는 건축물대장 |
| 전대차 | 전대차계약서 + 건물등기사항증명서 또는 건축물대장 + 전대차 사용 증명서류 |

원칙: 사무실 소재지 **방문 확인** (서류 심사로 충분하다고 판단 시 생략 가능, §4④ 단서)

#### ③ 자본금 (§5) — 종합업만 해당

| 사업자 유형 | 제출 서류 |
|-------------|-----------|
| 법인 | 신청일 기준 대차대조표·손익계산서 또는 자본금 보유 증명서류(법인등기부등본) |
| 개인사업자 | 은행잔고증명서 또는 자산 증명서류 |
| 타 법령 업종 보유자 | 산림기술용역업 영위에 필요한 자본금 별도 보유 증명서 |

세부 확인 필요 시: 산림사업법인 관리지침(고시 2023-133호) 별지 제2호서식 기업진단 보고서 제출 요구 가능 (§5②).

## 등록·변경·휴폐업·지위승계 절차

### 등록 처리 후 공고 (§6)

수탁기관은 등록증 발급·변경·휴폐업·지위승계 처리 즉시 **산림기술정보통합관리시스템**에 등재·공고.  
→ 시스템에서 실시간 확인 가능하도록 유지.

### 현장확인 (§7)

등록신청·변경신고·휴폐업신고 심사 중 필요 시, **사전 안내 후** 사무실 및 현장 방문 확인 가능:
- 기술인력 보유 현황
- 시설·장비 보유 현황
- 자본금 보유 현황
- 제출 자료 일치 확인

### 변경신고 (§8)

변경신고 접수 시 등록요건 적격 여부 재확인 의무.

| 변경 사항 | 증명 서류 |
|-----------|-----------|
| 명칭·대표자 | 법인등기사항증명서·사업자등록증 |
| 소재지 | 사무실 적격여부 확인 서류 일체 (§4 준용) |
| 소속 산림기술자 자격 종류·수 | 기술수준 적격여부 확인 서류 (§3 준용) |

> 산림기술자 퇴사 확인 기준: **고용보험 피보험자격 상실신고서의 상실연월일** (§8③)

### 지위승계 (§11)

양수인에 대한 등록요건·양도양수 적격 여부 확인 의무. 제출 서류:
1. 양도인의 산림기술용역업 등록증
2. 법인등기사항증명서 또는 사업자등록증
3. 양도·양수계약서 사본
4. 기술사사무소 개설등록증·엔지니어링사업자 신고증 사본 등 (해당자)
5. 산림사업 승계 수반 시: 사업 내역서

## 위반행위 보고 (§12) — 수탁기관 의무

수탁기관이 다음 사실 인지 시 **산림청장에게 즉시 통보**:

| 위반 유형 |
|-----------|
| 거짓·부정 방법으로 등록한 경우 |
| 등록요건에 맞지 않게 된 경우 |
| 중요 변경사항 기간 내 미신고 |
| 타인에게 상호·성명 사용 허가 또는 등록증 대여 |
| 금지 행위 알선 또는 타인 등록증 사용 |
| 지위승계 기간 내 미신고 |
| 시스템에서 위반행위 확인 |
