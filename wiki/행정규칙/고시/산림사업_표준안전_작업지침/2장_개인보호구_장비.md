## 제3장 개인보호구

### 보호구 일반

제23조는 작업조건에 맞는 인증 보호구 지급·사용을 요구한다. 산림사업시행업자는 보호구를 상시 점검하고 이상이 있으면 수리·교환해야 하며, 지급 또는 착용 지시를 받은 작업자는 착용해야 한다.

| 보호구 근거 | 예시 |
|-------------|------|
| 「임업 및 산촌 진흥촉진에 관한 법률」 제18조의12 품질인증 보호구 | 산림작업복, 산림작업 안전장갑, 산림작업 안전화 등 |
| 「산업안전보건법 시행령」 제74조제1항제3호 보호구 | 안전모, 보안면, 방진마스크 등 안전인증 보호구 |
| 산림사업장 안전사고 예방에 필요하다고 인정되는 보호구 | 작업복 상·하의, 안면보호구 등 |

### 부위별 보호 기준

| 조문 | 보호 부위 | 기준 |
|------|-----------|------|
| 제24조 | 머리·얼굴 | 낙하·비래·추락 위험 작업은 인증 안전모 제공. 톱밥·파편·휘어진 가지 비산 위험 작업은 보안면 등 눈·안면부 보호장비 제공. 벌 피해·온열질환 예방이 필요한 풀베기사업장은 위험요인 제거 후 벌 보호망 부착 작업모로 대체 가능 |
| 제25조 | 상체 | 임업용 동력기계톱 사용 작업자에게 인증 산림작업복 상의 제공 |
| 제26조 | 손 | 손 부상 위험 작업자에게 인증 산림작업 안전장갑 제공. 동력기계톱·예불기 조작 시 일일 국소진동이 4m/s2 이상이면 방진장갑 착용 |

| 제29조 | 청력 | 동력기계톱·예불기 등 소음 발생 기계장비 작업은 귀덮개 제공 |
| 제30조 | 호흡기 | 분진·약제 등 호흡기 질환 우려 작업은 개인 전용 호흡용 보호구 지급·착용 |

## 제4장 도구 및 장비의 취급

### 일반 준수사항

제31조는 모든 도구·장비에 대해 작업 전후 안전검사, 안전장치·제어장치 점검, 결함 장비 보고·수리·교환, 적합 용도 사용, 작업자 간 안전거리, 연료 공급 시 화기·흡연 금지, 장비 정지 후 연료 공급, 칼날 부착 도구의 어깨 운반 금지, 미사용 시 안전한 장소 보관을 요구한다.

### 임업용 동력기계톱

| 항목 | 기준 |
|------|------|
| 작업 전 | 안전장치·제어장치 이상 여부 검사, 제조사 지침에 따른 작동·유지관리 |
| 필수 안전장치 | 체인 브레이크, 오른손보호대, 안내판, 지레발톱, 체인받침, 방진고무, 엑셀잠금장치 |
| 사용 금지 | 안전장치 미작동, 공회전 중 체인 고정 불량, 공회전 상태 불량, 안내판·손잡이·제어장치 느슨함, 부품 손상·누락 |
| 이동 | 엔진 정지, 날 보호 덮개 사용 |
| 연료 | 작동 중 연료 공급 금지 |
| 시동 | 오른발로 오른손 보호대 고정, 왼손으로 앞 손잡이 고정, 오른손으로 시동 |
| 대기 | 작동이 필요하지 않으면 체인 브레이크 작동 |

### 임업용 예불기

| 항목 | 기준 |
|------|------|
| 작업 전 | 안전장치·제어장치 검사, 안전받침대 장착, 칼날 고정·회전상태 확인 |
| 이동 | 풀베기 외 이동 시 칼날 제거 또는 보호 덮개, 작업자 간 5m 이상 안전거리 |
| 장착 | 어깨끈 조절 기능 정상, 현장에 적합한 칼날 |
| 접근 금지 | 엔진 회전 중 칼날이 멈춰 있어도 접근 또는 다른 작업자 근접 금지 |
| 사용 중지 | 칼날이 암석 등 장애물에 부딪힌 경우, 칼날에 잡초·덩굴 등이 얽힌 경우 |
| 휴식 | 시동 정지 후 미끄러지지 않는 안전한 장소에 칼날이 잘 보이도록 보관 |

### 이동식 기계장비

제34조는 문 닫기, 안전벨트, 운전자 숙련, 결함 부품 보고·교체, 노출 회전부 보호, 예비 스위치, 무자격자 탑승·조작 금지, 급출발·급제동·선회·주행·원목 이동 시 주의, 헤드가드 부착, 종료·정차 시 구름방지 장치를 요구한다.

헤드가드 대상 예시: 동력상하차기, 임업용 굴착기, 임업용 트랙터, 동력임내차, 원목크레인, 프로세서, 하베스터, 펠러번처, 포워더, 스키더.

### 와이어로프와 달기구

| 조문 | 기준 |
|------|------|
| 제35조 | 설치 전 로프 종류·직경을 작업자에게 알림 |
| 제35조 | 이음매, 한 꼬임 내 소선 10% 이상 절단, 공칭지름 7% 초과 감소, 꼬임, 심한 변형·부식, 열·전기충격 손상 와이어로프 사용 금지 |
| 제36조 | 작업자 탑승 운반구 지지 달기와이어로프·체인 안전계수 10 이상 |
| 제36조 | 화물 하중 직접 지지 달기와이어로프·체인 안전계수 5 이상 |
| 제36조 | 훅·샤클·클램프·리프팅 빔 안전계수 3 이상 |
| 제36조 | 그 밖의 달기구 안전계수 4 이상, 최대허용하중 표식 부착 |

## 현장 체크포인트

- 공종별 보호구 지급대장과 착용 사진이 있는가.
- 체인톱·예불기 작업 전 안전장치 점검표가 있는가.
- 결함 장비가 발견되었을 때 수리·교환 기록을 남겼는가.
- 예불기 이동 시 5m 이상, 작업 시 15m 이상 안전거리를 지키는가.
- 와이어로프 폐기 기준과 달기구 안전계수를 작업자가 알고 있는가.
