## 제5장 제1절 임업용 동력기계톱을 이용하는 작업

### 체인톱 작업방법

| 조문 | 핵심 기준 |
|------|-----------|
| 제37조 | 비상시 사용할 예비 체인 휴대 |
| 제38조 | 작업 전 작업순서와 작업자 간 소통 방법 숙지 |
| 제38조 | 안전장치·제어장치, 톱날 마모상태·장력 확인 후 작업 |
| 제38조 | 시동 전 톱날이 사람·물건에 접촉하지 않는지 확인하고 견고한 지반에서 균형 유지 |
| 제38조 | 톱날 끼임 발생 시 체인브레이크 작동·시동 정지 후 지렛대 등으로 틈새벌림, 톱날 빼기 순서로 조치 |
| 제38조 | 이동 시 엔진 정지, 날 안전 덮개 사용, 이동자 간 3m 이상 안전거리 |
| 제38조 | 연료 공급 시 엔진 정지, 평평한 장소에서 안정 상태 유지 |
| 제38조 | 벌목작업 시 벌도목 수고 2배 이상 안전거리 확보 |
| 제38조 | 벌도목 주변 장애물 제거, 대피로·대피장소 선정 |
| 제38조 | 흉고직경 20cm 이상 벌목 시 수구각 30도 이상, 따라베기, 벌도맥, 안전보조장치 적용 |

### 산불피해지 작업

제39조는 산불피해지에서 미끄러짐과 산불피해목 붕괴 위험을 작업자에게 알리고 사전 교육하도록 한다. 산불피해지 벌채·위험목 제거는 일반 벌목보다 불확실성이 크므로 작업 전 위험성평가와 대피계획을 별도로 남긴다.

### 작업자 배치와 벌목 전 준비

| 조문 | 기준 |
|------|------|
| 제40조 | 벌도목 수고 2배 이상 거리 확보, 벌목 전 상호 소통 |
| 제40조 | 원목·돌 등이 굴러떨어질 위험이 있는 장소 위·아래에 작업자 배치 금지 |
| 제41조 | 임도·작업로·통행로·작업자 위치, 인접 수목, 지형, 풍향·풍속 확인 |
| 제41조 | 나무 크기·수형·기울기, 덩굴류, 고사목 분포 확인 |
| 제41조 | 튀어 오름 위험 나무·가지·고사목, 관목·덩굴·뜬돌 등 장애물 확인 |
| 제41조 | 산주·주민·탐방객 등 작업자 외 사람 활동 여부 확인 |

### 특수 벌목 상황

| 조문 | 상황 | 기준 |
|------|------|------|
| 제42조 | 가지가 얽힌 나무 | 윈치 등 보조도구 사용 |
| 제42조 | 덩굴에 얽힌 나무 | 벌목 전 덩굴 제거, 불가능하면 윈치 등 보조도구 사용 |
| 제43조 | 걸림목 | 직경급·장소·지형 확인, 대피장소 선정, 접근금지 표식, 견인기구·벌목 안전보조장치·기계집재 장치 사용 |
| 제43조 | 걸림목 처리 금지 | 버티는 나무 벌목, 걸림목 위에 다른 나무 벌목, 중간 절단 낙하, 어깨 운반, 가지 제거로 걸림목 제거 금지 |
| 제44조 | 상단베기 | 동력기계톱을 어깨보다 높이 올리고 작업 금지 |
| 제45조 | 기울어진 나무·쪼개지기 쉬운 나무 | 밧줄 3~5회 묶기 또는 자동바 사용 |

### 대피장소·대피로·벌목 절단

| 조문 | 기준 |
|------|------|
| 제46조 | 대피장소는 벌목방향 반대 45도 방향, 벌도목에서 충분한 거리의 안전한 곳 |
| 제47조 | 대피로의 가지·조릿대 등 장애물 제거, 적설 시 눈을 밟아 다져 대피 원활화 |
| 제48조 | 방향베기 깊이는 벌근직경의 1/3~1/4 |
| 제48조 | 방향베기 각도는 30도 이상, 바닥절단부 깊이와 상부·하부절단부 끝부분 일치 |
| 제48조 | 따라베기 위치는 방향베기 바닥절단부보다 직경의 1/10 이상 높게 |
| 제48조 | 벌도맥 폭은 벌근직경의 약 1/10 남기고 과도하게 깊은 따라베기 금지 |
| 제49조 | 역방향 기울기, 흉고직경 20cm 이상 방향 조정, 바람 등 외부 위험 시 2개 이상 쐐기 또는 안전보조장치 권고 |

### 조재작업

제50조는 강풍·강설 등 피해목 조재 시 산림기술자와 작업방법을 논의하고, 감리 또는 공사감독관이 작업방식·안전 사전교육을 하도록 한다. 원목 굴러떨어짐 위험 점검, 말뚝 등 방지조치, 장애물 제거, 비탈면 위쪽 작업, 원목 안정 후 가지정리, 압력 응력 방향에 따른 절단 순서, 동일 원목에 2명 이상 동시 작업 금지를 포함한다.

## 제5장 제2절 피해목의 처리 작업

### 피해목 일반

| 조문 | 기준 |
|------|------|
| 제51조 | 비탈면 상부 및 바깥쪽부터 순서대로 작업 |
| 제51조 | 원목 낙하·굴러떨어짐 위험 확인 |
| 제51조 | 발디딤 장소 정비, 대피로 확보 후 작업 |
| 제51조 | 압축력·장력에 의한 쪼개짐·타격 위험 위치 작업 금지 |

### 피해목 유형별 처리

| 조문 | 대상 | 핵심 기준 |
|------|------|-----------|
| 제52조 | 기울어진 나무·활 모양 나무 | 굴러떨어짐·넘어짐 범위 진입 금지, 밧줄 3~5회 또는 자동바, 반대방향 기울기는 윈치 사용 |
| 제53조 | 수관 훼손 나무 | 훼손 부분을 견인도구로 끌어내린 후 작업 |
| 제54조 | 겹쳐 쓰러진 나무 | 기계장비 등으로 가까운 곳과 위쪽부터 끌어내 안전한 장소에서 작업, 쓰러진 나무 위 이동·작업 금지 |
| 제55조 | 수간이 굽은 나무 | 밧줄·자동바 사용, 압축력 쪽 선절단 후 장력 쪽 절단, 반발력 방향에 위치 금지 |
| 제56조 | 쓰러진 나무의 수간·뿌리 절단 | 가지치기·원목 절단으로 응력 약화 후 수간·뿌리 분리, 지주·받침목·와이어로프 등으로 고정 |
| 제57조 | 가지제거 | 원목 상태 확인, 안전한 발디딤 확보, 원목 위 작업 금지, 반발력 있는 가지는 안쪽을 먼저 약하게 처리 |
| 제58조 | 우드그래플+체인톱 | 우드그래플로 움직일 부분을 눌러 안정화, 체인톱 작업자는 기울어진 나무·걸림목 아래 진입 금지 |
| 제59조 | 하베스터·프로세서 | 수간 절단 후 안전한 가지치기·조재, 겹쳐 쓰러진 피해목은 위쪽부터 정리 |
| 제60조 | 스윙야더 | 견인거리 짧게, 걸림목은 일으키거나 당겨 쓰러뜨린 후 처리 |

## 현장 체크포인트

- 벌도목 수고 2배 이상 작업자 이격이 확보되었는가.
- 대피장소가 벌목방향 반대 45도 방향에 지정되었는가.
- 걸림목 처리는 금지행위 없이 견인기구·기계장비로 처리하는가.
- 흉고직경 20cm 이상 벌목 시 방향베기·따라베기·벌도맥 기준을 적용했는가.
- 산불피해지·풍수해·설해 피해목은 일반 벌목과 별도로 위험성평가했는가.
