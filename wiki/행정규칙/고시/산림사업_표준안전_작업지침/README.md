## 성격

산림사업 시행상의 안전에 관하여 산림사업시행업자와 작업자에게 지도·권고할 기술상의 지침이다. 기존 위키의 조림·숲가꾸기 지침이 설계·감리·품질·사업비 중심이라면, 이 고시는 현장 안전작업 방법을 직접 규율한다.

특히 다음 항목은 산림사업 안전관리의 기준축으로 보아야 한다.

- 작업 전 안전점검 회의
- 상하작업 금지와 작업 안전거리
- 응급상황 대비 절차와 구급용구
- 위험요소 사전확인, 수시 확인, 주 1회 이상 현장점검
- 악천후 작업중지, 휴식시간, 온열질환 예방
- 개인보호구 지급·착용
- 동력기계톱·예불기·와이어로프·이동식 기계장비 취급
- 벌목, 피해목 처리, 집재·운재, 약제살포, 산림토목 작업별 안전수칙

## 현장 증빙 매핑

| 지침 항목 | 현장에서 남겨야 할 자료 |
|-----------|--------------------------|
| 작업 전 안전점검 회의 | 회의일지, 작업자 건강상태 확인표, 당일 작업내용·위험요인 공유 기록 |
| 위험요소 사전확인 및 점검 | 위험성평가표, 작업방법 변경·신규장비 도입 시 수시점검표, 주간 현장점검표 |
| 응급상황 대비 | 비상연락망, 진입 가능 지점·통신 가능 지점 위치도, 구급용구 비치대장 |
| 개인보호구 | 지급대장, 착용 확인 사진, 보호구 점검·교체 기록 |
| 동력기계톱·예불기 | 작업 전 안전장치 점검표, 장비 정비기록, 작업자 교육기록 |
| 벌목·피해목 처리 | 벌도목 수고 2배 안전거리 확인, 대피로·대피장소 사진, 걸림목 처리 기록 |
| 집재·운재 | 와이어로프 점검표, 장비 안정도·최대하중 확인, 작업구역 출입통제 사진 |
| 약제 작업 | 현수막 게시 사진, 약제 사용설명 숙지·교육기록, 보호구 착용 확인, 잔여 약제 반납기록 |
| 산림토목 | 작업계획서, 굴착기 사전조사, 유도자 지정, 붕괴·추락 방지 조치 사진 |
