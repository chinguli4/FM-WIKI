# 제3장 작업장 관리

> 근거: 산림청고시 제2025-82호 「산림사업 표준품셈」 별표

---

## 3-1. 경계표시

| 소요인력 | 인력구분 |
|---------|---------|
| 0.2인/ha | 보통인부 |

- 사업장 경계를 따라 10m 내외 간격으로 흰색 페인트 표시 (구분 곤란 시 황색)
- 계곡부·어린나무 생육지 등 육안으로 명확히 구분 가능하거나 실시설계자가 표시한 경우 생략 가능
- 할인·할증: 집단화정도(1-4-3), 경사도(1-4-5)

---

## 3-2. 작업로 선정

| 소요인력 | 인력구분 |
|---------|---------|
| 1.0인/1일·1km | 초급기술자 |

- 숲가꾸기 작업로 중 **대작업로 선정**에 적용
- 예정선을 선정하고 5m 내외 간격으로 표식 (적색 임업용 마킹테이프)
- 할인·할증: 경사도(1-4-5), 장애물의 정도(1-4-8)

---

## 3-3. 작업로 설치

| 구분 | 소요인력(인/km) | 인력구성 |
|------|--------------|---------|
| 소작업로 | 2.0 | 벌목부 50%, 보통인부 50% |
| 대작업로 | 3.0 | 벌목부 50%, 보통인부 50% |

- **소작업로**: 폭 1.5m 내외 (이동 및 주행 방해 식생 제거)
- **대작업로**: 폭 3.0m 내외 (임업기계 주행 및 숲가꾸기 산물 수집)
- 제거해야 할 상층목은 솎아베기 본수에 포함 (본 품 미포함)
- 제거 식생은 임내에 버리는 것이 기준 (수집 불포함)
- 토공 추가 시설 시: 임산물 운반로 시설기준 적용
- 할인·할증: 경사도(1-4-5), 제거대상 식생(1-4-9)

> **소요인력 산출 예시** — 대작업로 2.0㎞ 설치:
> 벌목부 2.0㎞ × 3.0인/㎞ × 50% = 3.0인
> 보통인부 2.0㎞ × 3.0인/㎞ × 50% = 3.0인

---

## 3-4. 임산물 운반로 신설 및 보수·복구

### 3-4-1. 신설 및 보수 기준

1. 지형 여건에 맞는 환경친화적 방법으로 설치, 시설거리 최소화
2. 규모: 최소 규모, 가선 집재장비 활용, **노폭 2.5m 내외**, 편구배 4% 내외, 영선 기준 시설
3. 사업완료 후 지속 이용 시: 절·성토면 복구 실시 (산지관리법 시행령 제47조1호)
4. 신설·보수·복구에는 기계화 작업로 포함
5. 표준 횡단경사 50% 기준, 횡단경사별 할증 적용
   - 65% 이상: 20% 가산
   - 55∼65% 미만: 10% 가산

### 3-4-2. 임산물 운반로 및 작업로 신설비 산정

| 항목 | 기준값 |
|------|--------|
| m당 절토량 | 0.64㎥ (성토는 무대처리) |
| 시간당 작업량(Q) | 8.95㎥/hr |
| 표준 노폭 | 2.5m |
| 표준 횡단경사 | 50% |

**m당 시설단비 예시 (2020년 하반기 단가 기준)**

| 구분 | 단가(원/m) |
|------|-----------|
| 노무비 | 3,037 |
| 재료비 | 410 |
| 경비 | 880 |
| 합계 | 4,327 |

### 3-4-3. 임산물 운반로 및 작업로 보수비 산정

**완전복구 보수 (A)**
- 파괴된 기설 운반로에 대해 현지조사 기반 보수 설계서 작성, 1회에 한해 완전복구 경비 산출

**상용인부 고용경비 (B)**

| 입지·기상 조건 | 기준 |
|--------------|------|
| 불량 | 2km당 1일 1인 |
| 보통 | 3km당 1일 1인 |
| 양호 | 4km당 1일 1인 |

> 보수비 합계 = A + B

**보수공정표 (100㎡ 기준)**

| 종별 | 인부 | 내용 |
|------|------|------|
| 노면정지 | 1.5인 | 잡초 및 직경 5cm 이상 소석 제거, 노면 고르기 |
| 노면굴기 | 4.0인 | 노면두께 10cm 내외 괭이고르기·다지기, 소석 제거 |

## 3-5. 예정지정리

| 구분 | 작업내용 | 소요인력(인/ha) |
|------|---------|--------------|
| 보완사업지 정리 | | 2.4 |
| | 모든 벌채산물 임내존치지역 | 5.5 |
| 벌채부산물 임내존치지역 | 벌채부산물 임내정리 | 6.0 |
| 휴경지 등 정리 | | 10.0 |
| 관목지 등 정리 | | 10.0 |
| 산불 등 피해지 정리 | | 10.0 |
| 불량림 정리 인력 정리 | | 9.0 |
| | 벌채부산물 임내정리 | 7.0 |
| 벌채와 동시정리지역 | 모든 벌채부산물 반출 | 9.0 |

- 인력구성: 특별인부 50%, 보통인부 50%

**주요 주석**
- 보완사업지: 조림 후 1년 미만 경과 산림
- 휴경지 등: 관목·덩굴류 피복, 조림 후 1∼3년 미만 산림 포함
- 관목지 등: 조림 후 3년 이상 경과 산림 포함
- 우드그랩 임내 주행 시: 산지경사도 25° 이하 제한
- 미이용 산림바이오매스 정리량 100㎥/ha 이상: 10% 할증, 50∼100㎥/ha 미만: 5% 할증
- 침엽수림: 10% 할증, 활엽수림: -10% 할인
- 개벌지 700㎡ 이하: 10% 할증, 3,000㎡ 미만: 5% 할증
- 할인·할증: 산지경사(1-4-5), 작업장까지 이동거리(1-4-6)

(㎥/1인·1일)

|----------------|--------|--------|--------|--------|--------|--------|--------|--------|--------|--------|
| 임내정리 | 3.9 | 4.1 | 4.3 | 4.5 | 4.6 | 4.7 | 4.8 | 4.9 | 5.0 | 5.1 |

- 인력구분: 보통인부
- 수간부와 가지부 분리하여 등고선 방향으로 정리 (높이 1m 이하)
- 산물임내정리 구간: 최대 30m 이내
- 정리산물 수량: 지조 포함
- 할인·할증: 경사도(1-4-5)

> 16㎥ ÷ 4.3㎥ = 3.72인

## 3-8. 드론 영상 촬영

(인/ha)

| 종별 | 드론조종자 | 부조종자 | 특별인부(신호수) | 계 |
|------|-----------|---------|--------------|---|
| ① 촬영 계획 수립 | 0.005 | - | - | |
| ② 드론 촬영(RTK 포함) | 0.010 | 0.010 | - | |
| ③ 영상 처리 및 정사영상 생성 | - | - | 0.010 | |
| ④ 결과물 점검 및 보고서 작성 | - | - | 0.005 | |
| **합계** | **0.015** | **0.010** | **0.015** | **0.040** |

- 사업 시행 전·후 2회 촬영 기준
- 촬영계획 수립: 1인 1일 200ha 기준
- 드론촬영: 1대 2인 1조, 약 100ha 기준
