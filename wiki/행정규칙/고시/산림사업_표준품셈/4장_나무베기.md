# 제4장 나무베기

> 근거: 산림청고시 제2025-82호 「산림사업 표준품셈」 별표

---

## 4-1. 수확베기

### 4-1-1. 임업용 동력기계톱

**생산방법 구분**

| 구분 | 내용 |
|------|------|
| 단목 | 산지에서 벌목 + 가지제거 + 조재 완료 |
| 전간목 | 벌목 + 가지제거 + 초두부 절단 완료 |
| 전목 | 벌목만 실행 |

**벌목 소요품 (㎥/1인·1일)**

| 벌도목 구분 | 벌목량 | 인력구분 |
|------------|--------|---------|
| 단목 | 20.17 | 벌목부 |
| 전간목 | 25.50 | 벌목부 |
| 전목 | 40.34 | 벌목부 |

**주요 주석**
- 벌도목 평균경급 20cm 미만: 10% 할증
- 벌도목 평균경급 20∼30cm: 5% 할증
- 산불피해지 피해목 벌목: 20% 할증
- 할인·할증: 경사도(1-4-5), 장애물 종류·정도(1-4-8)
- 박피품은 별도 적용

### 4-1-2. 하베스터 (부착형 스트로크)

**벌도·조재 공정량**

| 작업 | 공정량(㎥/1인·1일) | 인력구분 |
|------|----------------|---------|
| 벌도·조재 (벌도+가지제거+조재) | 59.20 | 건설기계운전기사 |
| 조재만 (가지제거+조재, 프로세서 포함) | 154 | 건설기계운전기사 |

## 4-2. 단목베기

### 4-2-1. 100본당

(인/100본당)

| 흉고직경(cm) | 벌목 | 가지제거 | 토막내기 | 작업보조 |
|------------|------|---------|---------|---------|
| 10 이하 | 0.2 | 0.1 | 0.1 | |
| 12∼14 | 0.4 | 0.1 | 0.1 | |
| 16∼18 | 0.6 | 0.2 | 0.2 | |
| 20∼22 | 0.7 | 0.2 | 0.3 | |
| 24∼26 | 1.0 | 0.2 | 0.2 | |
| 28∼30 | 1.2 | 0.4 | 0.4 | |
| 32∼34 | 1.9 | 0.5 | 0.5 | 0.6 |
| 36∼38 | 2.4 | 0.7 | 0.7 | |
| 40∼42 | 2.9 | 0.8 | 0.8 | |
| 44∼46 | 3.6 | 1.1 | 1.1 | |
| 48 이상 | 4.2 | 1.3 | 1.3 | |

- 인력구분: 벌목(체인톱 사용) → 벌목부, 작업보조 → 보통인부
- 체인톱 사용인부: 벌목·가지제거·토막내기 담당
- 작업보조: 벌도대상목 주변정리 및 재료운반
- 관목류 제거 시: 보통인부 30% 할증
- 규격재 생산을 위한 조재 포함 시: 20% 할증
- 할인·할증: 집단화정도(1-4-3), 경사도(1-4-5), 장애물 정도(1-4-8), 규격재 생산(1-4-16)
  - 산림병해충방제: 집단화정도(1-4-3) 미적용, 방제대상목 분포(1-4-17) 반영

### 4-2-2. 1,000㎡당

(인/1,000㎡당)

| 구분 | 나무높이 5m 미만 | 5∼8m 미만 | 8m 이상 |
|------|--------------|-----------|--------|
| 벌목부 | 2.14 | 2.80 | 3.65 |
| 보통인부 | 0.51 | 0.66 | 0.87 |
| 굴착기+부착용집게 (0.2㎥, hr) | 2.71 | 3.54 | 4.61 |

- 집재거리 100m까지 기준. 초과 시 매 100m마다 인력품 30% 가산
- 나무베기 + 잔가지 정리 + 집재·반출 정리 포함
- 공구손료 및 경장비 기계경비: 10% 계상

## 4-3. 위험목 베기

(인/본당)

| 가슴높이지름(cm) | 특별인부 | 벌목부 | 보통인부 | 굴삭기 우드그랩(hr) | 크레인(hr) |
|---------------|---------|------|---------|-----------------|---------|
| 16∼20 | 0.14 | 0.28 | 0.28 | 0.28 | 0.28 |
| 22∼30 | 0.18 | 0.36 | 0.36 | 0.36 | 0.36 |
| 32∼40 | 0.27 | 0.53 | 0.53 | 0.53 | 0.53 |
| 42∼50 | 0.35 | 0.70 | 0.70 | 0.70 | 0.70 |
| 52 이상 | 0.38 | 0.75 | 0.75 | 0.75 | 0.75 |

- 적용 대상: 숲가꾸기 사업, 위험 해안가·절벽지·농경지·주택지 주변 특수지역, 소형윈치·안전로프 등 벌목보조장치 필요 시
- 가슴높이지름 16cm 이상 적용, 16cm 미만은 솎아베기 품 적용
- 장비 설치보조·로프설치·안전관리·작업장관리 포함, 50m 이내 집재 포함
- **숲가꾸기 위험목 또는 인력 작업 시**: 벌목부의 100% 가산
- **산림병해충·소나무재선충병 발생지역 내 인력만으로 제거 시**: 보통인부의 100% 가산
- 장비: 굴착기 우드그랩 0.2㎥, 크레인 5톤 트럭탑재형 기준

> **소요인력 산출 예시** — 흉고직경 22∼30cm, 20본 제거 (숲가꾸기):
> 벌목부: 0.36인 × 20본 × 2배 = 14.4인
> 특별인부: 0.18인 × 20본 = 3.6인
> 보통인부: 0.36인 × 20본 = 7.2인

## 4-5. 벌도 위험목 점검

| 소요인력 | 인력구분 |
|---------|---------|
| 0.13인/ha | 벌목부 |

- 벌목작업 중 위험목(덩굴식물 피압목·초두부 고사목·수형 불균형 수목 등)을 사전 점검
- 눈높이에 표식을 남겨 안전관리 수행
- 가시거리 15m 기준, 1ha당 총 330m 보행 거리 필요
- 임내 보행 속도 2.5km/hr → 1ha당 0.132시간 소요
