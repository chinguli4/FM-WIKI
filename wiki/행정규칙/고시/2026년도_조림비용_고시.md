## 개요

이 고시는 2026년도 국고보조 조림사업에 적용되는 수종별·지역별·작업방법별 **조림비용 단가(보조금 산정 기준)**를 고시하는 것이다.

매년 새로 제정되며, 전년도 고시를 대체한다. 단가는 물가 변동, 인건비 기준 등을 반영하여 매년 갱신된다.

---

## 활용 목적

| 용도 | 내용 |
|------|------|
| 조림 보조금 산정 | 국·공·사유림 조림사업 국고지원금 계산 기준 |
| 작업설계서 작성 | 수종별 단가 기준으로 조림 설계 내역 작성 |
| 원가 계산 | 조림 사업비 적정 여부 판단 기준 |

## 2026년도 조림비용

| 구분 | 금액 |
|------|------|
| 재료비 | 2,519,149원 |
| 노무비 | 5,367,380원 |
| 경비 | 926,945원 |
| 일반관리비 | 705,077원 |
| 이윤 | 1,049,910원 |
| 부가가치세 | 1,056,846원 |
| 합계 | 11,625,000원 |

### 주요 산출 기준

| 항목 | 기준 |
|------|------|
| 묘목대 | 2025년산 낙엽송 1-1묘 3,000본/ha |
| 직접노무비 | 조림예정지정리 9인, 식재 12인, 소운반 1.2인, 표시봉 2.1인 |
| 간접노무비 | 직접노무비의 16.2% |
| 산업재해보상보험료 | 노무비의 5.86% |
| 고용보험료 | 노무비의 1.15% |
| 산업안전보건관리비 | 재료비 + 직접노무비의 2.07% |
| 기타경비 | 재료비 + 노무비의 4.6% |
| 일반관리비 | 재료비 + 노무비 + 경비의 8.0% |
| 이윤 | 노무비 + 경비 + 일반관리비의 15% |

## 산불피해지 매뉴얼 2025 단가와의 관계

산불피해지 조림복원 매뉴얼의 6cm 미만 입목 피해액 산정 예시는 2025년 조림비용 11,205,000원/ha를 사용한다. 2026년도 산정에는 이 고시의 11,625,000원/ha를 적용해야 한다.

| 기준 | 단가 | 비고 |
|------|------|------|
| 2025 매뉴얼 기준 | 11,205,000원/ha | 원문 붙임 5·서식 작성요령 |
| 2026 고시 기준 | 11,625,000원/ha | 2026년 산정 시 적용 |
| 차이 | +420,000원/ha | +3.75% |
