## 제1장 총칙 (제1조~제3조)

### 제2조 (정의)

#### ① "산림" 제외 토지 (법 §2①1호 단서)

1. 과수원, 차밭, 꺾꽂이순·접순 채취원
2. 입목·대나무가 생육하고 있는 **건물 담장 안의 토지**
3. 입목·대나무가 생육하고 있는 **논두렁·밭두렁**
4. 입목·대나무가 생육하고 있는 **하천·제방·도랑·연못**

#### ② "산림사업" 11종 (법 §2 3호)

법 [[법률_1장_총칙#제2조-정의|§2 정의]] 참고 — 임도·산림관리기반시설 / 조림·숲가꾸기·벌채 / 산림복원 / 산불예방·진화 / 산지관리법·백두대간법·산림재난방지법·국유림법·산림문화휴양법·산림교육법·수목장림·임업산촌진흥법·수목원정원법·사방사업법·도시숲법 / 그 밖에 농림축산식품부령.

#### ⑤ "임산물" 추가 6종 (법 §2 7호)

1. 조경수·분재수
2. 가지·꽃·열매·생잎·장작·톱밥·나무조각
3. 대나무류·초본류·덩굴류·이끼류
4. 산림버섯·떼
5. 숯(톱밥숯 포함)·수액(태워서 얻는 응축액 포함)
6. **목재제품** (합판·단판·섬유판·집성재·성형재·마루판·목재펠릿 등 — 「목재의 지속가능한 이용에 관한 법률」 §2 2호)

## 제2장 산림자원의 조성·육성 (제3조의2~제39조의5)

### 제1절 지속가능한 산림경영

- **제3조의2** (산림조림계획) — 시행령으로 정한 사항: 조림사업 기술 / 이력 관리 등 자원조성. 수립·변경 시 관계 행정기관장·시도지사 의견 청취
- **제4조** (산림지속성지수) — 「산림기본법」 §13 기준·지표 반영, 연구개발·모니터링·평가·환류
- **제5조** (조림예외지역) — **법 §10① 단서** 자연조림 인정 3가지 ★:
  1. 벌채지 안에 **어린나무 1ha당 2만본** 정도
  2. **상수리·굴참·갈참 등 참나무류 어린나무 1ha당 5천본** 정도
  3. 참나무류 벌채지에 **움싹발생 왕성한 참나무류 그루터기 1ha당 900개** 정도 균일 분포
  - ② 사후관리(어린나무 발생촉진·움싹 보육) 의무
- **제6조** (유휴토지 범위) — ①한계농지 / ②2년 이상 본래 용도 미사용 / ③지목이 전·답·과수원·목장용지·잡종지인 산림 전환 희망 토지. 등기·산림경영계획 작성·조림·숲가꾸기 비용 보조·융자

### 제2절 산림경영계획

- **제7조** (공유림 산림경영계획) — 「산림기본법」 §11 산림기본계획·**지역산림계획**·대상산림 현황·「자연공원법」 공원기본계획·공원계획에 부합. 시군구청장은 관할 시도지사 의견 청취
- **제8조** (작성단위) — 경영계획구 2종:
  - **공유림경영계획구** / **사유림경영계획구** (일반·협업·기업경영림 3종)
- **제9조** (계획서 작성) — ① 산림기술 진흥 및 관리에 관한 법률 시행령 별표3 **산림경영기술자** 작성. ② 계획서 5개 항목: 조림면적·수종별 수량 / 숲가꾸기 / 벌채·굴취 / 임도·작업로·운재로 / 산림소득. ③ 수종별 벌채시기·시설기준은 농림축산식품부령. ④ 세부사항은 산림청장
- **제10조** (실행확인) — 시군구청장은 계획 실행결과 확인·대장 기록·전산 입력
- **제11조** (인가취소) — ① **사업실적 50% 이하**가 §15 3호 "기준 이하". ② 예외: 산불·산사태 등 재난 / 산림소유자 사망 / 보전·이용·개발 지역 지정으로 제한

### 제3절 산림용 종묘 생산

- **제12조** (종묘생산업자 등록기준) ★
  - 산림용 종묘생산업자: **임업종묘기능사 이상** / 임업·녹지직 공무원 4년+종묘 1년 / 임업·원예·조경학과 졸업+종묘 1년 / 농업분야 특성화고 임업·조경학과 졸업+종묘 2년 / 상시고용 법인
  - 버섯종균생산업자: 버섯종균기능사 / 농업·생물·미생물학과 졸업+버섯종균 3년 / 외국 기술교육 1년+버섯종균 3년 / 농업특성화고 졸업+버섯종균 5년 + **살균실·접종실** 등 농림축산식품부령 시설
- **제13조** (변경등록) — 중요사항: 기술자격자 종류·수 / 생산시설 규모. 사유 발생 30일 이내 변경등록
- **제13조의2** (과징금) — **별표 1** 부과기준
- **제13조의3** (과징금 부과·납부) — 통지일부터 30일 이내 납부
- **제14조** (묘목생산사업 대행자격) — ① 종묘생산업 등록 후 **2년 경과 + 묘목 취급구분** / **5천㎡ 이상 포지** / 간이온실(용기묘). ③ 제외: 3년 90% 미달 / 5년 내 하도급 / 처분 종료 후 1년 미경과
- **제15조** (재해 보상) — 묘목생산비의 **100분의 2 이내** 재원 계상, 산림조합중앙회 대행 가능
- **제16조** (대행 종묘 가격) — 산림청장 고시 가격 적용. 지자체 시험·시범사업, 지자체별 특성 종묘는 지자체장 고시
- **제17조** (채종림등 공용·공공용 시설) — 도시·군계획시설 / 산업단지 / 군사시설 보호구역 / 학교시설

### 제4절 삭제 <2021.6.8>

### 제5절 산림사업의 시행 등

- **제20조** (정보제공 요청) — **기간통신사업자에 산림소유자 전화번호 요청** 시 산림청장 고시 정보제공 요청서 사용. 정보 이용자 8개 준수사항 (목적 외 사용 금지·파기·제3자 권리침해 금지·임의 제공·유출 금지·위변조 금지·분실·도난 방지·암호화·보안프로그램·기록 보존)
- **제21조** (산림시책 사업) — 법 §23①3호: 산림자원 조사 / 산림경영계획 작성 / 산림문화·휴양 시설 / 도시숲등 / 산림교육 시설
- **제22조** (산림경영능력자) — 50ha 이상 산림 소유·경영 법인·단체 / 산림기술 진흥 및 관리에 관한 법률 시행령 별표3 **기술중급 이상 산림경영기술자**
- **제23조** (긴급 산림보호) — 산사태·지진·해일 발생 응급조치 / 산불 예방·진화
- **제23조의2** (국유림영림단 등록요건) ★
  1. **구성인원 6~30명**
  2. 산림기술 진흥 및 관리에 관한 법률 시행령 별표3 **기능2급 이상 산림경영기술자** 비율:
     - 6~10명: **60%**
     - 11~30명: **50%**
  3. **기술초급 이상 산림경영기술자 1명 이상** 포함
- **제23조의3** (관리업무대행) — 산림조합·산림조합중앙회 / **한국산림재난안전기술공단** (산림재난방지법 §58) / **한국산지보전협회** (산지관리법 §46) / **한국산림기술인회** (산림기술법 §13). 단 산림사업 시행·대행·위탁 기관은 제외. 지정취소 추가사유: 재해예방 등 긴급조치 사항 미보고
- **제24조** (산림사업법인 사업 6종)
  1. **산림경영 계획·산림조사** 사업
  2. **숲가꾸기·병해충 방제** 사업
  3. **산림토목** 사업
  4. 자연휴양림 등 조성
  5. 도시숲등 조성·관리
  6. 숲길 조성·관리
  - 변경등록 중요사항: 법인 명칭 / 대표자 / 소재지 / 기술자격자 종류·수
- **제25조** (산림사업법인 등록기준) → **별표 2** (자본금·기술자·시설·장비)
- **제25조의2** (등록면제 추가) — 「목재의 지속가능한 이용에 관한 법률 시행령」 별표2 1호 **원목생산업 등록자**
- **제26조** (도급계약서 명시사항) — 15개 항목: 사업 구체 내용 / 도급금액 중 노임 / 착수·완성 시기 / 선급금·기성금 지급 시기·방법·금액 / 설계변경·중지·해제 시 손해부담 / 천재지변 면책 / 설계변경·물가변동 시 도급금액 변경 / 하도급대금 직접 지급 / 고용·산재·국민연금·건강보험료 / 인도검사·시기 / 완성 후 도급금액 지급 시기 / 위약금·지연이자 / 하자담보책임 / 폐기물 처리 / 그 밖에 합의
- **제27조** (특수산림사업지구 지정기준) ★
  1. **면적 300ha 이상**
  2. **자원조성 대상 면적 50% 이상**, **경영가능 지역 70% 이상**
  - 지정기간: **70년 범위 내**
- **제28조** (특수산림사업계획서) — 사업 목표 / 대상지 현황 / 산림사업 종류별 면적·사업량 / 연차별 계획
- **제29조** (특수지구 해제 공용·공공시설) — 방송·통신 / 석유 비축·저장 / 수도 / 하수도 / 군사시설

### 제6절 산림경영지도원

- **제31조** (자격기준) — ① 임업·조경 기능사 이상 / 임업·조경학과 졸업 / 고교 임업·조경학과 졸업 / 고졸·산림조합·관련 단체 2년 경력 / 산림조합·관련단체 3년 경력. ② 배치기준은 농림축산식품부령

### 제7절 산림자원의 조사 및 기술개발

- **제31조의2** (산림위성 관측망) — 산림위성 개발·지상국 구축·연구개발·기관 협력·전문인력 양성·관측자료 수집·생산·분석·배포·정확도 검증
- **제32조** (산림과학기술기본계획) — **국가과학기술자문회의 심의** 거침. 매년 시행계획 수립·시행·결과 보고. 산림청 소속 연구기관장 자체 중장기 기술개발계획 → 산림청장 승인
- **제33조** (공동연구 대상사업) — 신기술 개발·실용화 / 임업+타산업 기술협력 / 다른 기관 요청 / 국제공동연구 / 기초·기반 / 그 밖에. 매년 공동연구개발계획 수립 후 공모 또는 「국가연구개발혁신법」 §9④ 단서 지정
- **제34조** (협약체결) — 「국가연구개발혁신법」 §11①
- **제35조** (성과 사용) — 농림축산식품부령 계약 체결 + 기술사용료 납부. **사용료 면제 5가지**: 임업인 소득증대 / 수출진흥·수입대체 / 산림환경보전 / 중소기업지원 / 산림청장 인정
- **제36조** (보상금) — 「국가공무원 등 직무발명 처분·관리 및 보상 등에 관한 규정」 §16~§20·§22 따름
- **제37조** (출원중직무발명 사용)
- **제38조** (계약기간·사용료 산정) — **계약기간 3년 이내 원칙**, 산정방법은 농림축산식품부령
- **제39조** (지식재산처장 협의) — 국가승계·계약 승계·기술사용료 면제 등

### 제8절 무궁화의 보급 및 관리

- **제39조의2** (무궁화진흥계획 내용) — 무궁화 동산·거리 / 식재·관리기술 개발·교육 / 선양·홍보 / 연차 사업·재원 / 그 밖에
- **제39조의3** (연차별 시행계획) — 산림청장 **매년 1월 31일까지 수립**. 시도지사는 **전년도 12월 31일까지** 제출, 12월 31일까지 추진실적 제출
- **제39조의4** (실태조사) — 식재·보급·관리 / 생산기반. **정기조사 5년마다 / 수시조사 수시**
- **제39조의5** (지원 민간단체) — 국공립 연구기관 / 대학·산업대·전문대·기술대 / 비영리법인·단체

## 제3장 산림자원의 이용 (제40조~제47조)

### 제40조 (입목벌채등의 허가사항 변경) ★

법 §36① 후단 중요사항 변경:
1. **구역면적**
2. **벌채·굴취·채취면적**
3. **허가수량** (벌채는 **30% 이상 변경** 시)

### 제41조 (입목벌채등의 제한지역) ★

산사태·토사유출 등 산림재난 방지를 위해 **시군구청장·지방산림청장이 고시한 지역**. 고시 시 축척 **2만 5천분의 1 이상** 지적 표시 지형도 + 지번·면적 명시, 산림소유자 통지.

### 제42조 (제한지역 허가 가능 사유)

법 §36② 단서:
1. 병충해 예방·제거
2. 산불·산사태 등 재해 피해임지
3. **어린나무가꾸기·솎아베기 등 숲가꾸기**

### 제42조의2 (친환경벌채 지원기준 및 지원절차) ★

- ① 지원기준:
  1. 대상자: 친환경벌채 실시한 산림소유자 (토지·입목 소유자 다른 경우 토지소유자 동의 입목소유자)
  2. **예상 수익금 산정**: 입목 시장가격 − 벌채·운반 비용
  3. **지원대상 입목축적 범위**: 벌채구역 내 **총 입목축적의 20% 이하**
  4. **지원대상 벌채구역 면적**: **5만㎡ 이상**
- ② 지원신청서 → 시군구청장·지방산림청장
- ③ 전자정부법 §36 행정정보 공동이용으로 토지 등기사항증명서 확인, 현지조사 결과 반영
- ④ **5년 이내 친환경벌채 남겨진 입목 벌채 시 반환** — 일할계산 이자. 단 법 §23④ 긴급사유는 반환 면제
- ⑤ 세부사항은 [[../../행정규칙/고시/친환경벌채_운영요령]] 고시

### 제42조의3 (신고에 따른 입목벌채등) ★

법 §36⑤ 신고 가능 사유:
1. **산지전용허가·신고·산지일시사용허가·신고**에 따른 형질변경 면적 외 추가 벌채
2. 토사유출·산사태 우려 없는 지역에서 **병해충·산불·풍설해 등 자연재해로 넘어지거나 줄기 부러진 입목** 벌채
3. **불량치수림 수종갱신** 벌채
4. 그 밖에 농림축산식품부령 경미한 벌채·굴취

### 제43조 (임의로 하는 입목벌채등) ★

법 §36⑧ 허가·신고 없이 가능 9종:
1. **풀베기·가지치기·어린나무 가꾸기 벌채**
1의2. 지목 임야 아닌 토지에 **목재펠릿·목재칩 등 산림바이오매스에너지 생산** 위해 새로 조림 후 5년 이내 수시 벌채
1의3. **산불 예방 위해 건축물·가설건축물 외곽 25m 이내** 입목 벌채 (산림소유자 동의)
2. 「국유림법」 §9 국유림경영계획 승인 받은 경우
3. 「산림문화·휴양법」 §14 자연휴양림조성계획 승인
4. 「수목원·정원법」 §7 수목원조성계획 승인
5. 산림청장 소속 시험연구기관이 소관 국유림 시험·연구
6. 국가유산청장 소관 국유림 국가유산 보호
7. 산지전용허가·일시사용허가·신고에 수반되는 벌채
8. 토석 굴취·채취 수반 벌채 (토석채취·채석신고·토사채취신고 / 국유림 산지 토석 매각·무상양여)
9. 그 밖에 농림축산식품부령

### 제43조의2 (사전타당성조사 대상·전문기관) ★

- ① **대상**: **10만㎡ 면적 이상** 입목벌채등
- ② **전문기관 3개**:
  1. **한국산림재난안전기술공단** (산림재난방지법 §58)
  2. **한국산지보전협회** (산지관리법 §46)
  3. **한국산림기술인회** (산림기술법 §13)
- ③ **사전타당성조사 면제 사유 4가지**:
  1. 산불 예방·진화 긴급
  2. 병해충 방제 긴급
  3. 산사태·바람·비·눈·지진·해일 등 피해 예상·발생 긴급 복구
  4. 그 밖에 산림청장 고시

### 제43조의3 (사전타당성조사 절차·기준)

시군구청장·지방산림청장은 농림축산식품부령 **실시의뢰서** → 전문기관. **조사 기준 3가지**:
1. 면적의 적정성 등 사업내용 타당
2. 지형·환경 보전 등 **지속가능한 산림자원 관리 원칙** 부합
3. 산지경관·산림재난 영향 최소화

전문기관은 산불·산사태·숲가꾸기·벌채·개발행위허가 이력 자료 요청 가능 → 10일 내 회신. 산림자원정보체계·목재이용정보체계 활용 가능.

### 제43조의4 (전문기관 감독)

산림청장은 조사 방법·운용 / 전문인력 현황 / 그 밖에 감독. 자료 제출 요청 가능.

### 제43조의5 (사전타당성조사 결과 보관 — 3년)

보관 서류 3종: 사업계획서 / 현황 도면(전자도면 포함) / 결과서.

### 제43조의6 (입목벌채등의 심의위원회) ★

- ① 심의 사항:
  1. **입목벌채등 면적 20만㎡ 이상** 허가 사항
  2. 그 밖에 위원장 부의 사항
- ② **위원장 1 + 부위원장 1 포함 10명 이내**, 성별 고려
- ③ 위원장:
  - 산림청 소관 국유림: **지방산림청장**
  - 그 외: **시·군·구 부단체장** (2명 이상이면 지자체장 지명)
- ④ 위원 임명·위촉:
  1. 산림 분야 6급 이상 공무원
  2. 대학 산림분야 학과 조교수 이상
  3. **산림기술사** 자격 + 실무경험 3년 이상
  4. 그 밖에 위원장 위촉
  - **위촉 위원 과반수 이상**
- ⑤ 비공무원 위원 임기 **2년**
- ⑦ **재적 과반수 출석 + 출석 과반수 찬성** 의결
- ⑧ 해촉 사유 5가지: 심신장애 / 비위 / 직무태만·품위손상 / 회피 의무 미이행 / 본인 의사

### 제43조의7 (제척·기피·회피)

- 제척: 배우자·친족 / 안건 당사자 / 자문·연구·용역·감정 / 대리인
- 기피: 당사자 신청, 위원회 의결 (당사자인 위원은 의결 불참)
- 회피: 위원 자발적

### 제44조 (경제림육성단지 지정)

산림청장 지정 시 지형도(수치지형도 포함)·필지별 위치·면적 자료 작성 → 시도지사·시군구청장·지방산림청장 통보. 사유림 포함 시 인터넷 홈페이지 공고.

### 제45조 (기업경영림 경영 — 사업 7종)

1. 펄프업
2. 탄광업
3. **연간 3천㎥ 이상 국내산 원목 이용·가공**
4. 파티클보드 제조업
5. 목재칩 제조업
6. 목재펠릿 제조업
7. 「목재의 지속가능한 이용에 관한 법률」 §18 신기술 적용 제조업

### 제45조의2 (기업경영림 해제 공용·공공용 시설 — 15종)

공장·산업시설·물류 / 군사시설 / 사방시설·하천시설 / 도로·철도·전원·석유비축·가스공급·공항·LPG 저장 / 궤도시설 / 방풍·방화 / 관측시설 / 청사 / 임업 시험·연구시설 / 시험림 유지·관리 시설 / 생태숲·산림생태원 기능증진 / 수목원·정원 / 산림욕장 / 국가통신·전기통신 / **풍력 설비** (신재생에너지).

### 제47조 (임산물의 유통제한 — 솔잎)

법 §40①: **솔잎**만 해당 (구 송이는 2017-12-12 삭제).

    1. 산림복원 분야 **석사 이상** 또는 **산림기사 이상 자격 3명 이상**

- **제48조의9** (산림복원 재료) — **자생식물·자연재료** 사용 의무. 채집·채취 기준: 산림복원 대상지 주변 또는 **고도·기후대 유사 지역**. 공급 어려우면 연구기관·지원센터 자문 후 대체 가능

- **제52조의2** (특별수종육성권역) ★ — 요건: **단일 수종 집단 분포** + **역사적·문화적·자원적 가치** 보호·육성 필요. 지정 시 시도지사 통보 + 홈페이지 공고 (명칭·위치·면적·지정일)

- **제62조** (녹색자금회계) — 독립 설치, 발생주의 회계처리, 정부 회계연도
- **제62조의2** (운용계획 수립·관리) — 고려: 설치목적 적합 / 중복 방지 / 수입·지출 균형 / 안정성·유동성·수익성 / 지원대상 선정기준·집행·평가·여유자금
- **제63조** (운용·관리 위탁) — **한국산림복지진흥원**에 위탁. 매년 녹색자금관리계획 수립 → 산림청장 승인. 매분기 종료 후 **5일 이내** 집행·운용현황 제출. 결산보고서: 회계연도 종료 후 **1개월 이내** 산림청장 제출 → 승인
- **제64조** (녹색자금운용심의회) — 위원:
  1. 산림청 공무원 1
  2. 산림조합중앙회장 추천 1
  3. 한국산림복지진흥원 추천 이사 1
  4. 임업계·학계·언론계·시민단체 등 한국산림복지진흥원 추천 + 산림청장 위촉 **6명 이내**
  - 위촉 위원 임기 **2년**, 1회 연임 가능
  - **구성원 과반수 출석 + 출석 과반수 찬성** 의결. 사업제안 위원 의결권 없음
- **제64조의2** (해촉) — 심신장애·비위·직무태만·본인 의사

### 제69조 (산림사업보조금 반환) ★

- ① 반환 사유:
  1. **조림·숲가꾸기 산림** — 산지의 형질변경 → 면적 비율 반환
  2. **임도시설** — 폐지 시 → 거리 비율 반환. 잔여임도 이용 불가 시 전액 반환·우회임도 시설
  - 면제: 국가·지자체 공용·공공용 / 천재지변·부득이한 사유
- ② **준공일부터 5년 이내** 산지의 형질변경 허가 시 반환
- ③ 농림축산식품부령 이자 가산 (일할계산)

### 제70조 (보고·검사) — 산림용 종자·묘목 품질·규격 기준은 산림청장
