# 산림기술 진흥 및 관리에 관한 법률 (산림기술법)

> 종류: 법률 (산림기술자·산림기술용역업 등록·자격 관리·벌점·벌칙의 근거법)
> 현행 법률: 법률 제21405호 (2026-02-27 공포, **2026-08-28 시행**)
> 시행령: `raw/legalize-kr/kr/산림기술진흥및관리에관한법률/시행령.md` 참조
> 시행규칙: 농림축산식품부령 (별표 2·3·4 포함)
> 소관: 산림청
> 출처: <https://www.law.go.kr/법령/산림기술진흥및관리에관한법률>
> 원문: `raw/legalize-kr/kr/산림기술진흥및관리에관한법률/`

본 법은 2017-11-28 제정(법률 제15080호) 시 산림자원법 §27·§30(산림기술자) 등을 분리·이관하여 만든 법령이다. 산림기술자 자격증 관리·산림기술용역업 등록·산림기술자 양성·산림사업 부실측정·안전관리·벌칙의 통일적 근거를 제공한다.

## 담당자 빠른 참조

자격증 대여·이중취업·등록증 대여 사건 처리 시 흐름:

1. **사건 인지·조사**: §11⑤ 출입·질문·서류조사 (국가기술자격법 §15의3 준용)
2. **처분 트랙 결정** — [[../../산림작업/자원조성/산림기술자_자격관리_체계]] 5트랙 매트릭스 참조
3. **사전통지·의견청취**: 행정절차법 §21·§22 — 자격취소/등록취소는 **청문 의무**(시작 10일 전 통지), 정지/과징금은 **의견제출 의무**(기한 10일 이상)
4. **처분 결정**: 별표 2·3 세부기준 적용 → 가중(3년)·감경(자격정지 1/2·영업정지 1/2·소상공인 70/100) 검토
5. **처분서 작성·송달**: 행정절차법 §23 이유 제시 + §24 문서 + §26 행정심판·소송 고지 안내
6. **후속**: 자격증/등록증 반납(7일), 발주청 통지(10일/지체없이), 산림기술정보체계 등재, 계약 계속수행 판단(30일)

벌점은 별도 트랙: 시행규칙 §18·별표 4 → 30일 의견제출 → 5명 이상 검토 → 15일 내 결과 통지.
