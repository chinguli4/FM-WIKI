## 제1조 (목적) · 제2조 (정의)

**§2①** "대통령령으로 정하는 임업기계장비"(법 §2②2다목) = 「임업 및 산촌 진흥촉진법」 §2-3의4 임업기계장비

**§2②** "그 밖에 산림사업에 관한 기술"(법 §2②2라목):
1. 산림기술의 타당성 검토
2. 정보통신시스템을 이용한 산림기술 정보처리
3. **산림사업의 공정(工程) 분석 및 품셈**(단위당 비용 추정)

**§2③** 발주청 범위(법 §2-8) — 다음을 포함:
1. 「공공기관운영법」 §5 공기업·준정부기관
2. 「지방공기업법」 지방공사·지방공단
3. 국가·지방자치단체 출연기관
4. 국가·지방자치단체·공기업·준정부기관이 사업시행을 위탁한 자
5. 「사회기반시설에 대한 민간투자법」 사업시행자 (자본금 1/2 이상 출자 + 관계 중앙행정기관장이 발주청 승인한 경우만)

## 제3조 (산림기술진흥계획 수립)

**①** 산림청장은 진흥계획 수립·변경 시 미리 관계 중앙행정기관장·시·도지사와 **협의**. 경미한 사항(농림축산식품부령 정함)은 제외.
**③** 수립·변경 시 **관보 고시** + 관계 중앙행정기관장·시·도지사 통보.

## 제4조 (산림기술개발전문기관 지정)

- **①** 지정기준 → **별표 1**
- **②** 농림축산식품부령으로 신청
- **③** 지정서 발급 + **산림청 인터넷 홈페이지 공고**
- **④** 시정명령·지정취소 세부기준 → **별표 1의2**

## 제5조 (산림기술정보체계 정보범위)

법 §6②7호 "대통령령으로 정하는 사항":
1. 산림기술자 교육·훈련 정보 (법 §7②)
2. **산림사업 부실측정·벌점** 정보 (법 §24)
3. **산림기술자등 배치 정보** (법 §25)
4. 산림사업 안전관리·안전교육 정보 (법 §26·§27)

## 제6조 (산림기술정보체계 운영 위탁대상)

1. 산림기술개발전문기관
2. 산림기술자 교육기관
3. **한국산림기술인회**(기술인회)

## 제7조 (교육·훈련 대상 산림기술자)

다음 기관·단체·사업자에 **소속된 산림기술자**:
1. 산림기술용역업자
2. 국유림영림단(산림자원법 §23의2③)
3. 산림사업법인(산림자원법 §24①)
4. **산림조합 및 중앙회**
5. 원목생산업 등록자

## 제8조 (산림기술자 교육기관 지정)

**①** "대통령령으로 정하는 산림기술과 관련된 기관·단체"(법 §7④):
1. 기술인회
2. **대학** 중 산림기술 관련 교육과정 개설
3. **비영리법인** 중 산림기술 관련 교육과정 개설
4. 산림기술 관련 연구·업무 학회·기관·단체

**②** 교육기관 지정기준 → **별표 2**
**④** 신청 후 **30일 이내** 지정서 발급 + 산림청 홈페이지 공고

## 제9조 (교육기관 지정취소)

**①** 법 §7④3호 "대통령령으로 정하는 사항":
1. 지정 후 정당사유 없이 **1년 이내 교육과정 미개설**
2. 정당사유 없이 **1년 이상 교육기관 미운영**
3. **지정받은 강의실·실습림이 아닌 곳에서 교육과정 운영**

## 제10조 (산림기술자 종류·자격·업무) ★

**①** 산림기술자의 **종류·자격 요건·업무 범위 → 별표 3** (산림경영기술자·산림공학기술자·산림복원기술자·녹지조경기술자 등 종류별)
**②** 경력 인정 세부기준은 산림청장 고시 → [[../../행정규칙/고시/산림기술자_경력인정_세부기준]] 참조 (별도 ingest)

## 제11조 (한국산림기술인회 정관 기재사항)

회원 자격·임원·업무 외 정관 포함:
목적 / 명칭 / 사무소 소재지 / 사업·집행 / 회원 가입·탈퇴 / 회원 권리·의무 / 비용 부과 / 손실 처리 / 회의 / 회계 / 잔여재산 처분

## 제12조 (산림기술용역업 등록) ★

**①** 법 §15① 후단 "대통령령으로 정하는 산림사업" (제1호다·라목 한정 등록 시):
1. 유아숲체험원 조성사업
2. 숲길 조성사업
3. 도시숲·생활숲·가로수 조성·관리사업
4. 수목원 조성·등록·운영 사업

**②** 법 §15①1마목 "대통령령으로 정하는 자" = 별표 3 **녹지조경기술자**(별표 4 녹지조경업 등록 한정)

**③** 등록 요건 + 업무 범위 → **별표 4**

## 제13조 (설계·감리 대상 산림사업) ★★

**①** 법 §15③ "대통령령으로 정하는 종류·규모 이상":

**1호. 국가·지방자치단체 보조·지원 산림사업 중**:

| 가목 | 조림사업 | 3만㎡ 이상 |
|------|---------|----------|
| 나목 | 벌채사업 | 3만㎡ 이상 |
| 다목 | 산림병해충 방제사업 | 100만㎡ 이상 |
| 라목 | **솎아베기 동반 숲가꾸기** | **50만㎡ 이상** |
| 마목 | 임도사업 | 전부 (감리는 공사비 **2천만원 이상**) |
| 바목 | 사방사업 | 전부 (감리는 공사비 **1억원 이상**) |
| 사목 | 유아숲체험원·산림교육센터 조성 | 전부 (감리 공사비 **4천만원 이상**) |
| 아목 | 자연휴양림·산림욕장·치유의 숲·숲길·숲속야영장·산림레포츠시설 조성 | 좌동 |
| 자목 | 도시숲·생활숲·가로수 조성·관리 | 좌동 |
| 차목 | 수목원 조성 | 좌동 |
| 카목 | 수목장림 조성 | 좌동 |
| 타목 | **산림복원사업**(백두대간·사방·산림자원·산지관리) | 좌동 |

**2호. 산지관리법 §39①·②·§44① 산지복구·중간복구·불법산지전용지 복구** (감리는 「산지관리법 시행령」 §48의2 면적 이상)

**②** **시행업자 = 감리자 동일인 금지**. 숲가꾸기는 **설계자 = 시행업자, 설계자 = 감리자도 동일인 금지** (분리발주 의무).

## 제14조 (등록 변경사항 중요사항)

법 §15④ "대통령령으로 정하는 중요한 사항":
1. 산림기술용역업 명칭
2. 대표자
3. 소재지
4. **소속 산림기술자가 보유한 기술자격의 종류 또는 수**

## 제14조의2·제14조의3 (과징금)

- **§14의2** 과징금 부과기준 → **별표 4의2** (법 §18④ — 3천만원 이하)
- **§14의3** 부과·납부 절차: 산림청장이 위반행위 종류·금액 명시한 서면 통지 → 통지일부터 **30일 이내** 수납기관 납부 → 영수증 발급·산림청장 통보

## 제15조 (산림기술자등 배치) ★

**①** 배치기준 → **별표 5**
**②** **산림사업 착수와 동시에** 배치
**③** 배치·변경 시 농림축산식품부령에 따라 **발주청 확인**

## 제16조 (안전관리계획 수립) ★

**①** 안전관리계획 대상 산림사업:
1. **§13① 각 호 산림사업 모두** (설계·감리 대상과 동일)
2. 그 밖에 발주청이 안전관리 필요하다고 인정한 산림사업

**②** 수립 기준 → **별표 6**
**③** 산림사업 **착수 전** 발주청 제출·승인
**④** 발주청은 제출 후 **7일 이내** 심사:
- **적정**: 안전조치 구체·명료, 안전성 충분
- **조건부 적정**: 보완 필요
- **부적정**: 중대 결함, 안전사고 우려

**⑤** 적정·조건부 적정 시 **승인서 발급**(조건부는 보완사항 포함)
**⑥** 부적정 시 다시 수립 → 착수 전 제출·승인 (④·⑤ 준용)

## 제17조 (안전점검 대가)

**①** 안전점검 대가 합산:
1. 직접인건비
2. 직접경비 (여비·차량운행비)
3. 간접비
4. 기술료
5. 조사·시험비 등

**②** 세부 산출기준은 산림청장 고시.

## 제18조 (안전관리조직)

**①** 산림사업시행업자는 **안전총괄책임자 1명 + 분야별 안전관리담당자 1명 이상** 두어야 함.

**②** 직무:

| 안전총괄책임자 | 안전관리담당자 |
|---|---|
| 안전관리계획서 작성·제출 | 안전총괄책임자 직무 보조 |
| 비상동원·응급조치 | 안전점검 실시 |
| 안전관리비 집행·확인 | 안전교육 실시 |
| 안전관리 시설·장비 지원 | |
| 안전점검 지휘·감독 | |
| 안전교육 지휘·감독 | |

## 제19조 (안전교육 실시)

**①** 안전총괄책임자·안전관리담당자는 **산림사업 작업일마다 작업 시작 전** 작업 인력 대상 안전교육 실시.
**②** 교육 내용: 작업 시행방법·세부 시행순서·시행 주의사항
**③** 사업 준공 후 기록 내용을 발주청에 제출.

## 제20조 (권한 위임·위탁) ★★

**① 지방산림청장 위임** (법 §30①):
1. **§11⑤ 명의대여 위반 조사**
2. **§12① 산림기술자 자격취소·자격정지**
3. **§18①·② 산림기술용역업 등록취소·영업정지·등록말소**
3의2. **§18④·⑤ 과징금 부과·징수**
4. §22① 산림기술용역업자 지도·감독
5. **§33 과태료 부과·징수**

> 산림기술자 자격처분 + 등록취소 + 과징금 + 과태료 모두 **지방산림청장**(동부지방산림청장 포함)이 처분권자.

**② 산림청장 위탁 가능 업무** (법 §30②):
1. §9 자격증 신청·발급
2. §10 경력등 신고·증명서 발급
3. §12③ 자격증 반납·경력 기록 수정·말소
4. §15①·② 산림기술용역업 등록 접수·등록증 교부
5. §15④ 변경·휴업·폐업 신고 접수
6. §18③ 등록증 반납 접수
7. §19② 지위승계 신고 접수
8. §23 산림사업 실적 신고·증명서 발급
9. **§24③ 벌점 종합관리**

**③** 수탁기관: **기술인회** 또는 산림기술용역·산림사업시행 관련 기관·단체로 산림청장 인정.

## 제21조 (고유식별정보 처리)

산림청장(위임·위탁 포함) 또는 발주청은 다음 사무에서 **주민등록번호** 처리 가능: 자격증 발급·경력증명서·산림기술자 행정처분·산림기술용역업 등록·산림기술용역업자 행정처분·지위승계.

## 제22조 (규제 재검토)

3년 주기 재검토: 산림기술개발전문기관 지정기준, 교육기관 지정기준·지정취소 사유, 산림기술자 종류·자격·업무, 산림기술용역업 등록 요건, 산림사업 설계·감리 범위, 산림기술자등 배치기준 (기준일: 2021-01-01).

## 제23조 (과태료 부과기준)

법 §33①에 따른 과태료 부과기준 → **별표 7**
