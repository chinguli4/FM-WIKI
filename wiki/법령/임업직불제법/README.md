## 2026년 개정 핵심

2026-03-10 법률 개정은 농업 소규모농가직접지불금과의 중복 제외 기준을 신청일의 **직전 연도**에서 **해당 연도**로 바꾸었다(법 제8조제3항제3호, 제9조제1항 단서). 2026-05-19 시행령·시행규칙 개정본이 2026-06 법령 편람에 반영되었다.
