# 산지관리법 — 국유림 재산관리 연계 지도

> 소관: 산림청
> 현행 검증일: 2026-07-13
> 현행 법률: 법률 제21406호, 2026-05-28 시행
> 시행령: 대통령령 제36066호, 2026-02-01 시행
> 시행규칙: 농림축산식품부령 제740호, 2025-10-31 시행

## 현행성 주의

`raw/legalize-kr/kr/산지관리법/법률.md`는 법률 제21467호의 **2026-09-18 시행 예정본**까지 반영한다. 2026-07-13 현재 사건에는 법률 제21406호 시행본을 적용하고, 시행 예정 개정은 효력 발생 전까지 판단 근거로 사용하지 않는다.

## 왜 국유재산 업무의 필수 법령인가

국유림 재산 사건에는 서로 다른 세 질문이 겹친다.

| 질문 | 주된 법령 | 판단 내용 |
|---|---|---|
| 국가 소유 재산을 사용할 권원이 있는가 | 국유림법·국유재산법 | 보전/준보전국유림, 행정/일반재산, 사용허가·대부·처분 |
| 산지에서 해당 행위를 할 수 있는가 | 산지관리법 | 보전산지 행위제한, 전용·일시사용 대상과 허가기준 |
| 훼손 뒤 무엇을 납부·복구해야 하는가 | 산지관리법령·고시 | 대체산림자원조성비, 복구비, 설계·감리·준공·반환 |

국유림법 §21②의 의제가 성립해도 두 번째와 세 번째 질문이 사라지지 않는다. 산지관리법의 협의서류·허가기준·부담금·효력발생·복구 기준을 충족해야 한다.
