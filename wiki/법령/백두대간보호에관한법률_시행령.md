# 백두대간 보호에 관한 법률 시행령

> 종류: 대통령령  
> 법령번호: 제35811호 (2025-10-01 최종개정)  
> 시행: 2025-10-01  
> 소관: 산림청  
> 출처: https://www.law.go.kr/법령/백두대간보호에관한법률시행령  
> 원본: raw/legalize-kr/kr/백두대간보호에관한법률/시행령.md

## 제1조 (목적)

「백두대간 보호에 관한 법률」에서 위임된 사항과 그 시행에 필요한 사항을 규정한다.

---

## 제2조 (정맥) — 법 제2조제1호의2

"대통령령으로 정하는 산줄기"는 다음 9개 정맥:

| 정맥 | 구간 |
|------|------|
| 한북정맥 | 강원특별자치도 세포군 식개산 → 운악산, 한강봉 → 경기도 파주시 장명산 |
| 낙동정맥 | 강원특별자치도 태백시 매봉산 → 운주산, 엄광산 → 부산광역시 사하구 몰운대 |
| 한남금북정맥 | 충청북도 보은군 속리산 → 보광산, 소속리산 → 경기도 안성시 칠장산 |
| 한남정맥 | 경기도 안성시 칠장산 → 광교산, 가현산 → 경기도 김포시 문수산 |
| 금북정맥 | 경기도 안성시 칠장산 → 백월산, 오서산 → 충청남도 태안군 지령산 |
| 금남호남정맥 | 경상남도 함양군 영취산 → 장안산, 마이산 → 전북특별자치도 진안군 조약봉 |
| 금남정맥 | 전북특별자치도 진안군 조약봉 → 싸리재, 계룡산 → 충청남도 부여군 부소산 |
| 호남정맥 | 전북특별자치도 진안군 조약봉 → 내장산, 무등산 → 전라남도 광양시 백운산 |
| 낙남정맥 | 경상남도 산청군 지리산 → 옥산, 불모산 → 경상남도 김해시 분성산 |

## 제8조 (핵심구역에서의 허용행위) — 법 제7조제1항 세부

### 제2호 — 공용·공공용 시설

국가·지방자치단체 또는 정부 출자 50% 이상 기업체가 설치하는 다음 시설:

  1. 도로·철도·하천·궤도시설 또는 송전탑 (송전탑: 작업장은 최소기간·최소면적으로만)
  2. 방풍시설, 방화시설 또는 사방시설
  3. 국가통신시설
  4. 기상시설
  5. 지진·화산 관측소
  6. 광해방지시설

### 제4호 — 산림보호·임업시험연구 시설

  1. 병해충 예방·구제 시설
  2. 산불·산사태 등 산림재해 예방·복구 시설
  3. 산림보호와 산림자원 보전·증식을 위한 임산물 운반로 및 작업로
  4. 시험림, 산림보호구역, 보호수의 보전·관리 또는 시험연구 시설

### 제7호 — 조건부 광산 개발

  1. 광물의 탐사·시추시설 설치
  2. 갱구·선별·가공시설·진입로 포함 면적 **2만㎡ 미만**인 광물의 갱내채광
  3. 2004년 12월 31일 이전 인허가 받은 석회석 노천채광

### 제8호 — 지역주민 생활 관련 시설 (산지 외 토지에만)

  1. 농업생산에 필요한 시설 (농지법 시행령 별도 기준)
  2. 농작물재배업, 축산업 및 임업을 위한 시설
  3. 농가주택 및 부대시설로서 부지면적 합계 **660㎡ 미만**

### 제9호 — 부대시설

전기설비, 전기통신설비, 수도시설, 하수도

### 제10호 — 임시시설

진입로, 현장사무소, 작업장, 지질·토양 조사·탐사 시설, 주차장, 화장실, 창고, 숙소, 식당, 정화시설, 재해방지시설, 자재 적치·운반 시설

## 제9조 (완충구역에서의 허용행위) — 법 제7조제2항 세부

### 제2호 — 산림공익시설

산림욕장, 숲속수련장, 생태숲 등

### 제3호 — 산림경영 관련 시설

  1. 국가·지방자치단체가 설치하는 임도
  2. 임업인이 설치하는 다음 시설 — 해당 시설 부지면적 **3,000㎡ 미만**:
    - 산림경영관리사 (주거용 제외)
    - 임산물 건조·보관 시설
    - 비료·농약·기계 등 임업용 기자재 보관 시설
  3. 임산물소득원 생산·가공·유통 시설 — 부지면적 **3,000㎡ 미만**

### 제4호 — 교육·연구·기술개발 시설

국가과학기술자문회의에서 심의한 우주항공기술개발 관련 시설

### 제5호 — 주택·종교시설 규모 기준

  1. 증축: 종전 것 포함 종전 규모의 **130% 이하**
  2. 개축: 종전 것 포함 종전 규모의 **100% 이하**

### 제6호 — 전력·석유·가스 공급시설

전력·석유·가스 공급시설, 석유비축·저장시설, 상하수도시설

### 제7호 — 석회석 노천 채광

2004년 12월 31일 이전 인허가 받은 도별 개발면적 안, 훼손지 복구 조건: 면적 **3만㎡ 이상 30만㎡ 미만**

### 제8호 — 홍보·교육 시설

백두대간홍보관, 역사문화관, 생태교육장 등 — 국가·지방자치단체 설치

### 제9호의2 — 축산업 관련 체험시설

총면적 **1,000㎡ 이하** (판매·간이매점 등 제3호 시설은 **330㎡ 이하**)

## 제10조 (개발행위에 대한 사전협의 절차)

### 협의 불필요 행위

  1. 국방·군사시설 중 군사상 고도 기밀 또는 군사작전 긴급 시 (사전에 산림청장과 협의한 경우)
  2. 국가유산·전통사찰의 복원·보수
  3. 기후에너지환경부장관 또는 산림청장이 실행하는 환경보전사업·산림사업

### 협의 시 고려사항

  1. 백두대간 단절 없을 것
  2. 산림·경관·야생동식물 보호에 지장 없을 것
  3. 특별 보호가치 있는 지역에 해당하지 않을 것
  4. 다른 법률로 지정된 지역의 제한 행위에 해당하지 않을 것

### 협의 절차

- 협의 요청서 + 사업계획서, 토지명세서, 소유권 증명서류, 지형도(1:25,000 이상), 실측도(1:6,000~1:1,200) 제출
- 산림청장: 협의요청 받은 날부터 **30일 이내** 결과 통보 (부득이한 경우 10일 연장 가능)

## 제11조의3 (소득감소분 지원대상 요건)

다음 요건을 **모두** 충족하는 산림을 소유하고 벌채를 하지 아니하는 자:

  1. 보호지역 지정 당시 공익용산지 또는 입목벌채제한지역 **안에 있지 아니**할 것
  2. 입목의 평균수령이 「산림자원법 시행규칙」 제7조제2항 **별표 3 일반기준벌기령** 이상일 것

---

## 제11조의4 (소득감소분 지원기준)

> 지원금액 = 예상수익금의 **이자상당액**

- **예상수익금**: 입목의 시장가격 − 벌채·운반 등 소요비용
- **이자상당액**: 지원대상연도 1월 1일 현재 전국 영업구역 은행의 **정기예금금리 평균** 적용

---

## 제11조의5 (소득감소분 지원절차)

  **①** 지원대상자 → 소득감소분 지원신청서(별지 제3호서식) + 입목등기부 등본 → 산림소재지 관할 시장·군수 제출  
  **②** 시장·군수: 현지조사 실시 → 지원금액 산출  
  **③** 시장·군수: 지원금액·지급시기 통지  
  **④** 시장·군수 → 도지사 제출 → 지원대상연도 **12월말**까지 산림청장에 제출  
  **⑤** 산림청장이 지원금액 지급

## 제13조 (정책협의체의 구성·운영)

| 구분 | 광역협의체 | 지역협의체 |
|------|-----------|-----------|
| 대상 | 사업 대상지가 2개 도 이상에 속하는 경우 | 사업 대상지가 1개 도 관할 내에 속하는 경우 |
| 위원장 | 산림청 소속 **고위공무원단** 일반직공무원 | **지방산림청장** |
| 구성 | 위원장 포함 **15명 이내** | 위원장 포함 **15명 이내** |

**위원 구성**: 관계기관·지자체 소속 공무원, 지역주민·이해관계자, 전문가 — 성별 고려

**업무 (공통)**: 이해관계 조정, 의견수렴 및 제도개선 건의, 그 밖에 보호·관리에 필요한 사항 협의

## 제15조 (권한의 위임)

산림청장은 다음 권한을 위임:

| 권한 | 위임 대상 | 비고 |
|------|----------|------|
| 핵심구역 **5,000㎡ 미만** 개발행위 사전협의 | 지방자치단체 소유 → 도지사 / 국가 소유 → 지방산림청장 | |
| 완충구역 **1만㎡ 미만** 개발행위 사전협의 | 위와 동일 | |
| 지역단위 정책협의체 구성·운영 | 지방산림청장 | |
| 토지등 매수청구서 접수·통보 | 지방산림청국유림관리소장 | |
| 소득감소분 지원금액 지급 | 시장·군수 | |

> ⚠️ **예외**: 핵심구역+완충구역 동시 개발 시 총면적 1만㎡ 이상이거나 핵심구역 5,000㎡ 이상이면 위임하지 않고 산림청장이 직접 협의
