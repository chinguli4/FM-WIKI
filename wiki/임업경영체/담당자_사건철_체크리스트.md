# 임업경영체·직불금 담당자 사건철 체크리스트

> 용도: 신청접수, 요건심사, 현지조사, 등록·거부, 지급, 사후점검을 한 사건철에서 추적하기 위한 표준 구조

## 0. 업무 구분표

- [ ] 임업경영체 신규·변경·갱신등록 사건인가?
- [ ] 임산물생산업 직불금 사건인가?
- [ ] 육림업 직불금 사건인가?
- [ ] 소규모임가·면적·육림업 중 신청유형이 명확한가?
- [ ] 경영체 등록관할과 직불금 신청관할을 구분했는가?

## 1. 접수

- [ ] 신청서·접수증·보정요구서
- [ ] 신청자 인적·법인·계좌 정보
- [ ] 경영체 등록·변경 확인서
- [ ] 산지·임소반·품목·면적 명세
- [ ] 해당 연도 공고·시행지침·신청마감일 확인
- [ ] 2026년 비대면·온라인·방문 신청경로와 실제 공고기간 확인

## 2. 권원·면적

- [ ] 토지·임야대장, 등기사항
- [ ] 임대차·사용대차·상속·증여 서류
- [ ] 임대차 종료일이 8월 31일 이후인지, 계약·신청 면적과 당사자가 일치하는지 확인
- [ ] 재산세 납부자와 소유자가 다르면 납부·계약 갱신 여부 및 관계 증빙
- [ ] 지적·현황 경계, 중복·분할·공유지분 검토
- [ ] 휴경·폐경·전용·일시사용·개발예정지 면적 제외
- [ ] 임도·작업로·운반로는 일시사용신고 수리·의제 인·허가 확인

## 3. 실제 경영·실적

- [ ] 품목별 최소면적·표고자목·종묘등록 기준
- [ ] 복수 품목 산식과 면적 구획
- [ ] 판매·자재·사업실행 실적의 신청자·필지·품목 일치
- [ ] 육림업은 산림경영계획 인가서와 조림·풀베기·솟아베기 실적
- [ ] 발췌 HWP의 3년 유예를 적용했다면 정식 근거·전문·현행성 별도 첨부

## 4. 직불금 특별요건

- [ ] 2019-04-01~2022-09-30 등록 및 2022-10-01 유효 이력
- [ ] 국·공유림, 전용·일시사용, 타 직불금 중복, 개발지·휴경지 제외
- [ ] 직전 1년 계속·연 60일 종사
- [ ] 임산물생산업의 면적·판매액, 육림업의 면적·소유·육림실적
- [ ] 농촌 외 거주·소재자의 주업 요건
- [ ] 농업 외 종합소득 3,700만원 기준
- [ ] 소규모임가는 시행령 별표 1과 해당 연도 지침의 구성·면적·거주·종사·소득 전 요건
- [ ] 소규모임가 가족관계 인적정보 작성표와 분리세대 포함 여부

## 5. 조사·심사

- [ ] 현지사진·조사조서·위치도·면적산출
- [ ] 조사위원회 위원 위촉·제척·기피·회피 확인
- [ ] 서면·현지조사 결과와 행정정보 공동이용 조회결과
- [ ] 적합·부적합 판단표에 근거조문과 사실관계를 분리 기재
- [ ] 등록증 또는 사유·재심사 안내를 포함한 거부통보서

## 6. 지급·사후관리

- [ ] 구간별 면적·단가·상한 산출표
- [ ] 중복지급·계좌·등록변경 최종 확인
- [ ] 연 2시간 교육, 형상·기능, 농약·비료, 경영계획·입목 준수자료
- [ ] 등록증·계약·경영·교육 서류 2년 보관 안내
- [ ] 변경일부터 14일 이내 경영정보 변경등록 여부
- [ ] 공동활동 연 1회 이상, 기록은 신청연도 다음 연도부터 2년 보관 여부
- [ ] 준수사항 항목별 10%, 다수·반복위반 감액 산식과 의견제출 기록
- [ ] 조사 7일 전 통지, 조사결과·의견제출·처분·환수 기록

## 위험 신호

- 등록 이력을 사후에 소급 정리하여 직불금 역사요건을 맞추려는 사례
- 신청 직전 산지·권리 분할, 공유지분 취득, 임대차 급증
- 영수증의 명의·필지·품목·시기가 신청내용과 불일치
- 산림경영계획과 현장 수종·작업실적·임소반 경계 불일치
- 교육·준수사항 자료를 지급 직전에 일괄 작성한 흔적
