# 임업경영체 품목별 등록기준

> 근거: 「임야를 생산수단으로 하는 농업경영정보 등록기준의 세부 내용 및 운용 규정」 제6조

## 농업인 등록기준

정당한 권원이 있는 생산수단에서 다음 면적·규모 중 하나를 충족하고, **자기의 계산과 책임으로 수행한 판매실적 또는 농자재 사용 등 임업경영 실적**이 있어야 한다. 면적은 휴경·폐경을 제외한 실제 경영면적이다.

| 구분 | 최소 기준 | 특기사항 |
|---|---:|---|
| 수실류·약초류·약용류·수목부산물류·관상산림식물류(분재 제외)·그 밖의 임산물 | 1,000㎡ | 산양삼은 생산신고확인증의 파종·식재구역 면적 |
| 버섯류·산나물류·분재 | 300㎡ |  |
| 표고자목 | 20㎥ | 120cm 자목은 아래 환산개수 사용 가능 |
| 산림용 종자·묘목 | 면적기준 대신 법정등록 | 산림용 종자·묘목생산업자 등록 + 생산·판매실적 |
| 기타 목본·초본식물 | 30,000㎡ | 경영체 소유 생산수단, 인가받은 산림경영계획의 임·소반 전체면적 |

## 표고자목 환산개수

자목 길이가 120cm인 경우의 기준이며, 길이가 다르면 환산개수를 조정한다.

| 끝지름 | 최소 개수 |
|---|---:|
| 11cm 미만 | 1,378개 |
| 11cm 이상 16cm 미만 | 652개 |
| 16cm 이상 20cm 미만 | 417개 |
| 20cm 이상 | 290개 |

## 2개 이상 품목의 합산

품목별 면적을 명확히 구획하고 다음 합계가 1 이상이면 면적기준을 충족한 것으로 본다.

`(품목1 실제면적 ÷ 품목1 기준면적) + (품목2 실제면적 ÷ 품목2 기준면적) + … ≥ 1`

- 원칙: 하나의 수평투영면적에는 1개 품목만 등록
- 예외: 같은 수평투영면적에서 품목을 동시 생산하는 복합산림경영은 신청자 희망에 따라 품목별 면적을 구획하여 등록 가능

## 공통 추가요건

- 지적·계곡·하천·도로·말뚝 등으로 경계가 명확할 것
- 경영주외 농업인은 연 90일 이상 임업 종사자일 것
- 농업법인은 적법한 설립·등기, 정당한 생산수단 권원, 법인 명의의 임업경영 사업·활동실적을 갖출 것
- 변경등록 시에도 같은 등록기준을 계속 충족할 것

## 판단 예시

| 상황 | 판단 |
|---|---|
| 밤 700㎡ + 산나물 90㎡, 경계 명확 | `700/1,000 + 90/300 = 1.0` → 면적기준 충족 |
| 버섯 250㎡ + 약초 100㎡ | `250/300 + 100/1,000 ≈ 0.933` → 면적기준 미달 |
| 30,000㎡ 육림업 임·소반이지만 사업실적 없음 | 면적만으로 불가. 산림경영계획인가서와 사업실적 증빙 필요 |
