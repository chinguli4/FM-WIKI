# 벌채 지침

> 근거: 산림자원의 조성 및 관리에 관한 법률 제36조, 산림재난방지법 제48조·제49조, 산림작업 설계·감리 요령

## 1. 벌채의 종류

| 구분 | 내용 |
|------|------|
| **주벌** | 산림의 주요 수확을 목적으로 하는 벌채 |
| **간벌** | 임분 밀도 조절을 위한 중간 수확 벌채 |
| **갱신벌** | 후계림 조성을 전제로 한 벌채 (개벌·모수벌·산벌·택벌) |
| **위생벌** | 병해충 피해목·고사목 제거를 위한 벌채 |
| **수종갱신벌** | 임업적 가치가 낮은 수종을 우량 수종으로 교체하기 위한 벌채 |

## 2. 벌채 허가 기준

### 2.1 허가 대상
- 입목벌채: 산림청장·시·도지사·시장·군수·구청장 허가
- 허가 면적 기준: 1ha 미만 시·군·구청장, 1ha 이상 시·도지사

### 2.2 허가 제한 지역
- 산림보호구역
- 경사도 35° 이상 지역 (단, 수종갱신 목적 예외 가능)
- 1급 산사태 위험지역
- 백두대간 보호지역 핵심구역

### 2.3 존치 기준
- 완충지역: 계류변 양쪽 10m 이상 존치
- 능선부: 능선으로부터 20m 이내 존치 권장
- 임도변: 임도 양쪽 10m 이내 보존대 유지

## 3. 벌채 방법별 적용 기준

### 3.1 개벌 (皆伐)
- 한 작업급 전체를 일시에 벌채
- 적용 대상: 동령림, 인공림 위주
- 최대 개벌 면적: 50ha 이하 (산림청 지침)
- 후계림 조성 의무 (벌채 후 2년 이내 식재)

### 3.2 택벌 (擇伐)
- 성숙목·불량목을 선택적으로 벌채
- 적용 대상: 이령혼효림, 천연림
- 잔존 임분의 ha당 흉고단면적: 15㎡ 이상 유지
- 벌채 강도: 임분 재적의 30% 이내 권장

### 3.3 산벌 (傘伐)
- 예비벌 → 하종벌 → 후벌 3단계 갱신
- 갱신 기간: 수종에 따라 5~20년
- 천연갱신을 기본으로 함

### 3.4 모수벌 (母樹伐)
- ha당 30~50본의 모수 잔존
- 종자 산포 후 모수 제거

## 4. 벌채목 반출 기준

- 벌채 후 1년 이내 반출 완료 원칙
- 잔재물(가지·잎 등) 처리: 임내 분산 또는 집적 후 처리
- 표토 훼손 최소화: 집재로 지정, 임의 집재 금지
- 집재 방법: 가선집재·가반식집재·축력집재 우선 (임도 미설치 급경사지)

## 5. 산불피해지 위험목 제거·긴급벌채

산불피해지 벌채는 일반 벌채와 달리 산불피해 조사, 2차 피해영향권, 생활권 보호대상, 산사태·토사유출 우려를 함께 판단한다.

### 5.1 산불피해지역 구분

| 구분 | 기준 |
|------|------|
| 산불피해지역 | 실제 산불이 발생하여 피해가 발생한 전체 지역 |
| 위험목 제거사업 대상지역 | 피해목이 쓰러지거나 부러져 인명·가옥 등 시설물 피해를 줄 수 있는 지역 |
| 일반벌채지역 | 산불피해지역 중 위험목 제거사업 대상지역을 제외한 피해목 벌채지역 |
| 2차피해영향권 | 산림경계에서 보호대상 방향 50m 이내의 주택·도로·주요 관광지 등 인명·재산 피해 우려지역 |

### 5.2 위험목 제거 대상

| 기준 | 적용 |
|------|------|
| 공통거리 | 생활권에서 산림방향 수평투영거리 60m 또는 주계곡부 수평투영거리 30m 이내 |
| 포함 가능 | 계곡방향 최대 60m 내 피해 확인 지역, 보호대상에 2차 피해가 우려되는 유역 |
| 제외 | 피해경 지역, 생활권·도로·관광시설 등 보호대상 없음, 임연부 입목 미피해, 능선 너머 2차 피해 우려 없음 |
| 복구단가 | 위험목제거 31.7백만원/ha |

### 5.3 위험목 외 일반벌채지역

| 기준 | 적용 |
|------|------|
| 피해등급 | 피해심(High), 피해중(Moderate) 지역 |
| 범위 | 위성영상 산불피해지역 중 위험목 제거사업 대상지 제외 지역 |
| 제외 | 법정보호구역, 지목이 임야가 아닌 필지, 2차 피해 우려가 없는 산사태위험지역 |

> 산불피해지 피해목 벌목은 [[행정규칙/고시/산림사업_표준품셈/4장_나무베기]]에서 20% 할증이 명시되어 있다. 위험목 베기는 흉고직경 16cm 이상에 적용하며, 16cm 미만은 솎아베기 품을 적용한다.

## 6. 벌채 후 조치

1. 조림 계획 수립 (개벌 후 의무)
2. 임지 정리 (잔재물 처리)
3. 토사 유출 방지시설 설치 (경사지)
4. 작업 완료 보고: 벌채 완료 후 30일 이내

## 7. 관련 서식

- 입목벌채 허가 신청서
- 작업설계서 (→ [작업설계기준](./작업설계기준.md) 참조)
- 벌채 완료 보고서
