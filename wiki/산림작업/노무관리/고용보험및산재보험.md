# 산림사업시행업 — 고용보험·산재보험 적용 기준

> 근거: 고용보험법 시행령 제3조, 산업재해보상보험법 제6조·제124조  
> 원문 확인: 고용보험법 시행령 (대통령령 제36472호, 시행 2026-07-01) legalize-kr/kr/고용보험법/시행령.md

---

## 핵심 기준 요약

| 사업 유형 | 고용보험 | 산재보험 |
|-----------|----------|----------|
| 임업·법인 아닌 자·**상시 4명 이하** | **적용 제외** | 당연 적용 |
| 임업·법인 아닌 자·**상시 5명 이상** | **당연 적용** | 당연 적용 |
| 산림토목(건설업 분류) | 당연 적용 (인원 무관) | 당연 적용 |
| 법인 사업체 (인원 무관) | 당연 적용 | 당연 적용 |

---

## 1. 고용보험 적용제외 근거 (원문 확인)

### 고용보험법 시행령 제3조 (적용 제외 근로자)

> "**농업·임업 및 어업** 중 **법인이 아닌 자**가 **상시 4명 이하**의 근로자를 사용하는 사업에 종사하는 근로자"

**적용제외 3요건 (모두 충족해야 함):**

| 요건 | 내용 |
|------|------|
| 업종 | 농업·**임업**·어업 |
| 사업형태 | **법인이 아닌** 개인사업체 |
| 규모 | 상시 근로자 **4명 이하** |

> ⚠️ **주의**: "가족·친족이라서 적용제외"되는 조항은 현행 고용보험법·시행령에 **없음**  
> 적용제외의 근거는 업종·규모이며, 가족 여부는 별개

---

## 2. 산림사업 유형별 판단

### 숲가꾸기 시행업

- 업종 분류: **임업** (KSIC A02)
- 개인사업체 + 상시 4명 이하: 고용보험 적용 제외
- 개인사업체 + 상시 5명 이상: 고용보험 **당연 적용**

### 산림토목 시행업

- 업종 분류: **건설업** (KSIC F) — 임도·사방댐·산지복구 등 공사 행위
- 인원 수 무관하게 고용보험 **당연 적용** (임업 적용제외 조항 자체 해당 안 됨)

### 복합 사업체 (숲가꾸기 + 토목 병행)

- 전체 상시근로자 수 합산 기준 판단
- 숲가꾸기 7명 + 토목 5명 = 12명 → 4명 초과, 당연 적용

---

## 3. 산재보험 적용 기준 (원문 확인)

### 산업재해보상보험법 제6조 (적용 범위)

> "이 법은 근로자를 사용하는 **모든 사업** 또는 사업장에 적용한다.  
> 다만, 위험률·규모 및 장소 등을 고려하여 대통령령으로 정하는 사업에 대하여는 이 법을 적용하지 아니한다."

→ 산림사업시행업: 업종·인원·법인 여부 **무관하게 산재보험 당연 적용**

### 산업재해보상보험법 제124조 (가족 종사자 특례)

> "사업주의 **배우자** 또는 **4촌 이내 친족**으로서 해당 사업에 노무를 제공하는 사람은  
> **공단의 승인**을 받아 보험에 가입할 수 있다."  
> (가입 시 근로자로 본다)

→ 가족은 산재보험 **임의 가입** 가능 (자동 적용 아님, 공단 승인 필요)

---

## 4. 가족 근로자 고용보험 관련 주의

"가족이라서 고용보험 면제"라는 주장의 법령 근거는 없으며, 실제 면제 여부는 아래로만 판단:

```
① 업종이 임업인가?
② 법인이 아닌 개인사업체인가?
③ 상시 근로자가 4명 이하인가?
→ 3가지 모두 해당 시에만 적용제외 (가족 여부와 무관)
```

---

## 5. 미가입 시 제재

| 항목 | 내용 |
|------|------|
| 고용보험 미가입 | 보험료 소급 징수 + 과태료 |
| 산재보험 미가입 | 보험료 소급 + 재해 발생 시 사업주 부담 |
| 확인 방법 | 근로복지공단 피보험자격 조회 |
