## B. 설계서류

### B-1. 원가계산서

| | 항목 | 판정 기준 | 결과 |
|--|------|---------|------|
| [ ] | 핵심구역·일반구역 원가계산 **분리** 여부 | 일반구역만 있으면 사업 무효 → **지적** | |
| [ ] | 선목비 위치 | 시공 원가계산서 내 계상; 재경비·기술료 합산 → **지적** | |
| [ ] | 4~8 cm 중층 소나무류 별도 품목 여부 | 솎아베기(10 cm↑)와 구분; 중복 → **지적** | |
| [ ] | 단목 설계 시 조재품 중복 여부 | 단목 + 조재품 동시 계상 → **지적** | |
| [ ] | 집적 수량 = 이용목만 | 지조 합산 시 → **지적** | |
