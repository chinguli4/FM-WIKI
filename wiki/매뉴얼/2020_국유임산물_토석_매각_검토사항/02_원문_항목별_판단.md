# Ⅱ. 원문 항목별 판단

> 원문 범위: 13~61행

## 항목별 재구성

| 원문 항목 | 원문 사례의 핵심 | 일반화할 때 확인할 것 |
|---|---|---|
| 제한지역 | 부수 토석 예외 검토 | 보전국유림·공공시설·가시지역 등 제한사유와 §25의4 예외 |
| 기준 전부 제외 | 해당되지 않는 것으로 표시 | 재해복구·터널 부수토석·공익사업 등 법정 사유 특정 |
| 별표 8 허가기준 | 부수 토석은 일부 예외 표시 | 예외 조문과 남는 재해·복구 조건 분리 |
| 채석경제성 | 부수 석재 예외 표시 | 석재 용도·기존 허가·부수 발생·반출 여부 확인 |
| 재해·동의 | 영향거리·방지시설·관계자 동의 | 도시지역 예외는 지방산지관리위원회 심의 필요 여부 |
| 장비·인력 | 직접 가공하지 않는 부수 토석 예외 | 토석 용도·직접 가공 여부·현행 별표 8의2 |
| 외부 반입 | 60km·50% 등 조건 | 반입 출처·용도·거리·수량·복구설계 일치 |

## 원문에서 유지할 통제점

- 절·성토면 배수와 사면붕괴 방지
- 낙석·비사·분진·소음·진동 대책
- 표토와 폐석 처리계획
- 관계 주민·시설·분묘 영향 확인
- 채취구역 밖 토석의 반입 통제
- 장비·자격·업종등록의 용도별 확인

## 원문 밖에서 추가할 현행 통제점

- §25와 §35의 국유림/비국유림 경로 구분
- 국유임산물 일반매각기준에서 토석이 제외된다는 점
- 국유림관리소장·지방산림청장의 10만㎡ 권한 구분
- 측량 실측도·구적도와 누적 반출량 관리
- 감정평가 인원과 부수 토석 5만㎥·1천㎥ 가격 특례
- 복구비 예치, 복구설계, 감리, 준공검사, 하자보증
- 계약 외 채취·대금 미납·복구비 미예치 시 계약해제
