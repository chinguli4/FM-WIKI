## 1. 조림복원 개요

| 항목 | 기준 |
|------|------|
| 목적 | 산불로 훼손된 산림을 생태·경관·경제·환경적 가치가 높은 건강한 숲으로 회복 |
| 대상지 | 산불피해지 중 조림복원이 필요한 산림 |
| 추진절차 | 조사 → 응급복구 → 조림복원 → 모니터링 및 사후관리 |
| 기본 방향 | 산림의 6대 기능과 연계하여 피해정도·입지여건별로 복원 |

### 복원·복구 유형과 소관

| 구분 | 개념 | 소관부서 |
|------|------|----------|
| 응급복구 | 산불 이후 2차 피해 예방을 위한 사방 복구 | 산사태방지과 |
| 조림복원 | 산주의견과 임지 여건을 고려해 경제적·환경적으로 산림 기능을 회복 | 산림자원과 |
| 생태복원 | 자연회복력 기반으로 자생식물·자연재료 등을 활용해 생태계 기능 회복 | 산림생태복원과 |
| 자연복원 | 피해가 경미하여 자연적 회복을 유도 | 별도 소관 명시 없음 |

## 2. 표준 의사결정 기준

산불피해지 복원 표준 의사결정 흐름도는 초기 정책 결정을 지원하기 위한 기준이며, 복원·복구계획 수립 시 현장여건, 전문가 의견, 산주·주민 의견을 반영하여 조정할 수 있다.

| 고려요소 | 적용 기준 |
|----------|----------|
| 의견수렴 | 산주 또는 지역주민 의견을 설명회·공청회 등으로 계획 수립 전·중·필요시 수시 반영 |
| 보호구역 | 산림유전자원보호구역, 백두대간보호지역, 자연공원, 야생생물보호구역, 생태경관보전지역, 특정도서 등은 생태복원 우선 검토 |
| 피해등급 | 국립산림과학원 산불피해등급도상 심(수관화), 중(열해), 경(지표화) 적용 |
| 식생피복도 | 드론으로 필지별 조사. 봄철 산불은 7~8월, 그 이후 산불은 발생 후 3개월 이상 경과 시 조사 |
| 평균경사도 | 필지 단위 평균 경사도 적용 |
| 임지생산력 | 피해 이전 임지생산능력급수 적용. 필요시 판정표 활용 |
| 목재생산림 | 기능구분도상 목재생산림 |
| 2차피해우려지 | 산사태취약지역, 민가·도로·숲길·임도·보호시설 주변 수고 높이 이상 범위 |
| 움싹갱신 | [[행정규칙/훈령/지속가능한_산림자원_관리지침_일반지침]] 기준 적용 |
| 기대회복력 | 임지생산력, 모암, 지형, 경사, 토양수분·양분 등 토양질 요소를 종합 고려 |

### 복원 유형 판단

| 유형 | 적용 방향 |
|------|----------|
| 조림복원 | 산림용 묘목 또는 종자파종으로 복원 |
| 생태복원 I | 도복목·토사유출 등 2차 피해가 우려되는 지역에서 위험목 제거, 토사유출 방지 등 재해예방 중심 |
| 생태복원 II | 자연회복력과 기대회복력이 낮아 자연 식생 유입이 어려운 지역에서 피해목 제거, 토양안정, 식재 등 적극 복원 |
| 생태복원 III | 자연회복력은 낮지만 기대회복력이 높아 최소 관리로 식생 유입이 가능한 지역에서 피해목 존치, 파종, 천연하종 등 소극 복원 |
| 자연복원 | 자연력 복원을 원칙으로 하되 필요시 움싹본수 조절, 생태보완조림 가능 |

## 3. 산림기능별 조림복원 방향

| 기능 | 조성 방법 | 피해목 처리 |
|------|----------|------------|
| 목재생산림 | 전략수종 규모화 조림, 입지조건별 주수종·부수종 배치, 용재수종·지역별 집중 조림수종 선정 | 대상 또는 군상벌채(5ha 미만) |
| 수원함양림 | 토양 안정이 가능하면 자연복원과 보완조림 병행, 참나무류·소나무 등 심근성 수종과 천근성 수종 혼합 | 산물은 최대한 수집·활용하거나 재해 안전구역으로 이동 |
| 산지재해방지림 | 사방지는 오리나무·아까시나무·싸리나무 등 질소고정 수종과 속성수 혼합, 토심 깊은 곳 활엽수·얕은 곳 침엽수 중심, ha당 5,000본 기준 | 산물은 최대한 수집·활용하거나 재해 안전구역으로 이동 |
| 자연환경보전림 | 자연복원 원칙, 맹아·천연치수 갱신, 토양 안정 후 필요시 맹아목 본수 조절 | 피해목 존치 |
| 산림휴양림 | 경관수종·화목류·관목류·식이수종·지역특색수종 선정, 식재조림·천연갱신으로 다층혼효림 조성 | 이용공간 산물은 수집·활용 또는 안전구역 이동, 복원방법에 따라 제거·존치 |
| 생활환경보전림 | 공원형·경관형 다층혼효림, 방풍·방음형 최소 30m 폭 계단식 다층림, 미세먼지 저감형 다층혼효림 | 이용공간 산물은 수집·활용 또는 안전구역 이동, 복원방법에 따라 제거·존치 |

### 목재생산림 기후대별 주요 조림수종

| 기후대 | 주수종 | 부수종 |
|--------|--------|--------|
| 온대 북부 | 잣나무, 낙엽송, 소나무, 자작나무, 참나무류, 백합나무 | 전나무, 스트로브잣나무, 물푸레나무, 거제수나무 |
| 온대 중부 | 잣나무, 낙엽송, 소나무, 자작나무, 참나무류, 백합나무 | 리기테다소나무, 가래나무, 스트로브잣나무, 물푸레나무, 거제수나무, 느티나무 |
| 온대 남부 | 낙엽송, 소나무, 자작나무, 참나무류, 백합나무 | 해송, 리기테다소나무, 물푸레나무, 박달나무, 가래나무 |
| 난대 | 해송, 편백, 백합나무 | 테다소나무, 삼나무, 가시나무류, 후박나무 |

## 4. 조림 방법별 복원

### 식재조림

| 항목 | 기준 |
|------|------|
| 대상지 | 경제수종 대량생산 산림, 천연갱신 곤란 벌채지·미립목지, 토사유출·풍해 예방 특수수종 식재지 |
| 정방형 식재 | 5,000본/ha는 1.4m 간격, 3,000본/ha는 1.8m 간격 |
| 3본 군상식재 | 조림목간 0.6m, 식재군간 3.3m×3.0m |
| 5본 군상식재 | 조림목간 1.2m, 식재군간 4.1m |
| 2열 부분밀식 | 조림목간 1m, 식재군간 6.6m |

**식재시기**

| 지역 | 춘기 | 추기 |
|------|------|------|
| 온대북부(경기·강원) | 3월 하순~4월 하순 | 9월 하순~10월 중순 |
| 온대중부(전북·경북·충청) | 3월 중순~4월 중순 | 10월 상순~10월 하순 |
| 온대남부(전남·경남) | 3월 상순~4월 상순 | 10월 중순~11월 상순 |
| 난대(제주·남부해안) | 2월 하순~3월 하순 | 10월 하순~11월 중순 |

> 가을철 조림은 최소화하고, 적설량이 적거나 바람이 심한 지역에서는 가을철 식재를 지양한다.

### 파종조림

| 항목 | 기준 |
|------|------|
| 대상지 | 발아가 잘되는 수종, 식재조림 활착률이 낮은 급경사지 등 특수지역 |
| 수종 | 소나무·해송 등 침엽수, 가래나무·밤나무·상수리나무·굴참나무·졸참나무·갈참나무·신갈나무 등 |
| 시기 | 봄: 중부 4월 상순, 남부 3월 하순 / 가을: 10~11월 |
| 방법 | 지름 50~60cm 지피물 제거, 중앙 30~40cm 경운, 10cm 높이 상 조성, 2~10립 파종, 종자 지름 2~3배 복토 |

### 용기묘 조림

| 항목 | 기준 |
|------|------|
| 대상지 | 세근 발달이 좋지 않은 직근성 수종, 경사지 등 일반 조림 곤란 지역 |
| 시기 | 연중 가능하나 봄철 3~4월, 가을철 10~11월 권장 |
| 운반 | 운반 2~3일 전 물주기 중지, 바람 차단 후 현지까지 운반, 그늘 보관 |
| 식재본수 | 상수리나무·소나무 등은 3,000본/ha 기준, 현지여건에 따라 조정 |

### 맹아갱신

| 항목 | 기준 |
|------|------|
| 대상지 | 피해정도 경~중인 목재생산림 |
| 최소 요건 | 피해목 벌채 후 최소 직경 7cm 이상의 참나무류 그루터기 900개/ha 이상 균일 분포 |
| 그루터기 정리 | 지상 10cm 이내로 낮게 절단, 물 고임 방지 |
| 풀베기 | 토사유실로 인한 임지척박화를 고려해 원칙적으로 실시하지 않음. 보완조림 시 조림목 주변 풀베기 가능 |
| 움싹 조절 | 자연경합 후 우열이 분명해지는 2~3년 후, 11월~다음해 5월 실시 |
| 남길 움싹 | 상수리·굴참나무 1~2개, 신갈·갈참나무 2~3개 |
| 보완조림 | 3년 이내 움싹이 3,000그루/ha 미만이면 실시 |
| 수확 | 15~20년 후 직경생장량을 고려해 수확 |

## 5. 내화수림대 조성 및 관리

### 대상지와 조성 방향

| 구분 | 기준 |
|------|------|
| 목적 구분 | 비화예방 내화수림, 시설물보호 내화수림 |
| 형태 구분 | 띠 모양 조성 또는 기존 임분 개선 |
| 대상지 | 산불발생통계, 산불취약지역 등으로 산불 재발 우려가 있는 곳 |
| 산물처리 | 산불 확산 가능 지조물은 최대한 수집하여 임외 반출 |
| 산불예방 숲가꾸기 사업지 | 핵심구역에서만 사업 추진(수하식재) |

### 내화수종 선정

| 기후대 | 층위 | 내화성 수종 |
|--------|------|-------------|
| 온대 | 교목성 | 은행나무, 굴참나무, 상수리나무, 떡갈나무, 느티나무, 물푸레나무, 황철나무, 황벽나무, 백합나무, 아까시나무 |
| 온대 | 아교목성 | 소태나무, 쇠물푸레, 마가목 |
| 온대 | 관목성 | 누리장나무, 닥나무, 사철나무, 탱자나무 |
| 난대 | 교목성 | 가시나무류, 녹나무, 먼나무, 생달나무, 후박나무, 참식나무, 육박나무, 소귀나무, 조록나무 |
| 난대 | 아교목성 | 아왜나무, 굴거리나무, 동백나무류, 붓순나무, 비쭈기나무, 후피향나무, 가마귀쪽나무 |
| 난대 | 관목성 | 사스레피나무, 식나무, 팔손이, 꽝꽝나무, 협죽도 |

### 내화수림대 세부 기준

| 구분 | 기준 |
|------|------|
| 시설물 보호 이격공간 | 건축물로부터 20~25m 내외 벌채, 초본·관목류 조림 또는 녹지대 조성 가능 |
| 완충지대 | 이격공간 밖 폭 30m 내외 기존 숲 강도 간벌 |
| 침엽수 밀도 | 침엽수 기존 숲 유지 시 최대 100본/ha 이하 |
| 비화예방 내화수림 | 서사면 6부~9부 능선 띠형태 활엽수 식재, 동사면 소나무 존치 |
| 짧은 사면 | 내화수림 폭을 총 사면길이의 35~40% 내외로 조정 |
| 임도 주변 | 임도 중심 위아래 사면에 20m 폭 이상 조성 |
| 시설물보호 내화수림 | 시설물 중심 서쪽방향 산림에 조성, 폭 50m 이상, 시설물과 최대 10m 이상 이격 |

### 사후관리

| 대상 | 기준 |
|------|------|
| 활엽수 맹아 | 맹아갱신에 준하여 관리, 발생상태에 따라 3년 내외 관리 |
| 침엽수 | 수관이 서로 닿지 않도록 최대 100본/ha 이하 관리 |
| 보완조림 | 맹아 생육 부진 또는 다량 고사 시 부분 인공조림 가능 |
| 관목·초본류 | 산불 확산 및 수관화 위험요인 제거를 위해 2~5년 관리 |

## 6. 기타 조림복원 고려사항

| 구분 | 대상지 | 조성 방법 | 피해목 처리 |
|------|--------|----------|------------|
| 송이복원 조림 | 산불피해 전 송이생산 지역과 인근 지역 | 산불피해지 인근 종자로 키운 노지묘·용기묘 소나무 조림 | 군상벌채, 수하식재 등 복원방법 고려 |
| 유실수 조림 | 사면경사 20도 이하, 임지생산성 양호, 산주 희망 유실수 적지 | 밤나무·매실나무 등 유실수 식재 | 모두베기 |
| 경관조림 | 국도·시가지 주변, 문화경관자원·관광지 주변 | 산벚나무·복자기·마가목 등 1~2m 대묘 400~500본/ha | 모두베기 |
| 묘목보호구 | 정상 활착이 어렵다고 판단되는 곳 | 종자·묘목 보호, 저밀도 조림, 야생동물 피해 방지 | 해당 없음 |

| 구분 | 소요인력 | 인력구분 |
|------|----------|----------|
| 묘목보호구 100cm 이상 | 0.013 인/개 | 보통인부 |
| 묘목보호구 100cm 미만 | 0.011 인/개 | 보통인부 |

## 7. 목적사업별 추진방향

| 사업 구분 | 주요 내용 |
|----------|----------|
| 경제림 조성 | 경제림육성단지 중심 산업용재 공급 및 단기소득 조림. 3,000본/ha 기준 |
| 큰나무 조림 | 큰나무공익조림 350본/ha, 산림재해방지조림 1,500본/ha |
| 지역특화 조림 | 섬지역산림가꾸기, 지역특화림 조성. 지역 특성에 맞는 본수 적용 |
| 밀원수림 조성 | 안정적 채밀 공급을 위한 밀원수림. 3,000본/ha 기준 |
| 산림재해피해지 복구조림 | 대형산불 피해지 복구, 1,500본/ha, 15백만원/ha 기준 |
| 내화수림대 조성 | 대형산불 확산 방지, 1,500본/ha, 15백만원/ha 기준 |

### 재해피해지 조림복원 특이 기준

- 입지여건, 묘목수급, 산주 요구를 종합해 조림수종 선정
- 산불피해정도(심·중·경)를 고려해 자체 복원계획에 따라 조림복원과 자연복원을 병행
- 식재수종은 대묘 1,500본/ha 이상 큰나무를 원칙으로 하되, 현지여건과 묘목수급에 따라 소묘·관목류 병행 가능
- 소나무재선충병 반출금지구역은 「소나무재선충병 방제특별법」 제12조에 따라 소나무류 조림 금지
