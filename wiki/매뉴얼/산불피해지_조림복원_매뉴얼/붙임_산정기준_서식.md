## 4. 피해액 산정 참고단가

### 6cm 이상 입목 기준가격

| 수종 | 기준가격 |
|------|----------|
| 소나무 | 48,300원/㎥ |
| 잣나무 | 24,900원/㎥ |
| 낙엽송 | 28,700원/㎥ |
| 기타침엽수 | 10,900원/㎥ |
| 기타활엽수 | 6,340원/㎥ |

### 6cm 미만 입목 조림·육림비

| 항목 | 단가 또는 적용금액 |
|------|-------------------|
| 조림비 | 11,205,000원/ha |
| 풀베기 | 2,244,301원/ha × 7회 = 15,710,000원 |
| 덩굴제거 | 2,339,003원/ha × 4회 = 9,356,000원 |
| 어린나무가꾸기 | 2,385,246원/ha × 1회 = 2,385,000원 |
| 육림비 합계 | 27,415,000원/ha |

> 육림비는 피해지별 실제 시행된 육림사업만큼의 단비만 적용한다.

> 2026년도 조림비용 고시는 11,625,000원/ha이다. 매뉴얼의 2025 조림비용 11,205,000원/ha와의 차이는 [[행정규칙/고시/조림비용_2025_2026_비교]]에서 비교한다.

## 5. 복구단가

| 구분 | 기준단가 | 주요 산출 근거 |
|------|----------|---------------|
| 산불피해지 복구조림 | 15,000,000원/ha | 묘목 1,500본, 비료 180kg, 정리·식재·소운반·비료주기, 설계·감리비 포함 예산편성 기준 |
| 위험목제거사업(긴급벌채) | 31.7백만원/ha | 피해목 벌채·조재 19,180천원 + 수집 12,546천원 |
| 산불피해지 생태복원 | 48,390,000원/ha | 자생식물 채집·식재, 교목·관목류 굴취·식재, 새심기, 씨뿌리기 등 |

### 위험목제거 산출 기준

| 공정 | 산출 |
|------|------|
| 피해목 벌채 및 조재 | 27.4천원 × 700본/ha = 19,180천원 |
| 수집 | 102천원 × 123㎥/ha = 12,546천원 |

> 위험목제거 단가는 대규모 산불피해지 중 생활권 주변에 한하며, 현지 여건에 따라 수집·파쇄 비용 추가 적용 가능.

## 7. 서식 1: 산림피해지 조사(입목피해)

### 주요 입력 항목

| 구분 | 내용 |
|------|------|
| 기본정보 | 시군, 읍면, 리, 지번, 지적, 지목, 소유자 주소·성명 |
| 피해면적 | 지적면적, 전체 피해면적, 6cm 이상·미만 입목 피해면적 |
| 6cm 이상 입목 | 수종, ha당 본수·재적, 피해정도별 적용비율, 피해조사 결과 재적, 피해액 |
| 6cm 미만 입목 | 수종, 피해유형, 피해정도, 조림비, 육림비, 피해액 |

### 작성요령

- 지적은 토지대장상 면적으로 기재
- 지목은 피해지 내 임야인 필지만 입력
- 전체 피해면적은 도상 수평투영면적 기준이나 지적면적을 초과하지 않도록 입력
- 수종은 소나무, 낙엽송, 잣나무, 기타침엽수, 기타활엽수로 입력
- 필지별 면적이 4ha 이상이고 2개 이상 수종이 분포하면 수종별 면적 0.5ha 이상 집단화 시 구분
- 필지 면적 4ha 미만은 혼효율 낮은 수종이 25% 이상이면 구분, 25% 미만이면 우점 수종으로 일괄 작성
- 피해정도는 극심, 중, 경, 소생 중 하나만 선택
- 소나무·잣나무는 지표화가 있어도 지면 0.5m 이상 수간이 탄 경우 극심으로 작성하고 소생 입력 금지
- 낙엽송은 수관 전소 시 극심, 지표에서 수간 1m 이하 탄 경우 소생, 1m 이상 탄 경우 중 또는 경 입력
- 활엽수는 수관 전소라도 극심 입력 금지. 수관 전소는 중, 일부 피해는 경, 수간만 피해는 소생 입력

## 8. 서식 2: 산림 복구계획

### 주요 입력 항목

| 구분 | 내용 |
|------|------|
| 기본정보 | 시군, 읍면, 리, 지번, 지적, 지목, 소유자 |
| 면적 | 전체 피해면적, 소생 및 제외면적, 인공복구 대상면적 |
| 긴급벌채지 | 면적, 비용, 비고 |
| 조림복원지 | 면적, 조림비용, 응급·항구 구분 |
| 산림생태복원 | 면적, 조림비용, 응급·항구 구분 |

### 작성 공식

| 구분 | 공식 |
|------|------|
| 긴급벌채지 | 31,700천원/ha × 피해면적 |
| 조림복원지 | 15,000천원/ha × 피해면적 |
| 산림생태복원지 | 48,390천원/ha × 피해면적 |
