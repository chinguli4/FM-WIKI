## 2. 2022년 강릉 산불 조림복원 계획 사례

### 큰나무 조림

| 항목 | 계획 기준 |
|------|----------|
| 조림면적 | 가옥, 시설지, 도로 등 현장조사 결과를 반영하여 결정 |
| 식재유형 | 식재유형별로 조림면적 구분 산출 |
| 수종 선정 | 대상지구분과 식재유형별 추천수종 중 강릉시 실정에 맞게 선택 |
| 경관·내화 요구지 | 가옥·도로 주변은 화목류 식재 |
| 토질 양호지 | 유실수를 입지·토양조건에 따라 선정 |
| 묘목 규격 | 산불피해지 큰나무 조림단비표에 따른 수고 2.5m 내외 분뜨기묘 |
| 조림시기 | 2023년 봄철 완료 계획 |
| 사업비 | 2022년 산불피해지 큰나무조림 추진계획의 단비표 일괄 적용 |

### 항구복원 조림

| 항목 | 계획 기준 |
|------|----------|
| 면적 산출 | 현장조사 결과를 바탕으로 자연복원지역과 제외지를 빼고 조림 가능 지역만 선발 |
| 수종 선정 기준 | 시설물 유무, 맞춤형 조림지도 추천수종, 접근성, 현장 확인 |
| 내화수종 우선순위 | 내화수림대 → 대규모 시설 인접지 → 소규모 시설 인접지 |
| 소나무류 우선순위 | 환경보전림 → 능선부 → 척박지(저수고) → 소나무 단순림 |
| 참나무류 우선순위 | 토양조건 양호지역 → 계곡부 → 임연부 |
| 유실수 우선순위 | 산주 요청지역 → 토양조건 양호한 평탄지 → 계곡부 |
| 조림시기 | 2023~2025년 3년간 실시 |
| 사업비 | 산불피해지 조림복원 단가표 ha당 15백만원 적용 |

### 묘목수급 및 사후관리

| 항목 | 기준 |
|------|------|
| 큰나무 묘목 | 강릉시 보유 현황과 나무시장 공급 활용 |
| 묘목구입 우선순위 | 동해안 지역 → 강원 도내 → 경기도 → 경상북도 → 충청남북도 → 전라북도 |
| 항구복구 묘목 | 강원도 내 지정양묘 우선, 없으면 타 지역 구입 |
| 조림예정지 덩굴걷기 | 2023~2025년, 남은 조림대상 면적의 10% 일괄 적용 |
| 조림지 사후관리 | 식재 후 3년간 풀베기 6회, 덩굴제거 2회 |
| 사후관리 기간 | 2023~2027년 |

## 3. 2022년 울진 산불 조림복원 계획 사례

### 복원구역 결정

| 구분 | 면적 | 비율 |
|------|------|------|
| 조림복원 | 6,900ha | 73.1% |
| 자연복원 | 2,540ha | 26.9% |

울진 사례는 의사결정 흐름도상 목재생산림만 인공복원구역으로 결정한 결과를 그대로 따르지 않고, 대면적 피해 특성과 주민·산주·전문가 의견을 반영하여 수원함양림, 산사태방지림, 자연환경보전림 일부를 조림복원구역에 추가 편입했다.

### 복원대상 구분

| 유형 | 기준 |
|------|------|
| 인공복원 | 피해 정도가 중 이상이며 조림·갱신으로 생태계 서비스와 경제적 가치를 높일 수 있는 곳 |
| 조림복원 | 원칙적으로 목재생산림 대상. 필요시 자연환경보전림을 제외한 5대 기능림도 흐름도에 따라 조림복원 |
| 송이산 복원 | 기존 송이 발생지를 대상으로 소나무 조림 또는 자연력 기반 소나무림 복원 |
| 단기소득 | 주민 의견을 수렴하되 토양조건, 사후관리, 산지전용 여부를 고려 |
| 내화수림 | 수관화·비산화를 줄일 수 있는 위치에 내화성수종 조림 또는 자연복원력 활용 |
| 자연복원 | 수관층 유지, 낮은 임지생산력, 급경사지 등 조림복원 외 지역 |

### 기능별 복원 방향

| 기능 | 계획 방향 |
|------|----------|
| 목재생산림 | 경제수, 밀원수, 복층림, 활엽수림, 특용수림으로 계획 |
| 생활환경보전림 | 경관수·보호림, 특용수 생산단지, 밀원수, 해안경관림, 내화수림 등으로 계획 |
| 산림재해방지림·수원함양림 | 활엽수 복층림, 해안경관림, 수원함양림 등으로 계획 |

> 울진 사례의 기능구분과 조림수종은 현장 또는 주민 의견에 따라 변동 가능성이 있음을 전제로 한다.

## 4. 2023년 홍성 산불 조림복원 계획 사례

### 계획 반영요소

| 요소 | 내용 |
|------|------|
| 입목벌채허가구역 | 산주의 조림신청수종 반영. 소나무·편백 신청이 많고 유실수·경관림 의견도 반영 |
| 산불 전 임상도 | 소나무·리기다소나무·곰솔 등 소나무과가 65.4%로 가장 많고, 상수리나무·참나무류·아까시나무 등 활엽수도 분포 |
| 맞춤형 조림지도 | 소나무, 곰솔, 굴참나무, 편백 등 적합 수종 확인 |
| 내화수림대 | 수관화·비산화를 줄일 수 있는 위치에 내화성 수종 조림 또는 자연복원력으로 활엽수림 유도 |
| 중요시설 주변 | 민가·중요시설 주변 식재 가능 지역은 내화수종 식재로 내화수림대 조성 |
| 능선부 | 산정에서 시작되는 능선부를 주된 내화수림대로 구획 |
| 방풍림 | 해안선 가까이 연속 설치, 끊어진 부분 없도록 계획. 주요 수종은 곰솔 |
| 경관림 | 해안가 주변 단일수종 활엽수림 계획 |

## 5. 사례 활용 시 유의점

- 강릉·울진·홍성 사례는 표준단가와 의사결정 기준을 현장 여건, 산주 의견, 묘목수급, 지역산림 특성에 맞춰 조정한 예시다.
- 사례 수종·면적을 그대로 적용하기보다 [[3장_조림복원_실행]]의 기능별 기준과 [[2장_복원복구계획_피해조사]]의 제외지 기준을 먼저 적용한다.
- 묘목수급, 사후관리 횟수, 단가 적용은 사업연도 고시와 예산편성 기준을 반드시 확인한다.
