## 1. 산불피해 복구지 모니터링

| 항목 | 기준 |
|------|------|
| 목적 | 복구사업 종결 후 사업지 모니터링으로 복구사업 성공 여부 판정 및 향후 관리 방향 제시 |
| 기본 시기 | 작업 완료 후 3~5년 사이 1회 점검 |
| 5년 계획 복구지 | 중간 1회 점검 + 최종점검 실시 |
| 조사 방법 | 전수조사, 샘플조사, 개략조사 중 모니터링팀이 결정 |

### 모니터링팀 구성

| 피해지 유형 | 구성 기준 |
|------------|----------|
| 재난성 피해지 | 산림청 주관으로 지자체, 학계, NGO가 참여하는 공동조사단 구성 |
| 일반 피해지 | 시·도(시·군) 및 지방청(국유림관리소) 실시, 필요시 학계·NGO 참여 공동조사단 구성 |

### 결과 피드백

| 판정 | 후속 조치 |
|------|----------|
| 성공지 | 일반시업기준에 포함해 덩굴제거, 어린나무가꾸기 등 실시 |
| 실패지 | 재조림 또는 보식 |

---

## 2. 풀베기

| 항목 | 기준 |
|------|------|
| 대상 | 소묘 조림지 |
| 기간 | 조림 후 3~5년간 |
| 횟수 | 연 1~2회 원칙, 현장여건에 따라 조정 |
| 방법 | 낫 사용 원칙, 지역실정에 따라 작업방법 선택 |

---

## 3. 덩굴제거

| 항목 | 기준 |
|------|------|
| 기본 방법 | 물리적 방법 우선 |
| 화학적 방법 | 불가피한 경우에 한하여 실행 |

---

## 4. 비료주기

| 항목 | 기준 |
|------|------|
| 시기 | 묘목 활착이 완료된 5~6월 |
| 자재 | 완효성 비료 |
| 시비량·방법 | 수종과 묘목 상태를 감안해 결정 |
