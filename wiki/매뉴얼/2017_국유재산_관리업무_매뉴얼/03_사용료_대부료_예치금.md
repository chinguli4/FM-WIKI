# Ⅲ. 사용료·대부료와 예치금

> 원문: 574~881행  
> 숫자·요율·이자율·고시단가는 2017년 기준이므로 현행값으로 사용 금지.

## 사정조서의 역할

사정조서는 재산의 표시, 사용목적, 재산가액, 적용요율, 기간, 감면·조정 근거와 최종 부과액을 한 문서에서 검산하게 하는 통제문서다.

## 산정 시 확인 순서

1. 재산의 종류와 적용 법률 확정
2. 실제 이용상태와 목적사업 확정
3. 재산가액 기준자료 확인
4. 법정 요율과 특례·감면 근거 확인
5. 일할계산·기간·부가세 등 적용 확인
6. 전년도 금액과 조정 규정 확인
7. 분납·이자·담보 조건 확인
8. 사정조서와 시스템 금액 상호 검산

## 비용 항목

| 항목 | 발생 맥락 | 원문의 통제점 |
|---|---|---|
| 사용료·대부료 | 국유림 사용·수익 | 목적별 요율과 실제 이용상태 |
| 대체산림자원조성비 | 산지전용·일시사용 수반 | 감면·분납 여부와 납부 확인 |
| 산지복구비 | 산지 훼손·복구의무 | 매년 고시액 재산정과 추가예치 |
| 철거비 | 시설물 철거·원상회복 | 견적, 기간 할증, 보증서 유효기간 |
| 이행보증 | 분납·원상회복 담보 | 금액·보증기간·피보험자 확인 |

## 실제 이용상태 확인

공부상 지목만으로 재산가액과 요율을 확정하지 않는다. 임야가 농경지·공장부지 등으로 이용되는 경우와 같이 공부와 현황이 다르면 현황측량·인근 유사토지·감정평가 필요성을 검토한다.

## 시스템 처리

원문은 당시 dBrain의 요청·조정·수의계약·징수요청·수납정산 흐름을 설명한다. 이 부분은 화면 절차의 역사적 기록으로만 보존하며, 현행 시스템에서는 다음 결과만 공통 통제점으로 삼는다.

- 처분·계약 정보와 부과정보의 연결
- 법정 요율·감면근거의 기록
- 납부기한과 수납결과 확인
- 미수납 상태에서 권리효력 발생 여부 확인
- 회계·재산대장·문서관리시스템 간 일치

## 현행 검증 필요사항

- 정기 부과시기
- 분납 기준금액·횟수·이자율
- 사용료·대부료 요율과 감면 대상
- 조정 산식과 신규·갱신 구분
- 복구비·철거비 보증기간
- 전자납부와 시스템 명칭
