# Ⅻ. 원문·현행 기준 검증표

> 검증기준일: 2026-07-13  
> 이 표는 2017 매뉴얼을 현행 업무에 사용할 때의 우선 확인사항을 정리한다.

## 법령 기준선

| 자료 | 2026-07-13 상태 | 적용 |
|---|---|---|
| 국유림법 | 법률 제20079호, 2024-07-24 시행본이 현행 | 적용 |
| 국유림법 법률 제21562호 | 2026-10-22 시행 예정, §29 국유임산물 무상양여 대상 추가 | 아직 현행 아님 |
| 국유림법 시행령 | 대통령령 제34742호, 2024-07-24 시행 | 적용 |
| 국유림법 시행규칙 | 농림축산식품부령 제734호, 2025-09-15 시행 | 적용 |
| 산림청 소관 국유재산관리규정 | 산림청훈령 제1672호, 2025-01-07 시행 | 적용 |
| 국유재산법 | 법률 제21065호, 2026-01-02 시행 | 적용 |
| 국유재산법 법률 제21344호 | 2026-08-20 시행 예정, 일정 규모 이상 처분의 국회 보고 신설 | 아직 현행 아님 |
| 국유재산법 시행령 | 대통령령 제36423호, 2026-07-01 시행 | 적용 |
| 국유재산법 시행규칙 | 재정경제부령 제1호, 2026-01-02 시행 | 적용 |

## 2017 내용별 분류

| 주제 | 상태 | 처리 |
|---|---|---|
| 사용허가와 대부의 법적 성격 | 기본구조 유효 | 현행 조문으로 인용 갱신 |
| 국유림 구분과 재산종류 대응 | 단순화 위험 | 재산대장·국유림 구분을 별도 확인 |
| 복수 인허가·의제 원칙 | 기본구조 유효 | 의제 조문과 협의기관 현행 확인 |
| 허용행위·면적·신청자격 | 변경 가능성 높음 | 현행 법령·관리규정 우선 |
| 기간갱신·양도·명의변경 | 절차 참고 가능 | 기간·효력·서류 현행 확인 |
| 사용료·대부료·분납·감면 | 숫자 사용 금지 | 처분일 현재 법령·고시로 재산정 |
| 복구비·철거비·보증 | 구조 참고 가능 | 당해연도 고시액·보증기간 확인 |
| 실태조사·청문 | 구조 유효 | 2025 관리규정·행정절차법 적용 |
| 매각·교환·매수 | 구조 유효 | 심의·승인·평가 기준 현행 확인 |
| 무단점유·변상금 | 증거수집 구조 유효 | 산식·기간·소멸시효·집행수단 재검증 |
| dBrain·FLIS 화면 | 노후 | 현행 시스템 매뉴얼 확보 전 참고만 |
| 법정·내부 서식 | 노후 가능 | 현행 별지서식과 대조 후 사용 |

## 시행예정·입법예고 감시

- 국유림법 제21562호: 2026-10-22 시행 예정. 관리업무 본체보다 국유임산물 무상양여 범위에 영향.
- 국유재산법 제21344호: 2026-08-20 시행 예정. 일정 규모 이상 처분의 국회 보고 절차에 영향.
- 국유림법 시행령 일부개정안: 2026-05-22~07-01 입법예고. 장기 독점사용의 기간갱신 심사와 최초연도 대부료등 조정 제한을 다룸. **공포 전에는 적용하지 않는다.**
- 국유림법 시행규칙 일부개정안: 2026-06-18~07-28 입법예고. 국유림 입목수확 사업장 안전관리 사항을 다룸. **공포 전에는 적용하지 않는다.**
