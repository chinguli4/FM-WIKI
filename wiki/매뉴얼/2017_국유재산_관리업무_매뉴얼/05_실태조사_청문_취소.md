# Ⅴ. 실태조사·청문·취소

> 원문: 908~1012행

## 실태조사 목적

- 허가·대부 목적대로 사용되는지 확인
- 면적·용도·시설물 변경과 무단확장 확인
- 제3자 사용·전대·권리양도 여부 확인
- 사용료·대부료·부담금·보증의 이행 확인
- 산림훼손·재해·민원과 복구상태 확인
- 갱신·시정·취소·반환 등 후속조치 결정

## 조사자료

- 재산대장·등기·지적·임야도
- 허가서·계약서·조건·사업계획도
- 기존 실태조사와 시정명령 이력
- 납부·체납·보증서 현황
- 항공사진·공간정보·현황측량
- 현장사진과 상대방 확인서

## 위반 발견 시 분기

| 유형 | 우선 조치 |
|---|---|
| 경미한 서류·관리 미비 | 보완·시정 요구와 기한 부여 |
| 목적사업 미진행 | 진행기준에 따른 평가, 갱신 제한·취소 검토 |
| 무단 용도변경·면적확장 | 현황측량, 변상금·원상회복·변경허가 가능성 검토 |
| 전대·무단 권리양도 | 사실관계와 계약·허가조건 확인, 취소 검토 |
| 체납 | 독촉·연체료·강제징수 또는 계약상 조치 검토 |
| 산림훼손·불법시설 | 산지·국유림·건축 등 법률별 조치 병행 |

## 청문 통제점

1. 취소·철거 등 처분의 법적 근거와 처분권자 확인
2. 처분 원인이 된 사실과 증거 특정
3. 사전통지에 법적 근거·사실·처분내용·의견제출 방법 명시
4. 청문 대상인지 의견제출 대상인지 구분
5. 상대방의 주장과 제출자료 조사
6. 처분사유별 판단과 비례·평등 검토
7. 이유제시와 불복방법 고지

## 현행 적용

청문·사전통지·이유제시는 [[타부처법령/법제처/행정절차법]]을 함께 적용한다. 사용허가 취소와 대부계약 해지는 법적 성격이 다르므로 동일한 서식과 절차를 기계적으로 사용하지 않는다.
