# 공익림가꾸기 사업시행 기준 — 제2장 추진체계·실시설계·선목

> 출처: 공익림가꾸기 사업시행 기준 (2026-04-10)  
> 관련 페이지: [[1장_적용기준]] · [[3장_감리_사업시행]] · [[4장_기능별_작업방법]]

---

## 1. 추진체계 (6단계)

| 단계 | 구분 | 추진주체 | 주요 역할 |
|------|------|---------|----------|
| 1 | 사업계획 (기본설계) | 담당공무원 | 산림경영계획에 따른 대상지 선정·확정, 당해연도 2월까지 계획 완료 |
| 2 | 실시설계 | 담당공무원·산림기술용역업체 | 기능배정·공간구분·산림관리목표 설정, 설계도·서 작성 |
| 3 | 선목 | 산림기술용역업자 (감리자 제외) | 교목층(10cm↑) 및 아교목층(4~8cm) 제거목 선목 |
| 4 | 사업시행 | 산림조합·산림사업법인·국유림영림단 | 실시설계서에 따라 사업시행, 감리자와 기술협의 |
| 5 | 감리·관리감독 | 담당공무원·산림기술용역업체 | 설계검토, 현장점검·기술자문, 감리완료보고서 제출 |
| 6 | 준공검사·사후관리 | 담당공무원 | 이력관리시스템 입력, 공간정보 관리대장 관리 |

---

## 2. 사업계획 (기본설계)

- 작성 주체: 담당공무원 (기본설계를 용역으로 시행 시 산림기술용역업자 가능)
- 작성 시기: 전년도부터 당해연도 **2월까지**
- 대상지 선정 기준: 교목층 **수관울폐도 71% 이상** (단, 생활환경보전림·산림휴양림 등 중층가꾸기 필요지역은 71% 미만도 가능)
- 사업이력 확인 후 **사업 후 5년이 경과하지 않은 산림은 제외**

---

## 3. 실시설계

### 가. 실시설계 실행 순서

```
(공통) 사업계획 검토 → 임·소반 구획 → 산림기능 배정 → 공간구분 → 산림관리목표 설정
(핵심) 구역경계 표시 → 핵심구역 경계표시 → 표준지 조사 → 선목 → 설계도·서 작성
(일반) 구역경계 표시 → 표준지 조사 → 설계도·서 작성
```

### 나. 목표임분 유형

| 유형 | 층위 구성 | 주요 적용 대상지 |
|------|----------|---------------|
| A형 (공익-단층림) | 교목층 | 공간이용지점, 미세먼지저감숲 |
| B형 (공익-2단림) | 교목+관목층 | 개방형 임연부, 개방형 산책로 |
| C형 (공익-3단림) | 교목+아교목+관목층 | 계곡부, 폐쇄형 임연부, 침엽수 밀생림 |

> 일반구역은 우량대경재 생산 목표의 목재생산림(D~F형)에 준하는 목표 설정

## 4. 선목

### 가. 선목 주체
- 산림기술용역업자 수행 (해당 사업의 감리자는 선목자로 선정 불가)

### 나. 선목 방법

| 구역 | 선목 여부 |
|------|---------|
| 핵심구역 | 반드시 선목 후 작업 시행 |
| 일반구역 | 필요성·사업여건에 따라 자율 결정 |

- 선목 대상: 교목층(흉고직경 **10 cm 이상**) + 중층가꾸기 시 아교목층(흉고직경 **4～8 cm**)
- 제거목 표시: 수간부 앞·뒷면에 **적색페인트**로 "O" 표시

**제거목 선목 순서**

| 남길 나무 | 제거할 나무 |
|----------|-----------|
| 우세하고 가치 있고 건강한 나무 | 보육대상목(목표수종)의 수관·줄기에 피해 주는 나무 |
| 혼효림으로 경영 가치 높은 나무 | 병들고 연약하여 죽어가는 나무 |
| 중·하층 구성 활력 있는 나무 | 줄기 가늘고 수관발달 빈약한 피압목 |
| 임연부 구성 나무 등 | 굽고 수형 불량한 나무, 줄기 결함목 |

### 다. 소요인력

**소요인력 (제거목 선목)**

| 구분 | 소요 인력 | 적용인력 |
|------|---------|---------|
| 기본품 | 0.5 인/ha | 초급기술자 |
| 제거목 선목 | 0.2 인/100본 | 초급기술자 |

> - 교목층(10 cm↑): 기본품 + 제거목 선목품 합산  
> - 아교목층(4～8 cm, 중층가꾸기용): 제거목 선목품만 적용  
> - 예) 교목 300본/ha + 아교목 200본/ha 선목 시: 0.5 + 0.2×(300+200)/100 = **1.5 인/ha**

**할인·할증**

| 구분 | 세부 | 할증률 |
|------|------|--------|
| 산지경사 | 급 (30° 초과) | +10% |
| | 중 (15～30°) | +5% |
| | 완 (15° 미만) | 0% |
| 장애물의 정도 | 가슴높이 이상 초본·관목 | +10% |
| | 가슴높이 미만 초본·관목 | +5% |
| | 무릎높이 이하 초본·관목 | 0% |
| 선목시기 | 5～9월 | +10% |
| | 12～2월 | +5% |
| | 10～11월·3～4월 | 0% |

**소요재료 (100본당)**

| 재료 | 소요량 | 잡품 |
|------|--------|------|
| 친환경 적색 수성페인트 | 0.17 ℓ (본당 1.7 ㎖) | 주재료비의 10% |

> 소요재료는 선목 본수(교목 + 아교목) 합산 적용
