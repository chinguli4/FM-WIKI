# 산불예방 숲가꾸기 사업시행 기준 — 제2장 추진체계·실시설계·선목

## 1. 추진체계

| 단계 | 구분 | 추진주체 | 비고 |
|------|------|---------|------|
| 1 | 사업계획 | 담당공무원 | 당해연도 2월까지 |
| 2 | 실시설계 | 기술용역업체 또는 담당공무원 (국유림) | 핵심구역 선목 통합 또는 분리 발주 가능 |
| 3 | 선목 | 산림기술용역업자 (감리자 제외) | 핵심구역 교목층(10 cm↑) 반드시 선목 |
| 4 | 감리 | 기술용역업체 또는 감독공무원 (국유림) | 설계용역 완료 전 ~ 사업완료 후 15일 이내 |
| 5 | 준공검사 | 담당공무원 | — |

## 2. 사업계획

### 가. 작성 주체·시기

- 담당공무원 작성 원칙; 전년도부터 당해연도 **2월까지**

### 나. 사업추진 시기

- 산불 위험이 높은 **봄철(2～5월), 가을철(11～12월) 이전** 추진 원칙
- 소나무류 반출금지구역은 **소나무재선충병 방제기간 내** 실시 (→ [[1장_적용기준]] 방제기간 표 참조)

### 다. 대상지 선정

- 사업 실행 후 **5년 이상 경과**한 산림 대상 (5년 미만이면 제외)
- '산불 숲 관리 우선지역 지도' **A·B등급** 우선 선정
- A·B등급 없어도 산불취약시설(도로·주택·농경지 등) 연접지이면 임상 조건 검토 후 선정 가능

### 라. 사업계획서 내용

| 항목 | 내용 |
|------|------|
| 사업개요 | 위치·면적·사업기간·소요예산 |
| 대상지 특성 | 산불 숲 관리 우선등급, 임황, 과거 산불 이력 |
| 설계지침 | 공간구분(핵심 50~100 m/일반), 작업강도, 선목방법, 산물처리 방법 |
| 붙임자료 | 소반·필지별 조서, 위치도 |

## 3. 실시설계

### 가. 작성주체

- 기술용역업체 또는 담당공무원 (국유림)
- 핵심구역 선목을 실시설계에 **통합 발주**하는 경우: 설계용역비(A) + 선목사업비(B)를 각각 산출하여 합산
- **분리 발주**하는 경우: 설계 완료 전 당초 설계용역자를 선목사업자로 지정 계약

### 나. 사업지 현황조사 및 구역확정

**소요인력 (ha당)**

| 구분 | 소요 인력 | 인력 구분 |
|------|---------|---------|
| 대상지 구역경계 표시 | 0.20 | 초급기술자 |

> 흰색 마킹테이프 10 m 내외 간격

**할인·할증**

| 구분 | 세부 | 할증률 |
|------|------|--------|
| 집단화 정도 | 1 ha 미만 | +10% |
| | 1～3 ha 미만 | +5% |
| | 3 ha 이상 | 0% |

**소요재료 (ha당)**: 임업용 흰색 마킹테이프 **0.44 Roll**

### 다. 핵심구역 경계표시

**소요인력 (ha당)**

| 구분 | 소요 인력 | 인력 구분 |
|------|---------|---------|
| 핵심구역 경계표시 | 0.20 | 초급기술자 |

**할인·할증**: 산지경사 급(+10%) / 중(+5%) / 완(0%)

**소요재료 (ha당)**: 임업용 흰색 마킹테이프 **0.73 Roll**

### 라. 표준지 조사

| 구분 | 조사비율 | 최소 배치 | 표준지 크기 |
|------|---------|---------|-----------|
| 핵심구역 | 5% 이상 | 최소 3개 (소반분리 시 소반별 1개↑) | 400 ㎡ (20×20 m) |
| 일반구역 | 1% 이상 | — | 400 ㎡ |

> - 핵심구역 표준지 경계: 근주부위(지면 30 cm 이하)에 흰색 페인트; 작업 실행  
> - 하층 소나무류 피복도 조사: 소(10% 미만)·중(20～50% 미만)·밀(50% 이상)

**소요인력 (ha당)**

| 구분 | 소요 인력 | 인력 구분 |
|------|---------|---------|
| 표준지 조사 (핵심) | 0.25 조 | 2인 1조 (고급 1 + 초급 1) |
| 표준지 조사 (일반) | 0.05 조 | 2인 1조 |

**할인·할증**

| 구분 | 세부 | 할증률 |
|------|------|--------|
| 조사계절 | 5～9월 | +10% |
| | 12～2월 | +5% |
| | 10～11월·3～4월 | 0% |
| 산물수집 | 수집 또는 정리 | +20% |
| 집단화 정도 | 1 ha 미만 | +10% |
| | 1～3 ha 미만 | +5% |
| | 3 ha 이상 | 0% |

**소요재료 (ha당)**: 친환경 백색수성페인트 **0.2 ℓ**, 잡품 주재료비의 10%

산불예방 숲가꾸기산물 수집·반출을 위한 작업로 선정 시 「산림사업 표준품셈」 3-2 적용

## 4. 선목

### 가. 선목 주체

- 산림기술용역업자 (해당 사업의 감리자는 선목자로 선정 불가)

### 나. 선목 방법

| 구역 | 선목 여부 |
|------|---------|
| 핵심구역 | 교목층 반드시 선목 후 작업 |
| 일반구역 | 필요성·사업여건에 따라 자율 결정 |

- 선목 대상: 흉고직경 **10 cm 이상** 소나무류 제거목
- 제거목 표시: 수간부 앞·뒷면에 **적색페인트**로 "O" 표시

**제거목 선정 순서**

| 우선 순위 | 대상 |
|---------|------|
| ① | 고사목 |
| ② | H/D율이 높은 나무 (수고 대비 경급 떨어지는 세장목) |
| ③ | 지하고가 낮은 나무 |
| ④ | 역지 이하 가지가 많은 나무 |

> 활엽수는 존치 원칙; 단, 과밀로 생육저하 우려되거나 폭목은 제거 가능  
> 선목비는 「숲가꾸기 설계·감리 및 사업시행 지침」 품셈 기준으로 원가계산

### 다. 소요인력

**소요인력 (제거목 선목)**

| 구분 | 소요 인력 | 적용인력 |
|------|---------|---------|
| 기본품 | 0.5 인/ha | 초급기술자 |
| 제거목 선목 | 0.2 인/100본 | 초급기술자 |

> 기본품 + 제거목 선목품 합산  
> 예) 제거목 300본/ha: 0.5 + 0.2×(300/100) = **1.1 인/ha**

**할인·할증**

| 구분 | 세부 | 할증률 |
|------|------|--------|
| 산지경사 | 급 (30° 초과) | +10% |
| | 중 (15～30°) | +5% |
| | 완 (15° 미만) | 0% |
| 장애물의 정도 | 가슴높이 이상 초본·관목 | +10% |
| | 가슴높이 미만 초본·관목 | +5% |
| | 무릎높이 이하 초본·관목 | 0% |
| 선목시기 | 5～9월 | +10% |
| | 12～2월 | +5% |
| | 10～11월·3～4월 | 0% |

**소요재료 (100본당)**

| 재료 | 소요량 | 잡품 |
|------|--------|------|
| 친환경 적색 수성페인트 | 0.17 ℓ (본당 1.7 ㎖) | 주재료비의 10% |
