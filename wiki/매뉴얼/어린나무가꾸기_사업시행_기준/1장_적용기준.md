## 1. 어린나무가꾸기 개념

### 정의

조림목과 경쟁하는 목적 이외의 수종, 그리고 조림목 중 형질불량목·폭목을 제거하여 조림목이 정상적으로 생장할 수 있도록 하는 작업. 풀베기 작업이 끝난 후 실시한다.

### 대상지 기본 요건

| 기준 | 범위 |
|------|------|
| 조림 후 경과년수 | 5～15년 |
| 조림목 평균 가슴높이지름 | 2～8 cm |
| 조림목 생존본수 비율 | 26% 이상 |

### 조림목 생존율별 대상지 구분

| 구분 | 기준 |
|------|------|
| 조림성공지 | 당초 식재본수 대비 50% 이상 생육 |
| 조림목 혼생지 | 조림목(유사수종 포함)이 26～49% 생육 |
| 제외 | 생존본수 비율 25% 이하 |

> ⚠️ 조림목 생존본수 비율 = 생존본수 ÷ 당초 식재본수 × 100

### 대상지 유형 구분 (제거대상의 수고 비율 기준)

| 유형 | 기준 | 사용 장비 |
|------|------|-----------|
| 유형1 | 제거대상의 50% 이상이 조림목 수고보다 **작다** | 예취기 |
| 유형2 | 제거대상의 50～80%가 조림목 수고보다 **크다** | 체인톱 |
| 유형3 | 제거대상의 81% 이상이 조림목 수고보다 **크다** | 체인톱 |

> ⚠️ 유형 구분은 실시설계 표준지 조사 결과로 확정하며, 현장 확인 후 변경 필요 시 발주처와 협의

### 사업추진 시기

- 연중 실시 가능
- 맹아 발생 억제가 필요한 경우: **5～9월** 집중 실시
- 제거대상이 조림목 생장을 방해하기 시작한 연도에 1회 실시; 피해 계속 시 반복 시행

## 2. 대상지 선정 방법

### 의사결정 흐름

```
① 조림 후 경과년수 5～15년 여부 확인
        ↓ YES
② 조림목 평균 가슴높이지름 2～8 cm 확인
        ↓ YES
③ 조림목 생존본수 비율 26% 이상 확인
        ↓ YES
④ 제거대상 수목 유·무 및 대상지 유형 구분
   ├─ 유형1: 제거대상의 50% 이상이 조림목 수고보다 작은 경우
   ├─ 유형2: 제거대상의 50% 이상이 조림목 수고보다 큰 경우
   └─ 유형3: 제거대상의 81% 이상이 조림목 수고보다 큰 경우

①②③④ 중 하나라도 미충족 → 대상지 제외
```

### 조사 데이터 출처

| 항목 | 확인 방법 |
|------|-----------|
| 조림 후 경과년수 | 조림대장, 국유림·사유림 경영정보시스템 |
| 조림목 평균 가슴높이지름 | 표준지 조사 결과 |
| 조림목 생존본수 비율 | (조사본수 ÷ 당초 식재본수) × 100 |
