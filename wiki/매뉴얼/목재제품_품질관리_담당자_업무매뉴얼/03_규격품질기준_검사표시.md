# Ⅲ. 목재제품 규격·품질기준 및 품질검사·표시

> 근거: 「목재의 지속가능한 이용에 관한 법률」 §20, 「목재제품의 규격과 품질기준」

## 대상 품목

제재목, 방부목재, 난연목재, 목재 플라스틱 복합재, 집성재, 합판, 파티클보드, 섬유판, 배향성 스트랜드보드, 목질바닥재, 목재펠릿, 목재칩, 목재브리켓, 성형숯, 숯의 15개 품목이다.

## 검사 흐름

1. 생산·수입 제품의 품목과 검사 대상 여부를 확정한다.
2. 지정 검사기관 등에 신청서와 품목별 첨부서류를 제출한다.
3. 시료채취·시험 후 결과통지서를 받는다.
4. 결과에 이의가 있으면 통지일부터 30일 이내 이의신청한다.
5. 검사 면제 해당 여부와 관련 증빙을 확인한다.
6. 검사결과 및 관계서류를 법정 기간 보관한다.

## 검사 수행 주체

- 국내 검사기관: 국립산림과학원장이 검사능력을 검증하여 품목별 지정·고시
- 외국 검사기관: 수출국 정부가 공인하고 국내 기준과 동등 이상의 검사능력을 갖춘 기관
- 자체검사공장: 법령상 요건과 검사능력을 갖춰 지정받은 공장
- 목재등급평가사: 등록 후 제재목·집성재의 법정 직무 범위에서 평가

## 품질표시

| 구분 | 표시 시점·방법 |
|---|---|
| 비연료형 | 검사받은 제품을 판매·유통·보관하려는 때 |
| 연료형 | 생산자는 판매·유통하려는 때, 수입자는 통관하려는 때 |
| 원칙 | 제품 낱개마다 표시 |
| 예외 | 낱개 표시가 곤란하면 최소 유통단위 묶음에 표시 |
| 언어·형식 | 한글 원칙(필요시 영문 병기), 인쇄·스탬프·스티커·압인 가능 |

표시항목은 품목별 부속서가 정하므로 결과통지서와 현물 표시를 항목별로 대조한다.
