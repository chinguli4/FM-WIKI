# Ⅰ. 개요

> 근거: 「목재의 지속가능한 이용에 관한 법률」 §17, §20, §22~§27, §36, §39~§47

## 목적과 적용범위

- 목재제품 품질관리·단속의 표준 세부절차를 제시하여 기관별 처리 지연과 편차를 줄인다.
- 지방자치단체와 지방산림청의 목재생산업 등록·감독, 제품 검사·표시, 안전성평가, 행정·사법처리에 활용한다.
- 업무 참고자료이며, 위법행위의 판단과 제재는 현행 법령·고시에 따른다.

## 핵심 용어

| 용어 | 의미 |
|---|---|
| 목재 | 입목·죽을 벌채한 산물로서 원목과 수입 산물을 포함 |
| 원목 | 벌채 후 제재하지 않은 통나무 |
| 목재제품 | 목재 또는 목재와 다른 원료를 가공한 제품 중 목재가 50% 이상 포함된 제품(수입품 포함) |
| 자가소비자 | 생산·수입한 제품을 판매·유통하지 않고 직접 최종 소비하는 자 |
| 연료형 목재제품 | 목재펠릿·목재브리켓·목재칩·성형숯·숯 |
| 비연료형 목재제품 | 연료형을 제외한 목재제품 |
| 보관 | 검사받은 제품을 판매·유통하려고 일시 적치한 상태. 원료·중간재·생산단계 제품은 제외 |
| 반송 | 수입통관을 거치지 않고 외국으로 다시 반출하는 것 |
| 생산국 | 원료·중간재를 이용해 판매·유통 또는 통관할 최종 제품을 만든 국가 |
