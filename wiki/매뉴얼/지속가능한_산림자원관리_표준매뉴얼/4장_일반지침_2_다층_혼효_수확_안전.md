## §4. 다층림(복층림)의 조성·관리 (line 5511~5561)

### 1. 단목택벌(單木擇伐)에 의한 조성

#### 대상지
- 입지조건 양호·집약적 산림관리 가능한 **Ⅴ영급 이상** 임지로 대경재 생산 가능
- 침엽수림 ha당 **300그루**, 활엽수림 ha당 **200그루** 가량 대경재 최종 수확 가능 산림
- 공익기능 유지·모두베기 부적당 산림

#### 조성방법
- 최종 수확본수 ha당 200~300그루 내외 조절·임분 내 고루 배치
- 상층목에서 **2m 떨어진 공간에 1.8m 간격 수하식재**
- 천연하종갱신 가능 임지: 갱신상 조성·움싹갱신·수하식재 병행

### 2. 대상벌채에 의한 조성

#### 대상지
- 산림병해충 피해지·입목형질 불량 임지 중 임분 전환·수종갱신 필요
- **Ⅲ영급 이상** 조림지·형질불량 활엽수림
- 인공림 일반소경재·천연림 특용·소경재 생산 가능 임분
- 공익기능 유지·모두베기 부적당

#### 조성방법
- 식재열 기준 **3~5열을 교호로 2회 대상벌채**하여 갱신
- 잔존구 벌채: 식재목이 하층식생 영향 벗어난 이후
- 잔존대로부터 **2m 떨어진 벌채대 내에 1.8m × 1.8m 간격 식재**
- 천연갱신 가능 임지: 갱신상·움싹갱신·식재조림 병행

## §5. 혼효림 조성 (line 5563~5621)

여러 수종의 특성으로 양질재 생산·지력 증진·각종 위해방지·산림경관 다양도 제고 → 단순림 결점 보완.

### 1. 식재에 의한 조성

#### 대상수종 혼합 — 4가지 원칙

1. **심근성 + 천근성**, 양수 + 음수, 상록 + 낙엽 (지상·지하부 빛·수분·양료 복합 활용)
2. **척박 임지** → 임지 비배효과 큰 **비료목**을 부수종으로 혼효
3. 주수종 가지발달 억제·양질재 생산·생력화 위해 **부림목** 혼효
4. **내화성 수종**·상록·낙엽 혼효 (산불·기상재해·병해충 방지)

#### 조성방법

| 방법 | 내용 |
|------|------|
| **군상혼효** | **3×3, 4×4, 5×5 그루** 등 임지 전면에 규칙적 모자이크 모자 배열 |
| **열상혼효** | 식재열 기준 1열·2열·3열…X열 등 수종 교대 배열 |
| **대상혼식** | 기존 임분 수고에 해당하는 폭만큼 벌채지를 대상으로 교대 수확·식재 |

### 2. 육림에 의한 조성

인공조림지에서 조림목과 혼효하는 **자생 유용수종을 무육**하여 일정 비율 혼효 전환.

- **정방형 조림지**: 어린나무가꾸기 작업 시 자생 유용수종 무육·혼효 → 전면적 침·활 혼효림
- **부분밀식 조림지**: 대상·군상 부분밀식 후 공간지 유용수종 충분 시 적용 → **모자이크 군상 혼효림**

## §6. 임연부(林緣部)의 조성·관리 (line 5623~5633)

### 대상지

산림과 산림 아닌 지역 경계 지점으로부터 **산림 지역 방향 30m 내외**. 다만 **생태적 격리로 판단되지 않는 5m 미만의 임도·시설물 등은 임연부에서 제외**.

### 숲가꾸기 방법

1. 다양한 수종 조성·생태계 종다양성·시각적 다양성 제고
2. **산림이 아닌 지역으로부터 초본 → 관목 → 아교목 → 교목 순 계단형** 조성·관리
3. 밀생 임분: **약도 솎아베기 5년 내외 간격 수회**
4. 임연부 내 산물 전량 수집·활용 또는 지면 닿도록 잘라 부식 촉진
5. 풍해 등 예방 위해 가지치기 미실시 (임내 투시·감상 지역·경관 중요 지역 제외)

## §7. 수확 ★★★ (line 5636~6300)

목재수확 = 입목 벌채 → 일정 규격 원목 조재 → 임도·집재장 집재 → 목재시장·제재공장 운반.

### 1. 작업계획 및 시스템 구성

#### 작업 조건 — 지형 경사 3구분 ★

| 구분 | 경사 | 적용 집재방법 |
|------|------|--------------|
| **완경사** | **30% 미만** | 지면끌기, 적재·운재 |
| **중경사** | 30~60% | 지면끌기, 가선집재 |
| **급경사** | **60% 이상** | 가선집재 |

#### 집재작업별 장·단점 [표 53]

| 구분 | 장점 | 단점 |
|------|------|------|
| **지면끌기 집재** | 기동성 높음·단순작업·낮은 비용·완중경사 | 집재거리 짧음(평균 50m)·**높은 임도밀도 필요** |
| **적재·운재** | 기동성 높음·단순작업 생산성·완경사 | **토양교란 큼**·**높은 노망밀도 필요** |
| **가선집재** | **잔존임분 피해 적음**·낮은 임도밀도·**급경사지 가능** | 기동성 떨어짐·고가 장비·숙련 기술·세밀한 작업계획·설치·철거시간 |

### 2. 벌기령 ★★★★

| **리기다소나무** | **30년** | **25년 (20년)** |

| **삼나무** | 50년 | 30년 (30년) |

#### 특수용도 기준벌기령

펄프·갱목·**표고·영지·천마 재배**·목공예·**숯**·**목초액**·섬유판(fiber board)·**산림바이오매스에너지** 용도 → 일반기준벌기령 중 **기업경영림 기준벌기령 적용**. **단 소나무는 특수용도 기준벌기령 미적용**.

#### 기준벌기령 예외

다음은 기준벌기령 미적용:
- **불량림 수종갱신**을 위한 벌채
- 피해목·옻나무·약용류·지장목 벌채
- 임지생산능력급수 **Ⅰ~Ⅲ급지의 리기다소나무 벌채**

> 본 매뉴얼의 기준벌기령은 [[../../법령/산림자원법/시행규칙]] **별표 3 기준벌기령**과 함께 산림 행정의 직접 근거.

### 3. 목재생산방법 — 3가지

| 방법 | 작업 위치 | 주요 장비 |
|------|----------|----------|
| **전목 생산** | 임도변·토장에서 가지치기·통나무 자르기 | 벌도(체인톱·하베스터) → 전목집재(스키더·타워야더·스윙야더) → 조재(프로세서) → 운재(트럭) |
| **전간 생산** | 임분 내 벌도·가지치기 후 임도변·집재장 집재 | 벌도·지타(체인톱·하베스터) → 전간집재(스키더·타워야더·스윙야더) → 조재(프로세서) → 운재(트럭) |
| **단목 생산** | 임분 내 체인톱으로 벌도·가지치기·통나무 자르기 일괄 (**우리나라에서 가장 많이 활용**) | 벌도(체인톱·하베스터) → 조재(체인톱·하베스터) → 단목집재(포워더·타워야더·스윙야더) → 운재(트럭) |

### 4. 지형지수에 의한 시스템 선정

지형지수(F) = f(P30, q60) — **P30**(경사 30% 미만 면적비율), **q60**(경사 60% 이상 면적비율).

| F 범위 | 시스템 |
|--------|--------|
| **F < 40** | **지상시스템** |
| 40 ≤ F < 60 | 혼합시스템 |
| **F ≥ 60** | **가선시스템** |

### 5. 작업시스템 — 경사별 [표 55~59]

| 경사 | 형태 | 표 |
|------|------|---|
| **완경사 (35% 미만)** | 벌목·조재 → 집재 → 운재 | [표 55] — 체인톱+인력/수라/임내차/포워더, 하베스터+포워더 |
| 완경사 | 벌목 → 집재 → 조재 → 운재 | [표 56] — 체인톱·트랙터·그래플쏘·포워더 |
| **중경사 (35~60%)** | 벌목·조재 → 집재 → 운재 | [표 57] — 체인톱+수라/임내차/HAM200+포워더 |
| 중경사 | 벌목 → 집재 → 조재 → 운재 | [표 58] — 체인톱·트랙터·그래플쏘·포워더 |
| **급경사 (60% 이상)** | 벌목 → 집재 → 조재 → 운재 | [표 59] — 체인톱+라디캐리·HAM200·타워야더+체인톱/프로세서/그래플쏘+포워더 |

### 6. 산림작업로 ★

#### 현행 산림작업로 종류 [표 60]

| 구분 | 내용 |
|------|------|
| **임산물 운반로** | 임산물 운반 위해 일시 산림 내 설치 도로. **너비 3m 이내**, **설치 후 복구 원칙** (배향곡선·대피소·돌리기 등 부득이한 경우·토석운반로는 3m 초과 가능) |
| **작업로** | 임산물 생산·관리 위해 산림 내 설치. **임산물 운반로와 동일한 시설기준** |

#### 지속적 활용 산림작업로 노선선정 절차

1. 대상경영계획구 전체 **임도망 기본계획** 수립
2. 임도노선이 목재수확대상지 통과 → **지속적 활용 산림작업로로 계획·사용 후 존치**
3. 기타 산림작업로는 효율적 산림자원 조성·관리 + 목재생산 고려 배치, **사용 후 복구**

|---------|-----------|----------------|----------------|
| 완경사 (~15°) | 차량계 | 30~40 | 70~210 |
| 중경사 (15~30°) | 차량계 | 23~34 | 52~165 |
|  | 가선계 | - | 2~41 |
| 급경사 (30~35°) | 차량계 | 16~26 | 35~124 |
|  | 가선계 | - | 0~24 |
| **급준지 (35°~)** | **가선계** | **5~15** | - |

### 7. 집재장

- 신규 조성 시 **운재로·작업로 최소 시설** 곳
- **다수 집재장 간 최소 200m 내외**
- **경사 5% 이내, 배수 양호, 지반 견고**
- 침식 발생 억제 위해 통나무 배수시설
- 장마철 사용 시 **목재 매트·자갈** 깔기

### 8. 벌목작업

#### 친환경 벌채실행방법 — [[../../행정규칙/고시/친환경벌채_운영요령]] (산림청고시 2023-60호)

**모두베기**:
- 벌채구역 **최대 30ha 이내**
- 벌채구역 **5ha 이상** 시 **면적의 20% 이상을 군상 또는 수림대로 남김**
- 인접 벌채 시 — 신규 **5ha 이상**: 폭 30m / **5ha 미만**: 폭 20m 이상 거리 유지

**골라베기**:
- 재적기준 **30% 이내**
- **버섯용 원목** 골라베기: **재적기준 50% 이내**

**모수작업**:
- 종자 결실 풍부·천연갱신 가능
- 벌채구역 **5ha 이내**
- 어미나무 형질 우수·바람 방향에 **ha당 15~20그루**
- 갱신상 조성 작업 완료
- 3년 이내 어린나무 발생량 **ha당 5,000그루 미만** 시 조림·보완조림

**왜림작업** (참나무 맹아 후계림):
- 생장휴지기 벌채
- 그루터기 높이 **지상 10cm 이내**
- 경사지 절단면은 지면과 평행, 물 고임 우려 시 약간 기울게

**불량림 수종갱신**:
- 산림청장 고시 **「임분의 수종갱신 판정표」**에 따른 갱신판정 임지

#### 벌도 절차

- **벌도 방향**: 나무 없는 공간으로 쓰러지도록 → 받침대·쐐기 사용
- **수구·추구** 정확히, **벌근 높이 지상 20cm 이하**, 밑부분 썩은 것은 위쪽 절단
- **걸림목 처리 전까지 다른 나무 벌채 금지** + 경고 표시
- 강우·강풍 등 악천후 시 작업 금지 (단 진행 중 벌도목은 완료)

#### 수확을 위한 벌채금지 구역 9가지 ★

1. 생태통로 역할 **8부 능선 이상부터 정상부**
2. 암석지·석력지·황폐우려지로 갱신 어려운 지역
3. 계곡부 양안 **홍수위**
4. 호소·저수지·하천 수변 만수위로부터 **30m 내외**
5. 도로변 도로로부터 **평균 수고폭**
6. **임연부**
7. **내화수림대**로 조성·관리되는 지역
8. 벌채구역 간 **20m 폭의 잔존수림대** (어린나무가꾸기 도달 시점에 벌채 가능)
9. 운재로·산림재해 복구 등 공익 목적 — 최소 필요부분만 실행

## §8. 안전관리 (line 6301~6360)

### 1. 안전보건관리계획 수립 — 9개 항목

1. 안전보건 관리조직 구성·운영
2. 안전점검계획
3. 사업장 안전보건 관리대책
4. 안전보건 교육계획
5. 산림사업재해 발생 시 대응조치
6. 위험요인 파악·개선 대책
7. 산림사업종사자 의견청취 관리 계획
8. 안전보건 성과·평가
9. 그 밖에 안전·보건 필요사항

### 2. 안전보건관리자 업무 — 5개

1. 산림사업종사자 안전보건 점검·이상 유무 확인
2. 작업복·보호구 점검·착용·사용 교육·지도
3. 산림사업재해 보고·응급조치
4. 작업장 정리정돈·통로 확보 확인·감독
5. 발주청·산림사업자 안전보건 협조·조언

### 3. 작업 전 안전점검회의(TBM, Tool Box Meeting)

- **1일 단위 진행**
- 주제: 작업 일정·안전 작업절차 / 위험 식별·검토 / 통제 방안 / 사례 공유

### 4. 위험성 평가 — 필요 조치 장소 5가지

1. 추락 위험
2. 나무 밀집·덩굴류 많음 → 맞음·깔림 위험
3. 광범위하여 종사자 안전 확인 어려움
4. **벌·뱀 등 독충 출현 많음**
5. 비·눈 등으로 미끄러질 위험

### 5. 개인보호구 — 6종 ★

- 산림작업 안전모
- 작업복 상의·하의·덧바지
- 안전화
- 안전장갑

> 미착용 시 업무 부여 금지. 휴게시설 제공 의무.

### 6. 작업중지·긴급조치

- 산업재해 발생·급박 위험 → 작업 중지·대피
- 재해 인지 시 즉시 작업 중지·대피·필요조치
- 부상·질병 발생 시 즉각적 치료 등 요양조치

## §9. 스마트 기술을 활용한 산림조사 방법 (line 6361~6488)

### 1. 라이다 (LiDAR, Light Detection and Ranging)

3차원 정보 구축 원격탐사 기술. 점군 자료로 **개체목 흉고직경·수고·수관면적·수관 간격·DSM·DEM** 생성.

#### 라이다 장비 5종

| 종류 | 플랫폼 | 특성 |
|------|--------|------|
| **지상형 고정라이다** (Terrestrial) | 고정형 | 건설·토목·문화재. **가장 정밀**, 취득시간 오래 |
| **지상형 이동라이다** (Mobile) | 차량·오토바이·백팩 | 도로·건축·도시계획. 정밀 |
| **휴대용 라이다** (Handheld) | 백팩·손잡이 | **이동성 뛰어남**, 낮은 점군 밀도 |
| **항공라이다** (Airborne) | 비행기·헬리콥터 | **대단위 면적 강점**, 빠른 속도, 높은 비용 |
| **무인항공라이다** (UAV) | 드론(고정익·회전익) | 다양한 분야, **저렴·높은 점군 밀도**, 기상 영향 |

### 2. 표준지 조사

- 표준지 중점에 **측고기(Haglof vertex) 폰더 설치**
- 표준목은 흉고직경 부근에 **번호판 부착** (라이다 식별)
- 장애물(하층식생·덩굴) 있으면 시야 가리지 않는 곳 재부착

### 3. 라이다 촬영 기준

#### 지상라이다
- **고정라이다 촬영 횟수**: 최소 **5~29회**
- **이동라이다(백팩·핸드헬드) 경로 간격**: 최소 **3~5m**
- **시작 위치 사면 경사 위쪽**, 지그재그(ㄹ형) 유지·장비 수평 유지

#### 공중라이다 (드론)
- **이착륙 지점 임도 또는 개활지**
- **촬영 간격 최소 50m** (상대고도 유지)
- **중첩율 종·횡 간격 최소 60~70%**
- **RTK 또는 PPK** 활용 정밀 공간정보 취득

### 4. 라이다 장비별 적용성

- **고정**: 일률적·정확성 높음, 시간 비효율적
- **백팩**: 효율적, SLAM 안정성 낮음 (구식)
- **핸드**: 효율적·SLAM 고도화로 안정성 높음
- **드론**: 대면적 효율적, **흉고직경 산출 어려움** → 본수 추출 + 흉고직경 산출식 개발 필요

## §10. 산림종묘 생산관리 (line 6489~7395)

### 1. 채종원

#### 수형목 선발 기준 (line 6505~6515) ★

- 선발 방법: 목측·비교·기선·절대표준치
- **천연림 ha당 1본, 인공림 ha당 2본**
- 기준 6가지:
  1. 상층 임관에 속할 것
  2. 주위 정상목 10본의 평균보다 **수고 5%, 직경 20% 이상** 클 것
  3. 수관이 좁고 가지가 가늘며 한쪽 치우치지 않음
  4. 수간이 완만하고 굵거나 비틀어지지 않으며 분지하지 않음
  5. 다양한 피해 저항성·재질 우수
  6. 상당량의 종자가 달림

> 수형목 지정·해제는 [[../../법령/산림자원법/법률_2장_조성육성]] §19에 규정. **지정 기간 10년, 연장 가능**.

#### 채종원 종류 4가지

| 종류 | 내용 |
|------|------|
| **차대검정 없는 클론채종원** | 수형목 접·삽수 채취·접삽목 후 채종원 조성. **유전력 높고 접삽목 잘 되는 수종** |
| **차대검정 있는 클론채종원** | 수형목 무성증식 + 차대검정. **널리 사용**, 검정 결과 불량목 제거 |
| **풍매종자에 의한 종묘채종원** | 접목 불화합·조기개화 잘 되는 수종 |
| **인공교배 종자에 의한 종묘채종원** | 풍매보다 개량효과 큼, 경비 많음 |

#### 채종원 조성 — 식재간격

- 우리나라 **주로 5m** (수형조절·종자채취 편리)
- **클론 채종원: 20~50개 클론**
- 자가교배 방지 위해 동일 클론·가계 이웃 배치 금지

#### 세대별 채종원

- **1세대**: 소나무·낙엽송·편백은 클론채종원 / 참나무류 등 무성번식 어려운 수종은 풍매종자 종묘채종원
- **2세대**: 소나무·곰솔·잣나무는 **인공교배 차대검정으로 특수조합능력 좋은 조합**의 2세대 클론채종원

#### 수형조절

- 단위면적당 결실면 확대·양질 종자 결실 유도
- **휴면기인 2~3월**에 주간 높이 **3~5m**, 마디 수 **3~5단**으로 정지작업
