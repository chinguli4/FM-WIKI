# 제2장 산림의 기능별 조성·관리 지침

> 상위: [[README]]
> 원본: `raw/manuals/2025년 지속가능한 산림자원관리 표준매뉴얼.md` line 209~380 (pp.9~18)

산림 기능구분은 본 매뉴얼·훈령 1454호·산림자원법 시행규칙 §3 공통 체계 — 6대 기능 + 3단계 절차.

---

## 1. 산림의 기능구분

### (1) 6대 산림기능 — 산림관리자가 소유·관리 산림을 다음 기능에 따라 구분

산림관리자 = 산림청장 + 국유림 소유·관리 중앙행정기관장 + 시·도지사·시장·군수·구청장.

| 기능 | 관리목표 |
|------|---------|
| **목재생산림** | 생태적 안정 기반, 국민경제 활동에 필요한 양질의 목재를 지속적·효율적으로 생산·공급 |
| **수원함양림** | 수자원 함양기능과 수질정화기능이 고도로 증진 |
| **산지재해방지림** | 산사태·토사유출·대형산불·산림병해충 등 각종 산림재해에 강한 산림 |
| **자연환경보전림** | 보호할 가치가 있는 산림자원이 건강하게 보전 |
| **산림휴양림** | 다양한 휴양기능 발휘, 종다양성 풍부·경관 다양 |
| **생활환경보전림** | 도시·생활권 주변의 경관유지 등 쾌적한 환경 제공 |

> 출처: 「지속가능한 산림자원 관리지침」(산림청훈령 제1454호, 2020-06-15)

### (2) 기능 중복 시

산림관리자가 산림의 위치, 입지조건, 지역주민·산주의 요구 등 경제적·사회적 여건을 종합적으로 고려하여 기능을 정할 수 있다.

### 기능구분 3단계 절차

1. **1단계**: 법적으로 지정·고시·관리되는 산림 → 산림이용기본도 적용
2. **2단계**: 1단계 미지정 산림 → 「지속가능한 산림자원 관리지침」 + 「국유림 영림계획 작성 및 운영요령」에 따라 산림관리자가 현장여건 고려 분류
3. **3단계**: 기능 중복 산림 → 관련법 상하체계 + 해당지역 정책방향·지역 요구사항 고려, 면적(임·소반)별 재조정

### 1단계 — 법적 지정 산림 기능구분 ★

| 기능 | 대상 산림 |
|------|----------|
| **목재생산** | 보전국유림, 경제림육성단지, 임업진흥권역 안의 목재생산을 위한 산림 |
| **수원함양** | 산림보호구역 중 **수원함양보호구역**, **상수원보호구역** 안의 산림, **5대강(한강·금강·영산강·섬진강·낙동강) 수계** 지역 안에서 수원함양에 직접 영향 산림, **댐으로 집수되는 자연경계구획** 산림 |
| **산지재해방지** | **사방지(산사태복구지 포함)**, 산림보호구역 중 **재해방지보호구역**, **소나무재선충병 피해지역** |
| **자연환경보전** | 채종림·채종원·시험림, 산림유전자원보호구역, **백두대간보호지역**, 보전녹지지역, 자연공원, 자연생태계·경관보전지역, **생태·자연도 1등급 권역**, 야생생물보호구역, 습지보호지역, 특정도서 산림, 사찰 소유 산림, 문화재보호구역, 수목원, 학술림, 학교 숲 |
| **산림휴양** | **자연휴양림**, **치유의 숲** |
| **생활환경보전** | **도시림**, 경관보호구역, 재해방지보호구역, 생활환경보호구역, **도시공원**, **개발제한구역** 안의 산림 |

### 2단계 — 1단계 미지정 산림 기능구분 ★

| 기능 | 대상 산림 |
|------|----------|
| **산지재해방지** | 산사태 우려 **과밀 임분(침엽수 단순림)**, 대형산불 우려 지역의 **침엽수 단순림**, 산림병해충 피해 우려 **단순림** |
| **생활환경보전** | 생활권 주변 등에 **미세먼지 저감** 기능 발휘 가능 산림 |
| 각 기능 | 산림관리자가 특정 기능 증진 필요 인정 산림 |

### 3단계 — 기능 중복 시 우선순위 ★★

법적 지정·고시 내용이나 현장여건상 기능이 중복된 경우:

1. **인명·재산 피해 유발 가능 산지재해방지림 우선 지정**
2. **국유림경영기본지침의 "보호기능 우선원칙"**에 의거 **자연환경보전림** 지정
3. **법적 지정·고시 면적이 큰 순**: 생활환경보전림 → 수원함양림 → 산림휴양림 → 목재생산림

기능 확정 후 **GIS 주제도**(지형도·임상도·산림입지도·산림이용기본도·임도망도 등)로 **산림기능도** 작성 → 국유림·민유림 임·소반 단위 영림계획 수립 활용.

## 2. 공익림가꾸기 개요

목재생산림을 제외한 **5대 공익기능**(수원함양·생활환경보전·산림휴양·자연환경보전·산지재해방지)을 최적 발휘 — 임상에 따른 별도 사업종 구분 없음.

### (1) 공익림가꾸기 공간 구분 — 핵심·완충·일반 ★

| 구역 | 정의 |
|------|------|
| **핵심구역** | 산림기능이 발휘되도록 **집중관리 필요** 지역. 관련법 구역 우선, 미정 시 산림관리자 결정 |
| **완충구역** | 핵심구역으로부터 일정거리까지의 **가상의 공간** — 현장 구획 아닌, 산림관리자가 현장여건에 부합하게 핵심구역을 조정할 수 있도록 마련. **핵심 또는 일반에 편입 가능** |
| **일반구역** | 핵심·완충 제외 지역. 생태적 특성 고려하여 **장벌기 목재생산림에 준하는 산림관리** 실시 |

### 완충구역 범위

- **계곡부**: 계곡의 **홍수위로부터 첫 번째 능선부까지**
- **산책로·임도·도로·시설지(동선) 주변**: **동선에서 가시되는 지역** (상부 = 첫 번째 능선부까지, 하부 = 계곡부까지)

### (2) 기능별 핵심관리 구간 ★★ ([표 4])

| 구분 | 핵심관리 구간 |
|------|---------------|
| **수원함양림** | **홍수위·만수위로부터 30m 이내** |
| **생활환경보전림** | **공원·산책로 등으로부터 30m 이내** |
| **산림휴양림** | **시설부지·산책로 등 공간이용지역으로부터 30m 이내** |
| **자연환경보전림** | 산림 보전관련 지정지역·보호대상 **30m 이내** |
| **산지재해방지림** | **산사태 위험지역·산불취약지역 주변(도로·주택·농경지 등 연접지로부터 50m 이내)** |

### (3) 공간 구분 방법 (4단계)

1. 관련 법규에서 기능별로 정한 공간을 **핵심구역**으로 우선 구획
2. 현장여건상 **추가적인 집중 관리 필요 지역**(침엽수 밀생임분·풍도목이 경관 저해·등산로·숲길 등)을 **핵심구역으로 추가 구획**
3. 핵심지역 중 공간의 **확대 또는 축소** 필요 지역을 **완충구역 범위 내에서 조정해 핵심구역 확정**
4. 사업대상지 중 핵심구역 제외 나머지를 **일반구역**으로 구분
