## 우선순위와 사용상 주의

- 시행지침은 **2026년도 사업 운영기준**이다. 법률·시행령·시행규칙과 충돌하면 상위 법령을 우선한다.
- 내부 표제상 2026-01-16 수정본으로, 2026-03-10 시행 법률과 2026-05-19 시행 시행령·시행규칙보다 먼저 작성되었다. 지침의 법령 인용·중복지급 기준은 현행 법령으로 다시 검증한다.
- 원문 추진일정에는 `변경될 수 있음`이라고 명시되어 있으므로 실제 공고일·접수기간은 해당 연도 공고와 시스템에서 다시 확인한다.
- 법정 지급요건 판단과 담당자 업무절차를 섞지 않는다. 신청을 접수했더라도 자격검증·조사위원회 심사·최종검증을 거쳐야 한다.
- PDF의 일부 글꼴은 텍스트 추출 시 깨지므로, 표·단가·감액률은 원문 화면으로 재확인하였다.
