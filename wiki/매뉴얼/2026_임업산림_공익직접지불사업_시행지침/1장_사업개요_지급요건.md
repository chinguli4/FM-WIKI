# 제1장 사업개요와 지급요건

> 원문: 2026년도 임업·산림 공익직접지불사업 시행지침 인쇄면 1~54  
> 적용: 2026년도 사업. 법정요건은 현행 임업직불제법령과 함께 확인

## 2026년 사업 구조

- 2026년 지급 예산: 532억원(국비 100%)
- 임산물생산업 직불금: 소규모임가직불금과 면적직불금
- 육림업 직불금: 면적직불금
- 2026년 도식상 최소 신청면적
  - 임산물생산업: 소규모임가·임업인 0.1ha, 농업법인 5ha
  - 육림업: 임업인 3ha, 농업법인 10ha

## 2026년 지급단가

| 유형 | 구간 | 단가 |
|---|---|---:|
| 소규모임가 | 지급대상 산지 0.1~0.5ha | 임가당 130만원 |
| 임산물생산업 면적 | 1구간 2ha 이하 | 94만원/ha |
|  | 2구간 2ha 초과~6ha 이하 | 82만원/ha |
|  | 3구간 6ha 초과 | 70만원/ha |
| 육림업 면적 | 1구간 10ha 이하 | 62만원/ha |
|  | 2구간 10ha 초과~20ha 이하 | 47만원/ha |
|  | 3구간 20ha 초과 | 32만원/ha |

> 송이는 별도 구간·단가 고시를 적용한다. 원문은 산림청고시 제2022-91호를 안내한다.

면적 상한은 임업인 30ha, 같은 임가 지급대상자가 2명 이상이면 임가 60ha, 농업법인 50ha이다. 임산물생산업과 육림업은 각각 상한을 계산한다.

## 동일·다른 경영지의 중복수령

- 동일 경영지에는 농업 기본형, 임산물생산업, 육림업 직불금을 중복 지급하지 않는다. 복합경영으로 둘 이상의 요건을 충족하면 하나를 선택한다.
- 다른 경영지는 농업과 임업의 면적직불금을 받을 수 있으나 농업+임업의 지급상한을 함께 확인한다.
- 농업 소규모농가직불금 수령자·농가 구성원과 임업 소규모임가직불금의 관계는 신청연도 법령과 지침 기준으로 별도 검증한다.

> ⚠️ 지침 인쇄면 25에는 농업 소규모농가직불금의 `직전 연도` 수령 여부를 표시하지만, 2026-03-10 시행 법률 제8조제3항제3호와 제9조제1항 단서는 **등록신청연도에 지급받은 자**를 기준으로 개정되었다. 이 부분은 현행 법률을 우선한다.

## 소규모임가 세부요건

임산물생산업 직불금의 기본 지급대상자 요건을 충족한 임가에서 다음을 모두 확인한다.

| 항목 | 2026년 기준 |
|---|---|
| 지급대상 산지 | 임가 내 모든 지급대상 산지 합계 0.5ha 이하 |
| 전체 산지소유 | 비임업인 포함 모든 구성원의 소유산지 합계 1.55ha 미만. 비지급대상·임대 산지도 포함 |
| 종사 | 모든 지급대상자가 신청연도 직전 계속 3년 이상 영농 종사 |
| 거주 | 모든 지급대상자가 신청연도 직전 계속 3년 이상 농촌 거주 |
| 개인 소득 | 지급대상자 각각의 전년도 농업 외 종합소득금액 2,000만원 미만 |
| 임가 소득 | 모든 구성원의 전년도 농업 외 종합소득금액 합계 4,500만원 미만 |
| 시설재배업 소득 | 지급대상자 각각 3,800만원 미만 |
| 농업 직불 | 농업 기본형 공익직불금을 지급받지 않은 자 |

주민등록표상 세대가 달라도 배우자, 미혼인 19세 미만 직계비속, 혼인 외 사유로 세대분리 3년 이내인 자는 같은 임가로 본다. 신청 시 주민등록등본과 가족관계증명서를 바탕으로 가족관계 인적정보 작성표를 완성한다.

지급대상 산지가 0.5ha를 초과하더라도 전체 소유산지가 1.55ha 미만이고 나머지 종사·거주·소득요건을 충족하며 면적직불금 합계가 130만원보다 적으면 소규모임가직불금과 면적직불금 중 선택할 수 있다. 소규모임가 요건은 미충족하나 면적직불금 요건을 충족하면 면적직불금으로 처리한다.

## 증명의 핵심

- 종사기간과 종사일수는 별도 요건이다. 2026년 신청은 원칙적으로 2025-01-01~2025-12-31의 계속 종사와 60일 이상 종사를 함께 본다.
- 60일 종사는 수기 또는 스마트 영림일지 등으로 증명한다. 간이영수증만으로 소규모임가 3년 종사를 증명하지 않는다.
- 배우자 등 같은 경영체 구성원의 산지는 분할하지 않고 같은 경영체의 다른 구성원이 신청하는 경우의 예외를 확인한다.
- 육림업은 벌채 후 신청일까지 재조림하지 않은 산지를 휴경으로 판단하는 원문 사례가 있다.
