# 국가산림경영정보시스템 입력 주의사항

> 시스템 화면은 개편될 수 있다. 메뉴명보다 입력자료·검산 관계·결재 증빙을 기준으로 사용한다.

## 후보지 조회

- 관리구역과 사업종을 반드시 `조건추가`한 뒤 검색
- 현차기 선택 확인
- 평균경급 검색은 24~100cm, 혼효율 50~100% 등 원문 조건을 후보지 추출에만 사용
- 평균경급 24cm 이상 검색결과를 `24cm 이상 입목 50% 분포`로 간주하지 않고 현장 확인
- 경영계획부, 사업이력, 제한구역 레이어의 기준일 기록

## 공간정보

- 경영계획부와 소반도의 조인 누락·중복 확인
- 목재생산·숲가꾸기 등 기존 사업이력 중첩 제외
- 제한지역 레이어 최신성 확인
- 벌채구역·벌채면적·잔존구역을 서로 다른 필드로 관리
- 벌채구역 밖 운재로 면적을 별도 산출
- 군상·수림대·산림영향권 면적 합계와 도형 면적을 교차검산

## 자재조사 입력

- 조사방법, 조사면적, 표준지 수·면적, 확대계수 일치
- 매목 본수·수고·재적의 수종·경급 합계 일치
- 공시목 안배조서와 조재율·품등조사서 일치
- 원자료 수정 시 집계표와 시스템을 함께 갱신하고 버전 기록

## 품등 입력

> 감사 고위험 항목

- 일괄입력 전 공시목 품등비율 원자료 확정
- 일괄입력 후 최대품등 자동적용 여부 확인
- 수종별·경급별 품등비율을 원자료와 대조
- 생산재적 합계와 품등별 재적 합계 일치
- 평정서 출력 뒤 품등비율을 다시 역검산

## 운재로·비용

- 기설·신설·보수 운재로 구분
- 벌채구역 안·밖 면적 구분
- 중토장 운영 여부와 대·소운반 거리 입력
- m당 구축비를 ㎥당 비용으로 환산한 분모가 생산재적인지 확인
- 노선 감소·변경 시 공제비용과 추가징수 검토

## 가격·평정

- 원목시장가격의 조사분기와 적용일 기록
- 특수재 별도 시가조사
- 노임·유류·기계경비·보험료 기준일 통일
- 입목재적·생산재적·조재율 산식 검산
- 시장별 산출금액 중 적용시장 선택 근거 확인
- 평정일·평정관·첨부파일·기안문 누락 확인

## 전자계약·반출

- 계약서 부기 면적과 시스템 면적 일치
- 대금납부일·인도일·반출 시작일·종료일 일치
- 연장 시 계약·시스템·보증보험 종료일 동시 변경
- 운재로 변경계약과 준공검사 문서 연결
- 완료검사 후에만 계약종결·보증금 반환 처리
