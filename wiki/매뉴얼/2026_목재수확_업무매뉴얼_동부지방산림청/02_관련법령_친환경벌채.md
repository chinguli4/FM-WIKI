# Ⅱ. 관련법령 및 친환경벌채

> 현행 확인: 2026-07-13. 법령과 행정규칙이 매뉴얼보다 우선한다.

## 근거 체계

| 영역 | 주된 근거 | 실무 적용 |
|---|---|---|
| 경영계획·목재생산 | 국유림법 §9·§10 | 경영계획 편성·변경 후 실행 |
| 국유임산물 매각 | 국유림법 §27·§28, 시행규칙 §27~§30·별표 2·별지 19호 | 매각방법·계약·해제·반환 |
| 벌채 허가·점검 | 산림자원법 §36 | 제한지역, 운반로 의제, 실행·복구 점검 |
| 대면적 벌채 | 산림자원법 §36의4·§36의5, 시행령 §43의2~§43의7 | 10ha 조사, 20ha 심의 |
| 벌채 세부기준 | 산림자원법 시행규칙 별표 3 | 기준벌기령·벌채면적·연접거리 |
| 친환경벌채 | 친환경벌채 운영요령 §3~§9 | 군상·수림대·산림영향권·사후관리 |
| 가격·반출 | 국유임산물 매각예정가격 사정기준 등 시행요령 | 조사·가격·유찰·운재로·반출기간 |
| 계약 | 국가계약법령 | 경쟁·수의·재입찰·재공고 |
| 운반로 | 산지관리법 §15의2·§38, 국유림법 시행규칙 §27⑤ | 일시사용·복구·계약서 부기 |

## 친환경벌채 적용 판단

친환경벌채 운영요령은 모두베기, 왜림작업, 불량림 수종갱신에 적용한다. 5ha 기준과 존치비율은 시행규칙 별표 3과 함께 읽는다.

| 항목 | 기준 |
|---|---|
| 하나의 벌채구역 | 최대 30ha |
| 군상·수림대 존치 | 벌채면적의 20% 이상 |
| 군상 | 1개 이상 권장, 최소 폭 50m 이상, 가급적 원형 |
| 수림대 | 가능한 최소 폭 20m 이상, 기존 임분과 연결 |
| 산림영향권 | 군상·수림대 경계에서 평균 수고만큼 |
| 합계 | 군상+수림대+산림영향권이 벌채구역의 50% 이상 |
| 배치 우선 | 8부 능선, 급경사지, 계곡부, 도로변, 임연부 |
| 연접벌채 | 5ha 이상 30m, 5ha 미만 20m; 조림 준공 후 4년 경과 예외 확인 |

## 사전점검

- 희귀·멸종위기 동식물과 서식처 조사, 발견 시 보호조치 반영
- 등산로·도로·주요 조망점의 경관 영향 확인
- 마을 상수원·저수지·농경지로의 토사유출 방지계획 수립
- 기능별 벌채금지구역, 계곡부, 암석지·석력지 제척
- 군상·수림대가 생태적으로 연결되는지 현장 확인
- 운재로가 수림대·계곡부를 불필요하게 단절하지 않는지 검토

## 실행·사후관리

- 잔존목 손상을 예방하고 손상목 발생 시 즉시 보고·수량조사·처리
- 조재부산물은 수집·활용을 우선하고, 존치 시 유실되지 않도록 정리
- 군상은 병해충 등 불가피한 경우 외에는 훼손 금지
- 벌채 전·중·후 경계, 산림영향권, 운재로 임의변경 여부를 확인
- 오벌·도벌은 피해수량을 조사하고 계약·형사 절차를 구분하여 처리

## 주의

`군상·수림대·산림영향권 50%`는 실제로 나무를 전혀 베지 않는 면적이 50%라는 뜻이 아니다. 시행규칙 별표 3의 존치면적 20%와 운영요령의 영향권 합계 50%를 구분해 도면과 면적표에 각각 표시한다.
