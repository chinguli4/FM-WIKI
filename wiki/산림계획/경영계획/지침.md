# 산림경영계획 지침

> 근거: 산림자원의 조성 및 관리에 관한 법률 제14조, 산림경영계획 작성 및 운영규정 (산림청 훈령)

## 1. 개요

산림경영계획이란 산림 소유자 또는 경영자가 산림을 지속적으로 경영하기 위해 수립하는 10년 단위의 종합 계획이다.

### 법적 근거
- **의무 대상**: 국유림, 공유림
- **임의 대상**: 사유림 (단, 50ha 이상은 산림청 인가 권장)
- **계획 기간**: 10년 (5년마다 중간 점검)

### 인가의 이점 (사유림)
- 조림·숲가꾸기 보조금 우대 지급
- 벌채 허가 절차 간소화
- 산림경영자금 융자 우선 지원
- 세금 감면 혜택 (산지세 등)

## 2. 계획 수립 절차

```
1. 산림 현황 조사
        ↓
2. 경영 목표 설정
        ↓
3. 작업급 편성
        ↓
4. 수확조절 계획 수립
        ↓
5. 연년 작업 계획 작성
        ↓
6. 인가 신청 및 인가
        ↓
7. 계획 실행 및 모니터링
        ↓
8. 중간 검토 (5년 후)
        ↓
9. 차기 계획 수립
```

## 3. 경영계획서 구성

| 장 | 내용 |
|----|------|
| 1장 | 사업지 개황 (위치, 면적, 행정구역) |
| 2장 | 산림 현황 (임상도, 영급, 재적) |
| 3장 | 경영 목표 및 방침 |
| 4장 | 작업급 편성 |
| 5장 | 수확 계획 (연년벌채량) |
| 6장 | 갱신 계획 (조림·천연갱신) |
| 7장 | 보육 계획 (숲가꾸기) |
| 8장 | 보호 계획 (산불·병해충) |
| 9장 | 연년 작업 계획 (10년 스케줄) |
| 부록 | 산림현황도, 경영계획도, 표준지 조사 원부 |

## 4. 인가 신청

### 4.1 신청 기관
- 국유림: 국유림관리소
- 공·사유림: 지역 산림청 또는 시·군·구청

### 4.2 제출 서류
- 산림경영계획서 (정본 1부, 사본 2부)
- 임야대장 등본
- 토지이용계획확인원
- 산림현황도 (임상도)
- 표준지 조사 원부

### 4.3 인가 처리 기간
- 접수 후 30일 이내 (보완 요청 시 기간 정지)

## 5. 산림 현황 조사 방법

### 5.1 임상 구분
| 임상 | 기준 |
|------|------|
| 침엽수림 | 침엽수 재적 비율 75% 이상 |
| 활엽수림 | 활엽수 재적 비율 75% 이상 |
| 혼효림 | 침·활엽수 비율 각각 25~75% |
| 무립목지 | ha당 30본 미만 (또는 재적 5㎥ 미만) |

### 5.2 영급 구분
| 영급 | 임령 |
|------|------|
| Ⅰ영급 | 1~10년 |
| Ⅱ영급 | 11~20년 |
| Ⅲ영급 | 21~30년 |
| Ⅳ영급 | 31~40년 |
| Ⅴ영급 | 41~50년 |
| Ⅵ영급 이상 | 51년 이상 |

### 5.3 지위지수 (Site Index)
- 수종별 우세목의 50년생 수고(m)로 표현
- 우량지 (S.I. 20 이상) / 중간지 (S.I. 14~20) / 불량지 (S.I. 14 미만)

## 6. 작업급 편성

### 6.1 작업급의 개념
동일한 수종·작업종·윤벌기로 관리되는 임분의 집합

### 6.2 편성 기준
| 구분 | 기준 |
|------|------|
| 수종 | 주수종별 구분 |
| 작업종 | 교림작업·왜림작업 구분 |
| 윤벌기 | 목표 벌채 임령 설정 |
| 작업급 면적 | 가급적 10ha 이상 |

### 6.3 주요 수종별 윤벌기 기준

| 수종 | 표준 윤벌기 |
|------|------------|
| 소나무 | 60~80년 |
| 잣나무 | 70~100년 |
| 낙엽송 | 40~50년 |
| 편백 | 60~80년 |
| 참나무류 | 70~100년 |
| 백합나무 | 30~50년 |

## 7. 계획 변경

- **경미한 변경**: 작업 시기 조정 (±1년), 사업량 10% 이내 변경
- **중요 변경**: 수확량 10% 이상 변경, 수종 변경, 작업급 재편성 → 재인가 필요
- 변경 신청: 사유 발생 후 30일 이내

## 8. 점검 및 보고

- 연도별 실행 실적 보고: 매년 1월 말 (전년도 실적)
- 5년 중간 점검: 계획 달성도 평가 및 필요 시 수정
- 인가기관의 현장 확인: 필요 시 수시
