## A. GitHub 웹에서 수정요청 (PR) 하기

> Git을 몰라도 됩니다. 웹 브라우저만 있으면 됩니다.

### 준비

1. GitHub 계정 만들기 → https://github.com/signup
2. 관리자에게 GitHub 아이디 알려주기 (초대 받아야 수정 가능)
3. 초대 메일 수락

### 수정요청 절차

**1단계 — 저장소 열기**

https://github.com/chinguli4/FM-WIKI 접속

**2단계 — 파일 찾기**

`wiki/` 폴더에서 수정할 파일 클릭

**3단계 — 편집 시작**

파일 우측 상단 연필 아이콘(✏️) 클릭

> "You don't have permission..." 메시지가 나오면 초대가 안 된 상태 → 관리자에게 문의

**4단계 — 내용 수정**

페이지 내용 직접 수정

**5단계 — 변경사항 저장**

우측 상단 **Commit changes...** 클릭
- 첫 번째 칸: 무엇을 고쳤는지 한 줄 요약 (예: `친환경벌채 면적 기준 수정`)
- **Create a new branch...** 선택 (기본값 유지)
- **Propose changes** 클릭

**6단계 — PR 제출**

바로 나오는 화면에서 **Create pull request** 클릭
- 수정 이유·참고사항 있으면 본문에 추가
- 다시 **Create pull request** 클릭

이후 관리자가 검토 후 반영합니다. 반영되면 GitHub에서 알림이 옵니다.

## B. Claude Code로 작업하기

> AI에게 질문하거나 수정을 맡기고 싶을 때 사용합니다.

### 준비 (최초 1회)

**1. Git 설치**

https://git-scm.com/download/win 에서 다운로드 후 설치 (기본값으로 진행)

**2. Claude Code 설치**

https://claude.ai/download 에서 데스크탑 앱 다운로드 후 설치  
또는 VS Code 확장 프로그램에서 `Claude Code` 검색 후 설치

**3. 저장소 복사 (clone)**

명령 프롬프트(cmd) 또는 터미널을 열고:

```bash
git clone https://github.com/chinguli4/FM-WIKI.git
cd FM-WIKI
```

### 사용 방법

**Claude Code 앱에서 폴더 열기**

- File → Open Folder → 방금 받은 `FM-WIKI` 폴더 선택

**질문하기 (조회만)**

채팅창에 자유롭게 질문합니다:

```
친환경벌채 시 군상 면적 기준이 얼마야?
솎아베기 잔존본수 기준 알려줘
산사태취약지역 기초조사 점수 기준이 뭐야?
```

Claude가 wiki 페이지를 읽고 조항을 인용해 답변합니다.

**수정요청하기**

```
친환경벌채 운영요령 페이지에서 군상 최소 면적 기준이 잘못됐어, 0.04ha → 0.05ha로 수정해줘
```

Claude가 파일을 직접 수정합니다.

**수정 후 PR 올리기**

수정이 완료되면 채팅창에 다음과 같이 요청합니다:

```
수정사항을 새 브랜치로 커밋하고 PR 올려줘
```

Claude가 브랜치 생성 → 커밋 → GitHub PR 생성까지 처리합니다.

> ⚠️ main 브랜치에는 직접 push가 막혀 있습니다. 반드시 PR을 통해 제출해야 합니다.

## 관리자 — PR 검토·승인 방법

1. https://github.com/chinguli4/FM-WIKI/pulls 접속
2. PR 클릭 → **Files changed** 탭에서 변경 내용 확인
3. 문제 없으면 **Review changes** → **Approve** → **Submit review**
4. **Merge pull request** 클릭
