# 국유재산 사건의 산지관리법 연계 판단

> 대상: 국유림관리소 재산관리 담당자
> 근거: 국유림법 §16·§21~§22·§26, 국유재산법 §6, 산지관리법 §4·§12·§14~§21·§37~§44
> 기준일: 2026-07-13

## 한 사건에 적용되는 세 층

| 층 | 질문 | 처분·계약 |
|---|---|---|
| 재산층 | 국가 소유 토지를 사용할 권원이 있는가 | 국유림 사용허가·대부, 국유재산 관리 |
| 산지층 | 해당 산지에서 목적행위가 허용되는가 | 산지전용·일시사용 허가·신고·의제협의 |
| 이행층 | 납부·재해방지·복구의무를 이행했는가 | 조성비·복구비·설계·감리·준공·반환 |

어느 한 층의 처분으로 다른 층의 판단을 대신하지 않는다.

## 단계별 처리

### 1. 접수·분류

- 소유자·관리청·재산대장·사용현황 확인
- 보전/준보전국유림과 행정/일반재산 확인
- 산지 해당 여부와 임업용/공익용/준보전산지 확인
- 보호구역·백두대간·휴양림·사방지 등 중첩 규제 확인

### 2. 목적사업 가능성

- 보전국유림이면 국유림법 §21①의 사용허가 목적에 해당하는지 확인
- 보전산지이면 산지관리법 §12의 허용행위에 해당하는지 확인
- 두 법 중 하나라도 허용하지 않으면 단순 서류보완으로 진행하지 않음

### 3. 전용·일시사용 분류

- 영구 시설·용도변경이면 산지전용 검토
- 복구조건부 임시사용·작업로·운반로이면 산지일시사용 검토
- 실제 형질변경 면적, 절·성토, 시설 존치기간, 복구방식을 기준으로 도면 확정

### 4. 의제 또는 별도처분

- 국유림법 §21② 의제 여부를 조문·대상행위·계약기재로 확인
- 의제이면 산지관리법령상 협의서류와 허가기준 심사를 수행
- 의제되지 않는 부분은 별도 허가·신고를 안내
- 의제 범위와 별도처분 범위를 같은 도면에 구분 표시

### 5. 금액과 효력

- 사용료·대부료, 조성비, 복구비, 철거보증, 지장목대금을 별도 산정
- 감면·면제는 각각의 법정근거 확인
- 다른 법률상 인허가, 조성비 납부, 복구비 예치가 필요한 사건은 완료 전 착수 금지

### 6. 허가·계약조건

- 필지·면적·도면번호·좌표
- 목적·시설·노선·공종·기간
- 벌채·굴취·채취 범위와 산물처리
- 재해방지·복구설계·감리·준공
- 변경허가·변경신고·재협의 기준
- 위반 시 중지·취소·철거·복구·비용부담

### 7. 변경·사후관리

- 노선·면적·시설·기간·목적 변경을 현장조정으로 처리하지 않음
- 국유림 대부등 변경과 산지전용·일시사용 변경을 함께 검토
- 복구비 재산정·추가예치 및 보증기간 갱신
- 실태조사 때 허가도면과 실제 점유·형질변경을 대조

### 8. 종결

- 시설물 철거, 복구설계 이행, 감리·준공검사 확인
- 하자보수보증 후 복구비 반환
- 국유림 계약·허가 종결과 미납·변상금·손해배상 별도 정리
- 시스템·대장·사건철에 최종 면적과 처분 결과 반영

## 자주 틀리는 판단

| 잘못된 판단 | 바로잡기 |
|---|---|
| 사용허가가 났으니 산지전용도 가능하다 | 권원과 행위규제는 별도 판단 |
| 의제이므로 산지관리 서류가 필요 없다 | 협의서류·허가기준·부담금·복구는 적용 |
| 계약서의 포괄 문구로 모든 변경이 의제된다 | 명시된 행위·구역·면적·조건 범위만 의제 |
| 복구비를 받았으니 철거보증은 불필요하다 | 실제 비용 포함 여부와 각 법적 의무를 대조 |
| 산지복구 준공으로 국유재산 사건도 끝난다 | 계약·원상회복·대장·미납을 별도 종결 |
