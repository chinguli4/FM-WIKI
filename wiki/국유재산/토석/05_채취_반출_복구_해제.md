# 토석 채취·반출·복구·계약해제

> 근거: 산지관리법 §36~§43, 시행규칙 §35~§45

## 착수 전

- 매각대금 납부 확인
- 복구비·추가복구비 예치와 보증기간 확인
- 승인 복구계획·복구설계서 확인
- 채취·완충구역 경계표시와 계획고·측량기점 확인
- 장비·기술인력·업종등록 유효성 확인
- 재해방지시설 선시공 조건 이행 확인

## 채취·반출 관리

- 일자별 채취량·반출량·잔량 관리
- 차량·계근표·반출증과 계약수량 대조
- 계약구역 밖 채취·계획고 초과 굴착 금지
- 외부 반입 토석과 매각 토석을 구분 계상
- 계약서 반출기간 내 국유림 밖으로 반출
- 부득이한 경우 별지 제35호로 1회 연장 검토

## 복구·종결

1. 중간복구와 완료구간을 확인한다.
2. 승인 설계대로 사면·배수·토사·녹화·시설철거를 이행한다.
3. 감리 대상이면 시정·완료자료를 확인한다.
4. 복구준공검사와 하자보수보증을 처리한다.
5. 대행·대집행 비용을 공제하고 복구비를 반환한다.
6. 최종 측량으로 채취면적·수량·복구범위를 확정한다.

## 계약해제·양여취소 분기

| 위반 | 조치 |
|---|---|
| 장비기준 미달 | 보완·중지·계약해제 검토 |
| 계약 외 토석 채취 | 중지·해제·원상복구·형사절차 검토 |
| 대금 미납 | 계약해제·채권관리 |
| 재해방지 명령 불이행 | 중지·해제·대행·대집행 |
| 복구비 미예치·재예치 불이행 | 착수금지·계약해제 |
| 거짓·부정한 매입·양여 | 의무적 해제·취소 |

처분과 계약조치는 근거·절차가 다르므로 통지, 의견제출·청문 대상, 계약해제 문서를 각각 검토한다.
