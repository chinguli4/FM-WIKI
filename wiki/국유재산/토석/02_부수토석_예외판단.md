# 산지전용·일시사용 부수 토석의 예외 판단

> 근거: 산지관리법 §25의2·§28②·§35⑤, 시행령 §37

## 부수 토석 판정

다음 자료가 서로 일치해야 한다.

- 선행 산지전용·일시사용 허가·신고·의제처분
- 선행 사업의 구역·공종·절성토 계획
- 토석 발생 위치와 원인
- 종류·수량·용도·가공·반출계획
- 토석을 얻기 위한 추가 굴착인지 여부

사업 수행 중 우연히 발생한 범위를 넘어 토석 취득을 목적으로 추가 굴착하면 부수 토석 예외를 적용하기 어렵다.

## 예외는 항목별 판단

| 검토항목 | 부수 토석 분기 |
|---|---|
| 토석채취제한지역 | §25의4·시행령 §32의4의 허용사유 확인 |
| 별표 8 허가기준 | 시행령 §37③2가에 따른 적용예외 가능성 |
| 채석경제성 | §26 단서·시행령 §34② 예외 확인 |
| 주민 등 동의 | 도시지역이면 지방산지관리위원회 심의 후 제외 가능성 |
| 장비·기술인력 | 직접 가공하지 않을 때 제외 가능성 |
| 반입 제한 | 예외사유에 따라 반입이 금지되는 분기까지 확인 |
| 재해·복구 | 면제 아님; 별도 적용 |

## 금지하는 판단 방식

- “부수 토석이므로 전부 면제”라고 일괄 결론
- 2020 원문의 `◉`를 다른 사건에 그대로 복사
- 선행 인허가 도면 밖에서 나온 토석을 부수 토석으로 처리
- 누적 물량을 나누어 가격평가·권한 기준을 낮춤
- 매각계약 전에 반출하거나 사용하게 함

## 기록 문장

`선행 처분 / 발생 공종 / 발생 위치 / 토석 종류·수량 / 부수성 근거 / 적용 제외 조문 / 계속 적용되는 기준 / 복구·반출 조건`을 표로 남긴다.
