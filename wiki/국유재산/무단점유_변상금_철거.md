# 무단점유·변상금·철거

> 근거: 국유재산법 §72~73, 국유림법 §22·§26·§33, 행정절차법, 국가재정법·국세징수법 관련 규정

## 5개 트랙

| 트랙 | 목적 |
|---|---|
| 사실확정 | 점유자·면적·기간·용도·시설 소유자 특정 |
| 금전 | 변상금·연체료·강제징수 |
| 재산회복 | 반환·철거·원상회복·경계회복 |
| 위반처리 | 산림훼손·불법시설의 행정·형사 조치 |
| 분쟁 | 경계·소유권·쟁송 대응 |

## 변상금

국유재산법 §72는 원칙적으로 사용료·대부료의 120% 상당을 변상금으로 징수한다. 변상금은 과거 무단사용에 대한 금전부담일 뿐 장래 사용권원이 아니다.

## 절차

대장·항공사진 조사 → 현황측량 → 점유자 확인 → 부과기간·용도·재산가액 산정 → 사전통지·의견 → 부과·고지 → 수납·체납 → 철거·복구 → 종결.

## 경계·기간 다툼

점유자 진술만으로 점유개시일과 면적을 확정하지 않는다. 항공사진, 과세자료, 건축물대장, 전기·수도, 과거 조사, 측량성과 등 객관자료를 사용한다.

## 철거

시설물 소유자와 설치시기, 행정대집행 가능 여부, 민사상 철거·인도청구 필요 여부, 산지복구명령을 구분한다. 자진철거 기한과 불이행 시 조치를 문서에 명확히 적는다.
