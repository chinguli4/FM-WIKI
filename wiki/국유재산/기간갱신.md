# 기간갱신

> 근거: 국유림법 §21④, 시행령 §19, 관리규정 §5·§11

## 원칙

국유림 대부등은 5년 이내이며 갱신할 수 있지만, 갱신기간도 매회 5년을 넘을 수 없다. 갱신은 자동 권리가 아니라 처분 시점의 현행기준에 따른 재심사다.

## 심사표

- 목적사업 진행정도와 실제 이용
- 허가·계약조건 준수
- 면적·용도·시설 변경
- 전대·무단양도·무단점유
- 사용료·대부료·부담금 체납
- 산림경영계획·사용계획과 충돌
- 관계법령 인허가 유효기간
- 복구비·철거비·보증서 갱신
- 민원·재해·공익상 필요

## 처리

만료일 사전관리 → 신청 접수 → 실태조사 → 관계부서 의견 → 현행기준 심사 → 금액·보증 재산정 → 갱신처분·계약 → 대장·시스템 반영.

> 2026년 장기 독점사용 갱신심사 강화와 최초연도 조정 제한을 내용으로 하는 시행령 개정안이 입법예고되었으나 2026-07-13 현재 공포 전이므로 직접 적용하지 않는다.
