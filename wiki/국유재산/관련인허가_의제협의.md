# 관련 인허가와 의제협의

> 근거: 국유림법 §21②, 산지관리법, 산림자원법 §36, 산림보호법, 각 목적사업법

## 두 질문을 분리한다

1. 국가 소유 토지를 사용할 권원이 있는가.
2. 그 장소에서 목적행위를 할 공법상 허가가 있는가.

둘 중 하나라도 없으면 사업을 시작할 수 없다.

## 협의 절차

1. 주된 인허가와 처분권자 특정
2. 의제되는 인허가의 조문·범위·조건 확인
3. 협의기관별 검토자료 목록 작성
4. 동일한 도면·면적·사업계획으로 일괄 협의
5. 보완·조건·부동의 사항 정리
6. 허가조건·계약·복구계획에 협의조건 반영
7. 변경 시 재협의 필요 여부 확인

## 주요 중첩

| 영역 | 확인 |
|---|---|
| 산지 | 산지전용·일시사용, 대체산림자원조성비, 복구 |
| 산림자원 | 입목벌채·굴취·채취, 산림경영계획 |
| 보호 | 산림보호구역, 산불·병해충, 보호종 |
| 공간계획 | 용도지역·지구, 개발행위, 건축 |
| 특수구역 | 백두대간, 자연휴양림, 사방지, 문화·자연유산 |
| 목적사업 | 전기·통신·도로·철도·광업·수도·공익사업 |

## 의제 검토 기록

`주된 처분 / 의제조문 / 의제대상 / 협의기관 / 협의결과 / 조건 / 도면번호 / 변경 시 재협의 기준`을 표로 남긴다.
