# 사용료·대부료·부담금·보증

> 근거: 국유림법 §22~23, 시행령 §20~24의2, 관리규정 §7~8의2

## 2026-07-13 국유림 대부료등 핵심

| 항목 | 현행 기준 |
|---|---|
| 기간 | 원칙적으로 5년 이내, 갱신마다 5년 이내(시행령 §19) |
| 기본 요율 | 목적별 국유림 가격의 1천분의 10·20·25·50 이상(시행령 §21) |
| 토지가격 | 원칙적으로 개별공시지가; 부적합·부재 시 표준지·2인 이상 감정평가 평균·유사토지 공시지가 |
| 통합징수 | 연간 대부료등 20만원 이하인 경우 가능 |
| 납부기한 | 고지일부터 60일 이내; 신규는 사용 시작 전 납부 |
| 분납 | 50만원 초과 시 연 12회 이내, 잔액에 고시이자율 적용 |
| 담보 | 연간 1천만원 이상이면 연간 대부료등의 50% 범위 보증 |
| 조정 | 전년 대비 9% 이상 증가 시 9% 증가액; 일부 농림어업·주거용은 5% 기준 |

## 부과 전 검산

- 목적사업이 적용 요율과 일치하는가.
- 공부상 지가가 실제 이용상태를 반영하는가.
- 면적·기간·일할계산이 도면과 일치하는가.
- 감면의 법적 근거와 직접사용 요건을 갖추었는가.
- 조정은 신규·갱신·변상금에 잘못 적용되지 않았는가.
- 부가세·분납이자·연체금이 구분되었는가.

## 복구·철거 보증

관리규정 §7은 철거·원상회복 비용을 전문업체 견적을 기준으로 대부등 기간에 따라 연 5% 할증하고, 대부등 기간에 6개월을 더한 기간 동안 예치하도록 한다. 산지복구비에 해당 비용이 포함되었는지 중복 여부를 확인한다.

## 시스템 통제

사정조서, 허가·계약, 고지, 수납, 보증서, 재산대장 금액이 모두 일치해야 한다. 법정 요율을 임의 수정하지 말고 특례·감면 근거를 문서와 시스템에 함께 남긴다.
